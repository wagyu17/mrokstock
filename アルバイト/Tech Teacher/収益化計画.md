# Minecraft MOD解説コンテンツ 収益化計画

## 目的

Minecraft MODの操作解説コンテンツを、YouTube・ブログ・PDF販売につなげて収益化する。  
最終的には、私が担当する作業を「解説書を見ながら実際にMinecraftを操作して録画すること」に寄せ、それ以外の解析・編集・字幕・記事化・販売導線作成はできるだけCodex / Claude Codeで自動化する。

## 基本戦略

収益化の導線は以下の流れにする。

```text
Shortsで認知獲得
↓
長尺動画で詳しい使い方を解説
↓
ブログ記事で検索流入を受ける
↓
詳しいPDFガイドを販売
↓
MOD別の解説PDF・動画・記事を横展開
```

Shortsは「かっこいい・自分でもできそう」と思わせる役割。  
長尺動画は「実際に使えるようになる」ための説明。  
PDFは「手元に置いて見返せる詳しい操作マニュアル」として販売する。

## ターゲット

主なターゲットは海外のMinecraft MODユーザー。

- Minecraft Java EditionでMODを使いたい人
- TaCZなどの銃MODを初めて使う人
- キー操作や設定でつまずいている人
- 動画だけでなく、一覧表やPDFで確認したい人

英語タイトル・英字幕・英語説明文を基本にする。

私が行う作業:

- Minecraftを起動

- 操作表示が見えるようにする

撮影する素材例:

### 2. 素材配置

録画素材は以下へ入れる。

```text

```


- TTS環境があればAI音声を生成

- 問題なければ本番動画を書き出す

## ブログ販売システムの方針

最初からログイン付きの本格的な有料記事システムを作るより、まずは以下の形にする。

```text
YouTube Shorts / 長尺動画
↓
無料ブログ記事
↓
有料PDF販売ページ
↓
購入後にPDFダウンロード
```

無料ブログ記事では、読者が「このPDFを買えば迷わず操作できそう」と分かるところまで見せる。  
ただし、PDF本体の価値になる一覧表、詳しい手順、トラブル対処、チェックリストは有料PDF側にまとめる。

### 1記事ごとの商品設計

1つのMOD記事につき、1つのPDF商品を対応させる。

例:

```text
無料記事:
TaCZ AWM Sniper Beginner Guide

有料PDF:
TaCZ AWM Sniper Complete Control Guide PDF
```

商品ページに入れる内容:

- PDFの対象者
- PDFで分かること
- 目次
- サンプル画像
- 価格
- 購入ボタン
- YouTube動画へのリンク
- CurseForgeなど公式配布ページへのリンク
- 免責表記

### 価格

最初のPDFは300円程度から始める。

海外向けには、円表示だけでなくドル表示も検討する。

```text
日本向け: 300円
海外向け: $1.99 - $2.99 前後
```

ただし、決済サービスによっては固定手数料があるため、300円の商品は利益率が低くなりやすい。  
販売開始後に反応があれば、以下のように価格帯を上げる。

```text
初回テスト: 300円程度
通常版: 500円 - 980円程度
複数MODまとめ版: 1,480円 - 2,980円程度
```

### 決済サービスの候補

最初は、購入後のPDF配布まで簡単にできる外部販売サービスを使う。

候補:

```text
Gumroad
Stripe Payment Links
BOOTH
note
```

海外向けを重視するなら、最初はGumroadが分かりやすい。  
Gumroadはデジタル商品、電子書籍、チュートリアル販売に向いており、国際的な税務処理も任せやすい。  
一方で、少額商品では手数料の影響が大きくなる。

Stripe Payment Linksは、自分のブログから決済ページへ直接飛ばせる。  
ただし、購入後のPDF配布、税務、返金対応などを自分で設計する必要があるため、最初は少し難しい。

最初の推奨:

```text
短期: GumroadでPDFを販売
中期: 売れる商品が見えたらStripe Payment Linksや独自販売ページを検討
長期: MOD別PDFセット、会員制、バンドル販売へ展開
```

### YouTubeからの導線

YouTubeの説明欄と固定コメントからブログ記事へ誘導する。

説明欄の例:

```text
Full TaCZ AWM guide PDF:
https://example.com/tacz-awm-guide

Learn the controls, reload, aim, customize, and fix common beginner mistakes.
```

固定コメントの例:

```text
Want the full TaCZ AWM controls guide?
I made a PDF with keybinds, setup steps, and beginner mistakes:
https://example.com/tacz-awm-guide
```

リンク先は直接PDFファイルではなく、必ずブログの商品ページにする。  
ブログページで内容と価値を説明してから購入ボタンへ誘導する。

### 注意点

- Minecraft公式ロゴをPDF表紙や商品画像に使わない
- `Minecraft` を商品名の主役にしすぎない
- `TaCZ Beginner Guide for Minecraft Java Edition` のように、解説対象として自然に使う
- MOD本体やjarファイルを再配布しない
- MOD作者の公式配布ページにリンクする
- PDFは「MODの使い方解説」として販売する
- YouTubeのリンクは、安全で誤解を招かないページにする

## 注意点

- MODのjarや有料素材を再配布しない
- CurseForgeなど公式配布ページへリンクする
- BGMは利用許可が明確なものだけ使う
- 他人の声を無断でAI音声化しない
- AI生成サムネの文字崩れに注意する
- 操作表示や字幕で照準・重要UIを隠さない
- PDFではMOD作者の権利を尊重し、解説コンテンツとして販売する
