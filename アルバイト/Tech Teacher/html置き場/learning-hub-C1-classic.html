Windowsの基本操作

  1. 1回クリックは「選ぶ」ファイルやフォルダを1回だけ押すと、青くなって選ばれます。
  2. ダブルクリックは「開く」フォルダやファイルを2回すばやく押すと開きます。開かなければ、あわてずもう一度やります。
  3. 上のバーを持つと画面を動かせるウィンドウの一番上をドラッグすると、画面の場所を変えられます。
  4. 下のタスクバーで戻る画面を見失ったら、下にあるアイコンからマイクラ、Clipchamp、エクスプローラー、このページに戻ります。
  5. エクスプローラーでファイルを探す黄色いフォルダのマークを開くと、デスクトップ、ダウンロード、ビデオ、ピクチャを探せます。

  1. マイクラが動かなくなったら `Esc`メニューや入力画面で止まったように見える時は、まず `Esc` キーを押して戻ります。
  2. 英数字を打つ前に「あ」か「A」を見る画面の下のバーが「あ」なら日本語入力です。コマンドや英数字を打つ時は、`半角 / 全角 漢字` キーで「A」にしてから操作します。

  4. ペーストは `Ctrl` + `V`貼りたい場所をクリックしてから、`Ctrl` を押しながら `V` を押します。

保存場所の探し方

Clipchampの動画書き出す前に「保存先」を見ます。見つからない時は、エクスプローラーで「ビデオ」「ダウンロード」「デスクトップ」を順に探します。動画は `.mp4` が多いです。
ダウンロードしたMODまず「ダウンロード」フォルダを見ます。`.zip` や `.jar` のファイルは、勝手に開かず大人に見せます。
スクリーンショット「ピクチャ」や「スクリーンショット」フォルダに入ることがあります。ゲーム内で保存した画像は、使ったアプリの保存場所も見ます。

マイクラのコマンド

コマンドは、マイクラの世界にお願いを出す言葉です。最初の `/`、スペース、数字の場所をよく見ます。長いコマンドは、1文字ずつではなく「かたまり」で読みます。

/time /time set day 朝や夜など、世界の時間を変えるコマンドです。暗くて作業しにくい時は day にします。
/effect /effect give @p speed 30 1 自分や相手に効果をつけます。`@p` は近くのプレイヤー、`30` は秒数、`1` は強さです。
/fill /fill x1 y1 z1 x2 y2 z2 stone 2つの場所の間をブロックでうめます。座標をまちがえると広く変わるので、テスト場所で使います。

コマンドを打つ前に見るところ

  1. 先頭に `/` があるか見るコマンドは `/time` のように、スラッシュから始まります。
  2. スペースで分かれているか見る`/time set day` は、`/time`、`set`、`day` の3つのかたまりです。
  3. 対象を見る`@p` は近くのプレイヤー、`@a` は全員、`@s` は自分です。
  4. /fill は小さい範囲で試す大きな範囲を一気に変えるので、最初は小さい場所で練習します。

CurseForgeでMOD導入

CurseForgeは、マイクラのMODやMODパックを探して入れる時に使います。大事なのは「マイクラのバージョン」「MODローダー」「入れた後にどこから起動するか」を見ることです。

バージョン

MODが対応している Minecraft のバージョンを見ます。合っていないと動かないことがあります。

MODローダー

Forge、Fabric、NeoForge などを見ます。MODとプロファイルの種類を合わせます。

起動

CurseForge Appで入れた時は、作ったプロファイルから Play して起動します。

CurseForgeで見る順番

  1. 入れたいMODを決める何ができるMODなのか、1行で言えるようにします。
  2. Minecraft Versionを見る自分が遊ぶバージョンと合っているか確認します。
  3. Forge / Fabric / NeoForge を見るMODローダーが合っているか確認します。
  4. 新しいプロファイルで試す大事なワールドをこわさないように、まずテスト用ワールドで確認します。
  5. 起動場所を覚えるCurseForgeで入れたMODは、CurseForge Appのプロファイルから起動することがあります。

クラフターズコロニーの配布ワールド

配布ワールドは、ほかの人が作ったワールドをダウンロードして遊ぶものです。最初に「統合版かJava版か」「ファイルがどこに保存されたか」「どうやってマイクラに入れるか」を見ます。

版を見る

統合版/BE向けか、Java版向けかを確認します。自分のマイクラと違う版だと遊べません。

保存場所

ダウンロードしたファイルは、まず「ダウンロード」フォルダで探します。

形式を見る

統合版では `.mcworld`、Java版では `.zip` など、ファイル形式を見ることが大事です。

配布ワールドを入れる時の順番

  1. ワールドの説明を読む何をするワールドか、対応しているマイクラの版とバージョンを見ます。
  2. ダウンロード後に場所を見るファイルが見つからない時は、エクスプローラーで「ダウンロード」を開きます。
  3. 統合版は `.mcworld` を見る`.mcworld` ファイルは、開くとマイクラにインポートされることがあります。
  4. Java版はフォルダの場所に注意するJava版のワールドは、解凍や保存場所の確認が必要なことがあります。わからなければ大人に聞きます。
  5. 入れた後はワールド一覧を見るマイクラを開いて、ワールド一覧に追加されているか確認します。

Clipchamp 動画づくり

ゴールは、15秒から30秒の短い動画を1本作ること。まず `Win + G` でマイクラの画面を録画し、Clipchampで短く切って見やすくします。

動画
文字
音

動画づくりの順番

  1. テーマを1つ決める例: 建築しょうかい、すごい装置、今日のベストシーン。
  2. `Win` + `G` で録画画面を開くキャプチャ画面が開いたら、赤い録画ボタンを押して画面収録を始めます。
  3. 素材を2つから5つ選ぶ動画や写真を入れすぎると見づらいので、少なめでOKです。
  4. いらないところを切る待っている時間、同じ場面、失敗している長い部分を短くします。
  5. 文字を1つ入れる見ている人に伝えたいことを、短い言葉で入れます。
  6. 保存先を見て書き出す音量を確認してから、どのフォルダに保存されるか見て、動画を書き出します。
  7. 保存した動画を見つけるエクスプローラーで保存先を開きます。わからない時は「ビデオ」「ダウンロード」「デスクトップ」を順に探します。
