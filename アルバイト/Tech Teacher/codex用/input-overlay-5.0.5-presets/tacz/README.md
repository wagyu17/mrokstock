# TaCZ full controls preset for OBS Input Overlay

OBS の「Input Overlay」ソースで以下を指定してください。

- Image/Texture: `tacz_full.png`
- Config/Layout: `tacz_full.json`

表示キー:
W/A/S/D, Shift, Space, LMB, RMB, R, G, Z, H, C, V, O, Alt, T

TaCZ の主な操作:
- LMB: 射撃
- RMB: ADS/狙う
- R: リロード
- G: 射撃モード切替
- Z: アタッチメント/カスタム画面
- H: 点検
- V: 近接攻撃、ADS中ズーム
- O: 銃を持った状態のインタラクト
- C: 伏せ
- Alt + T: 設定
