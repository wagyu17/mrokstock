# Techguns キー割り当てと操作方法

## 解析元

- `techguns-1.12.2-2.0.2.0_pre3.2.jar`

Minecraft 1.12.2 の mod なので、キーコードは LWJGL 2 系の値から復元しています。`TGKeybinds`、`PacketTGKeybindPress`、銃と Jetpack 系アイテムの処理を逆コンパイルして確認した内容です。

## キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| 左クリック | 射撃 | 多くの Techguns 銃は左クリックで射撃します。 |
| `R` | 強制リロード | 手に持っている銃をリロードします。 |
| `N` | ナイトビジョン切り替え | 対応する防具ボーナスがある場合、ナイトビジョンを切り替えます。 |
| `B` | セーフモード切り替え | セーフモードを切り替えます。設定や権限によっては強制的に有効へ戻されます。 |
| `V` | ステップアシスト切り替え | 対応装備のステップアシストを切り替えます。 |
| `J` | ジェットパック切り替え | ジェットパックの有効/無効を切り替えます。 |
| `Space` | ジェットパック上昇 | ジェットパック有効時、ジャンプキーで上昇・ブーストします。 |

## 銃の基本操作

1. 銃と対応する弾薬を用意します。
2. 左クリックで射撃します。
3. `R` で強制リロードします。
4. 武器によっては右クリック側に別アクションが設定されている場合があります。

両手に銃を持っている場合、`R` の強制リロードはリロード可能な銃を選びます。両方リロード可能な場合は、弾薬割合が低い側を優先する処理があります。

## 防具・補助機能

- `N` は、装備にナイトビジョン系の防具ボーナスがあるときだけ意味があります。
- `V` は、対応装備のステップアシストを有効/無効にします。
- `B` はセーフモードの切り替えです。サーバーや設定で unsafe mode が禁止されている場合は、オフにしても自動的にオンへ戻されます。

## ジェットパック

- `J` でジェットパック機能を有効/無効にします。
- 有効状態で `Space` を押すと上昇・ブーストします。
- 燃料や耐久がない場合は動作しません。
- ジャンプキーを押している最中に `J` を押した場合、内部的にジャンプ状態を解除する処理があります。

## 注意点

- GUI 用に HUD アイコン切り替え処理は存在しますが、この jar の `TGKeybinds` では独立したキーバインドとして登録されていませんでした。
- `R` は多くの銃 mod や戦闘 mod と競合します。
- `V` は TaCZ の近接/ズーム、JujutsuCraft のメイン技、ProjectE のチャージと競合しやすいです。
- `B` は Jujutsu Kasien Addon の 0.2s domain と競合します。
