# Jade キー割り当てと操作方法

## MOD概要

Jade は、見ているブロック・エンティティの名前、MOD名、状態、収納内容などを画面上に表示する情報表示系MODです。WAILA/HWYLA 系の後継として、探索・建築・MOD学習で非常に便利です。

## 解析元

- jar: `Jade-mc26.1-NeoForge-26.1.1.jar`
- modId: `jade`
- loader: NeoForge
- 主な解析対象: `snownee/jade/JadeClient.class`, `assets/jade/lang/ja_jp.json`, `snownee/jade/command/**`

## キー割り当て

`JadeClient` のキー登録処理から、次の割り当てを確認しました。

| キー | 機能 |
| --- | --- |
| テンキー `0` | Jade の設定を開く |
| テンキー `1` | オーバーレイ表示のオン・オフ |
| テンキー `2` | 液体表示の切り替え |
| テンキー `3` | レシピ表示 |
| テンキー `4` | 使用先表示 |
| テンキー `5` | 読み上げ |
| `左Shift` | 詳細表示 |
| 未割り当て | プロファイル 0-3 の切り替え |

テンキー `3` とテンキー `4` のレシピ・使用先表示は、JEI/REI/EMI などのレシピ表示MODと組み合わせたときに意味を持つ機能です。

## 独自コマンド

Jade にはサーバー側コマンドとクライアント側コマンドが含まれています。

| コマンド | 内容 |
| --- | --- |
| `/jade show` | Jade表示を有効化するコマンド候補 |
| `/jade hide` | Jade表示を非表示にするコマンド候補 |
| `/jadec config` | クライアント側設定を扱うコマンド候補 |
| `/jadec handlers` | 表示ハンドラーの確認・切り替え用候補 |
| `/jadec use_profile <index>` | 表示プロファイル切り替え候補 |
| `/jadec pin` | 現在見ている情報の固定表示候補 |

`/jadec` はクライアントコマンドとして実装されているため、通常のサーバー管理コマンドとは扱いが異なります。

## 操作方法

### ブロック情報を見る

1. ワールドに入ります。
2. ブロックやエンティティに照準を合わせます。
3. 画面上部に名前、MOD名、状態などが表示されます。

チェスト、かまど、タンク、機械などは、対応している場合に内容物や進行状況も表示されます。

### 詳細情報を一時的に表示する

`左Shift` を押すと、通常より詳しい情報を表示します。収納ブロックの中身、ブロック状態、追加説明など、普段は省略される情報を見るときに使います。

### オーバーレイを消す

画面表示が邪魔なときはテンキー `1` でオーバーレイを切り替えます。スクリーンショットを撮るときや、授業で画面をすっきり見せたいときに便利です。

### 液体情報を切り替える

テンキー `2` で液体表示を切り替えます。タンクや液体ブロックを扱うMOD環境では、液体名・量を確認する用途で使います。

### レシピ・使用先を開く

レシピ表示MODを入れている場合、対象ブロックやアイテムを見ながらテンキー `3` でレシピ、テンキー `4` で使用先を開けます。

## 注意点

- ノートPCなどテンキーがない環境では、キー設定画面から別のキーに変更するのがおすすめです。
- 情報表示内容は、Jade本体だけでなく連携先MODの対応状況にも依存します。
- サーバーで表示制限が設定されている場合、クライアント側だけでは表示できない情報があります。
