# Xaero's World Map キー割り当てと操作方法

## MOD概要

Xaero's World Map は、探索済みエリアを大きなワールドマップとして表示する地図MODです。ウェイポイント管理、マップ切り替え、PNG書き出し、サーバー別マップ管理などができます。

## 解析元

- jar: `xaeroworldmap-neoforge-26.1.2-1.40.16.jar`
- modId: `xaeroworldmap`
- loader: NeoForge
- 主な解析対象: `xaero/map/controls/ControlsRegister.class`, `xaero/map/controls/ControlsHandler.class`, `assets/xaeroworldmap/lang/*.json`

## キー割り当て

`ControlsRegister` の登録処理から、次の初期キーを確認しました。

| キー | 機能 |
| --- | --- |
| `M` | ワールドマップを開く |
| `]` | ワールドマップ設定を開く |
| 未割り当て | サーバー設定を開く |
| 未割り当て | マップ拡大 |
| 未割り当て | マップ縮小 |
| `右Shift` | クイック確認 |
| 未割り当て | ディメンション表示切り替え |
| 未割り当て | 追跡中プレイヤー表示切り替え |
| 未割り当て | PACチャンククレーム表示切り替え |

`追跡中プレイヤー` と `PACチャンククレーム` は、連携MODやサーバー機能がある場合に意味を持つ項目です。

## 独自コマンド

Xaero's World Map 固有のスラッシュコマンドは検出されませんでした。

## 操作方法

### マップを開く

`M` を押すとワールドマップが開きます。探索済みの地形が表示され、拠点や遠征先の位置関係を確認できます。

### マップを移動・拡大縮小する

| 操作 | 内容 |
| --- | --- |
| 左クリックドラッグ | マップをスクロール |
| マウスホイール | 拡大・縮小 |
| `Ctrl` を押しながらホイール | より細かくズーム調整 |
| 右クリック | マップ上のショートカットメニュー |
| マップ上の要素を右クリック | ウェイポイントなど対象ごとのオプション |

探索範囲が広いときは、ホイールで大きく縮小して全体を見てから、目的地周辺を拡大すると扱いやすいです。

### ウェイポイントを使う

マップ画面からウェイポイント画面を開き、地点の確認・作成・編集を行います。Xaero's Minimap と一緒に使うと、ミニマップ側のウェイポイントとも連携しやすくなります。

### マップ設定を開く

`]` キーでワールドマップ設定を開きます。表示、ズーム、マップ保存、地形色、ウェイポイント表示などを調整します。

### マップを切り替える

ワールドやサーバーが変わると、別マップとして扱われる場合があります。マップ選択画面から、作成済みマップの選択、名前変更、削除ができます。

### PNGとして書き出す

マップ画面のエクスポート機能から、探索済みマップをPNG画像として書き出せます。授業資料やサーバー案内用に地図画像を作るときに便利です。

### ディメンションを切り替える

ディメンション表示切り替えキーは未割り当てです。必要な場合はキー設定で割り当てます。`ControlsHandler` では、Shift状態を見て切り替え方向を変える処理が確認できます。

## 注意点

- 初期状態では `M` と `]` が重要です。ノートPCや日本語配列で `]` が押しづらい場合はキー設定で変更してください。
- マップデータはクライアント側に保存されます。別PCでは同じマップが見えない場合があります。
- サーバー設定や連携MODによって、プレイヤー表示・チャンククレーム表示などの項目は使えない場合があります。
