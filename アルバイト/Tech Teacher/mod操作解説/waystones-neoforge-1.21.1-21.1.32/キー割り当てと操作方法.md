# Waystones キー割り当てと操作方法

## MOD概要

Waystones は、設置したウェイストーン間をワープできるファストトラベルMODです。拠点、村、ダンジョン、遠征先などを登録して、移動時間を大きく短縮できます。

## 解析元

- jar: `waystones-neoforge-1.21.1-21.1.32.jar`
- modId: `waystones`
- loader: NeoForge
- 主な解析対象: `META-INF/neoforge.mods.toml`, `net/blay09/mods/waystones/command/ModCommands.class`, `assets/waystones/lang/*.json`

## キー割り当て

Waystones 専用のキー割り当ては検出されませんでした。

基本操作は、ウェイストーン本体やワープ系アイテムを右クリックして行います。

## 独自コマンド

`ModCommands` から `/waystones` ルートの管理コマンドを確認しました。多くは権限レベル2相当の管理者向けです。

| コマンド | 内容 |
| --- | --- |
| `/waystones activate <targets> <pos>` | 指定プレイヤーに指定位置のウェイストーンを登録 |
| `/waystones forget <targets> <pos>` | 指定位置のウェイストーン登録を解除 |
| `/waystones forget <targets> all` | 全ウェイストーン登録を解除 |
| `/waystones place <pos> <style> <name>` | 指定位置にウェイストーンを設置 |
| `/waystones count <player>` | プレイヤーの登録数を確認 |
| `/waystones list <player>` | 登録済みウェイストーン一覧を表示 |
| `/waystones list <player> all` | 追加情報付きで一覧表示 |
| `/waystones gui <player>` | 指定プレイヤーのウェイストーンGUIを開く |
| `/waystones cooldown <targets> reset all` | クールダウンを全リセット |
| `/waystones cooldown <targets> reset <identifier>` | 指定クールダウンをリセット |

## 操作方法

### ウェイストーンを登録する

1. ウェイストーンを設置します。
2. ウェイストーンを右クリックします。
3. 名前や登録状態を確認します。
4. 登録済みのウェイストーンがワープ先一覧に表示されるようになります。

村やダンジョンに自然生成される設定の場合は、見つけたウェイストーンを右クリックして有効化します。

### ワープする

登録済みウェイストーンを右クリックすると、ワープ先一覧が開きます。移動したい場所を選ぶとテレポートします。

設定によっては、ワープに経験値、アイテム、クールダウンが必要になる場合があります。

### グローバルウェイストーンを使う

グローバル設定のウェイストーンは、プレイヤーが個別に登録していなくても一覧に出るタイプです。サーバーの共通拠点、ロビー、主要都市などに向いています。

### Sharestoneを使う

Sharestone は色ごとに接続先が分かれる共有ワープ装置です。同じ色のSharestone同士をネットワークのように扱います。

チームごと、エリアごと、用途ごとに色を分けると整理しやすくなります。

### Portstoneを使う

Portstone は一方通行寄りのワープ装置として扱われます。通常のウェイストーンのように「ワープ先として戻る」用途には制限があるため、脱出地点やイベント用の移動先に向いています。

### Warp Plateを使う

Warp Plate は踏む・起動することでワープするプレート型の装置です。Attuned Shard、Dormant Shard、Deepslate Shard などの関連アイテムで行き先や接続を調整する仕組みが含まれています。

### Warp StoneやScrollを使う

Warp Stone やワープスクロール系アイテムを使うと、ウェイストーン本体から離れていてもワープ機能を使えます。探索先から拠点へ戻る用途で便利です。

## 注意点

- サーバー設定によって、ワープ費用・クールダウン・次元間移動・未発見地点への移動可否が変わります。
- コマンドは管理者向けです。通常プレイヤーは右クリック操作とワープアイテムを覚えれば十分です。
- 拠点用、村用、ダンジョン用など、名前を分かりやすく付けるとワープ先一覧が整理しやすくなります。
