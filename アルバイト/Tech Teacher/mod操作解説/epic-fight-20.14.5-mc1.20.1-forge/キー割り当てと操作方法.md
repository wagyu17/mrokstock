# Epic Fight 系 mod キー割り当てと操作方法

## 解析元

- `epic-fight-20.14.5-mc1.20.1-forge.jar`
- `epic-fight-invincible-lib-20.14.7.5-mc1.20.1-forge.jar`
- `sword_soaring-20.14.2.4-mc1.20.1-forge.jar`

キー登録クラス、入力処理、言語ファイルを逆コンパイルして確認した内容です。

## Epic Fight 本体

### キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| 左 `Shift` | 武器の固有スキルのツールチップを表示 | GUI 上で武器固有スキルの説明を見るためのキーです。 |
| `K` | スキルエディターを開く | 習得済みスキルの確認・装備画面を開きます。 |
| 未割り当て | 設定画面を開く | 必要ならキー設定から割り当てます。 |
| 左 `Alt` | 回避スキル | 戦闘モード中に回避スキルを使います。 |
| 右クリック | ガード | ガード可能な状態・スキルで防御します。 |
| 左クリック | 攻撃 | 戦闘モード中の通常攻撃です。 |
| 左クリック長押し | 武器の固有スキル | 攻撃キーと同じ物理入力です。一定時間押し続けると固有スキル側として処理されます。 |
| `Space` | モビリティスキル | 装備中の移動系スキルを使います。 |
| `R` | 戦闘/採掘モードの切り替え | Epic Fight の戦闘モードと採掘モードを切り替えます。 |
| `G` | ロックオン | 対象をロックオンします。 |
| `←` | ロックオン対象を左へ切り替え | ロックオン中の対象切り替えに使います。 |
| `→` | ロックオン対象を右へ切り替え | ロックオン中の対象切り替えに使います。 |
| 右クリック | ロックオン自由切り替え | カメラ/ロックオン操作用にも右クリックが登録されています。 |
| 未割り当て | バニラモデルデバッグ切り替え | デバッグ用です。通常プレイでは使いません。 |

### 基本操作

1. `R` で戦闘モードに切り替えます。
2. 左クリックで通常攻撃を行います。
3. 右クリックでガードします。
4. 左 `Alt` で回避します。
5. `Space` でモビリティスキルを使います。
6. `G` で敵をロックオンし、`←` / `→` で対象を切り替えます。
7. 武器固有スキルは、武器チャージやスキル条件を満たした状態で左クリックを長押しして発動します。

戦闘アニメーション中は、ホットバー切り替え、アイテムドロップ、オフハンド入れ替えなどが一時的にブロックされることがあります。これは Epic Fight 側の入力制御です。

## Invincible Lib

### キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| 左クリック | Primary Action | 主攻撃や主アクションとして扱われます。 |
| 右クリック | Secondary Action | ガード、照準、サブアクションとして扱われます。 |
| マウスボタン4 | Special Ability 1 | 対応スキルの特殊アクション1です。 |
| マウスボタン5 | Special Ability 2 | 対応スキルの特殊アクション2です。 |

### 操作方法

- Invincible Lib は単体の操作というより、Epic Fight のスキル入力を拡張するライブラリです。
- 対応する固有スキルを装備している場合、左クリック、右クリック、マウスボタン4/5、回避キー、武器固有スキルキーがコンボ入力として記録されます。
- `W/A/S/D` の押下状態もスキル条件として同期されます。方向入力で派生が変わるスキルがあるためです。

## Sword Soaring

### キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| `Space` 2回押し | 飛行開始 | Sword Soaring スキルを持ち、条件を満たしているときに飛行を開始します。 |
| `Space` 長押し | 加速 | 飛行中に加速します。 |
| `Tab` | 飛行モード切り替え | 習得済みの Sword Soaring 飛行スタイルを順に切り替えます。 |
| `Ctrl` + 右クリック | Sword Skill | Sword Controller 系の特殊剣技を発動します。 |
| マウスボタン5 | Sword Back | Vatansever を背中位置へ戻す要求を送ります。 |

### 操作方法

1. Epic Fight の戦闘モードに入ります。
2. Sword Soaring 系スキルを習得・装備します。
3. `Space` を素早く2回押して飛行を開始します。
4. 飛行中は `Space` 長押しで加速します。
5. `Tab` で飛行スタイルを切り替えます。
6. Sword Controller 系スキルを装備している場合、`Ctrl` + 右クリックで剣技を発動します。

## キー競合の注意

- `Space` はバニラジャンプ、Epic Fight のモビリティスキル、Sword Soaring の飛行開始/加速で重なります。挙動は状態と装備スキルによって分岐します。
- 右クリックは Epic Fight のガード、Invincible Lib の Secondary Action、Sword Soaring の `Ctrl` + 右クリックと重なります。
- 左クリックは通常攻撃、武器固有スキル、Invincible Lib の Primary Action が同じ物理入力を共有します。短押しと長押しで処理が変わります。
- 左 `Alt` は Epic Fight の回避と Create のツールメニューが衝突します。同時導入する場合はどちらかを別キーにするのがおすすめです。
- `R` は Epic Fight の戦闘/採掘モード切り替えとして重要ですが、他の戦闘 mod や工業 mod でも使われやすいキーです。
- `G` は Epic Fight のロックオン、Mekanism の胴装備モード、他 mod の特殊操作と競合しやすいです。
