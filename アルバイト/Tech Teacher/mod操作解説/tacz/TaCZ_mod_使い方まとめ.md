# TaCZ mod 使い方まとめ

対象ファイル: `tacz-1.20.1-1.1.8-release.jar`

この mod は **Timeless & Classics Guns: Zero / TaCZ** です。Minecraft に銃、弾薬、アタッチメント、ガンスミステーブルなどを追加します。

## 概要

TaCZ は、銃をクラフトして、弾薬を装填し、スコープやマズルなどのアタッチメントでカスタマイズしながら使う銃器 mod です。

jar 内のデフォルトパックには、ガンスミステーブルのクラフトレシピ基準で以下の要素が含まれています。

| 種類 | 数 |
|---|---:|
| 銃 | 53種類 |
| 弾薬 | 24種類 |
| アタッチメント | 95種類 |

内部の定義ファイルまで含めた装着対応は、別ファイルの [TaCZ_アタッチメント対応表.md](TaCZ_アタッチメント対応表.md) にまとめています。

## デフォルトキー割り当て

ゲーム内の「設定 > 操作設定 > Timeless & Classics Guns: Zero」から変更できます。

| 操作       |   デフォルトキー | 説明                                |
| -------- | --------: | --------------------------------- |
| 射撃       |     左クリック | 銃を撃つ                              |
| 狙う / ADS |     右クリック | 銃を構えて照準を合わせる                      |
| リロード     |       `R` | 銃に弾薬を装填する                         |
| 射撃モード切替  |       `G` | セミ、バースト、フルオートなどを切り替える             |
| 武器を眺める   |       `H` | 点検・鑑賞アニメーションを再生する                 |
| カスタマイズ   |       `Z` | 銃のアタッチメント編集画面を開く                  |
| インタラクト   |       `O` | 銃を持ったままブロックやエンティティに触る             |
| 近接攻撃     |       `V` | ADSしていない時に近接攻撃する                  |
| ズーム      |       `V` | ADS中にスコープ倍率を切り替える                 |
| 伏せる      |       `C` | 対応銃を持って伏せる                        |
| 設定画面     | `Alt + T` | Cloth Config API がある場合に mod 設定を開く |

`V` は近接攻撃とズームで共通です。ADS中はズーム、ADSしていない時は近接攻撃として動きます。

## 初期設定の挙動

| 設定 | 初期値 | 内容 |
|---|---:|---|
| 長押しで狙う | 有効 | 右クリックを押している間だけADSする |
| ホールドして伏せる | 有効 | `C` を押している間だけ伏せる |
| 自動リロード | 無効 | 弾切れ時に自動リロードしない |
| 伏せ機能 | 有効 | サーバー設定で伏せが許可されている |

設定画面をゲーム内で開くには、別 mod の **Cloth Config API** が必要です。入っていない場合、`Alt + T` を押すと導入案内が表示されます。

## 基本の遊び方

1. ガンスミステーブルを作る
2. ガンスミステーブルで銃、弾薬、アタッチメントを作る
3. 銃と対応弾薬をインベントリに入れる
4. 銃を手に持って `R` でリロードする
5. 右クリックで狙う
6. 左クリックで撃つ
7. 必要に応じて `G` で射撃モードを切り替える
8. `Z` でアタッチメントを付け替える

## ガンスミステーブル

TaCZ の中心になる作業台です。銃、弾薬、アタッチメントの作成に使います。

### レシピ

| 素材 | 数 |
|---|---:|
| 原木タグのブロック | 3 |
| 鉄インゴット | 4 |
| 鉄ブロック | 1 |

クラフト形状:

```text
LLL
IBI
I I
```

`L` は原木、`I` は鉄インゴット、`B` は鉄ブロックです。

## 銃の使い方

銃を手に持つと、通常の右クリック操作が ADS に使われます。

| 操作 | 方法 |
|---|---|
| 撃つ | 左クリック |
| 狙う | 右クリック長押し |
| リロード | `R` |
| 射撃モード変更 | `G` |
| 武器を見る | `H` |
| カスタマイズ | `Z` |

弾薬がない、または銃に対応していない弾薬しか持っていない場合はリロードできません。

## アタッチメント

銃を持って `Z` を押すと、カスタマイズ画面を開けます。

どのアタッチメントがどの銃に付くかは、[TaCZ_アタッチメント対応表.md](TaCZ_アタッチメント対応表.md) を参照してください。

取り付けできる主なアタッチメント:

| 種類 | 内容 |
|---|---|
| スコープ / サイト | 照準器、ズーム倍率の変更 |
| マズル | サプレッサー、ブレーキなど |
| ストック | 反動や操作性の調整 |
| グリップ | 腰撃ち精度や反動の調整 |
| レーザー | 照準補助 |
| 拡張マガジン | 装弾数を増やす |

カスタマイズ画面では、ダメージ、RPM、装弾数、反動、ADS、弾速、重量などの性能も確認できます。

## スコープとズーム

スコープを付けた銃で右クリックして ADS した状態で `V` を押すと、ズーム倍率を切り替えられます。

ADSしていない状態で `V` を押すと近接攻撃になるため、ズームしたい時は必ず右クリックで構えてから押します。

## インタラクトキー

銃を持っていると右クリックが ADS に使われるため、チェストや作業台などを普通に開きづらい場合があります。

その時は `O` を押します。対象に照準を合わせた状態で `O` を押すと、銃を持ったままインタラクトできます。

主な対応対象:

| 種類 | 例 |
|---|---|
| ブロック | チェスト、作業台、かまど、金床、ベッド、ドア、レバー、ガンスミステーブルなど |
| エンティティ | 村人、行商人、馬、ラバ、ボート、トロッコ、額縁など |

画面中央に「キーを押してインタラクト」と表示される場合があります。

## 伏せ

`C` を押すと伏せられます。

初期設定では `C` を押している間だけ伏せます。銃側が伏せに対応していない場合や、サーバー設定で伏せが無効の場合は使えません。

## 弾薬箱

弾薬箱は弾薬をまとめて管理するアイテムです。

| 操作 | 方法 |
|---|---|
| 弾薬を入れる | インベントリ内で弾薬箱を持ち上げ、弾薬にカーソルを合わせて右クリック |
| 弾薬を取り出す | 弾薬箱を持ち、空スロットを右クリック |
| クリエイティブ用弾薬箱 | 指定タイプの弾薬を無限供給できる |
| 全種クリエイティブ弾薬箱 | 全種類の弾薬を無限供給できる |

## よく使う流れ

### 銃を作って撃つ

1. ガンスミステーブルを設置する
2. 銃を作る
3. その銃に対応する弾薬を作る
4. 銃と弾薬をインベントリに入れる
5. 銃を持って `R` でリロードする
6. 右クリックで狙い、左クリックで撃つ

### アタッチメントを付ける

1. ガンスミステーブルでアタッチメントを作る
2. 銃を持つ
3. `Z` を押す
4. 対応スロットにアタッチメントを装着する
5. 性能表示を見ながら調整する

### チェストや作業台を開く

1. 銃を持ったまま対象に照準を合わせる
2. `O` を押す
3. 通常の右クリック操作の代わりにインタラクトする

## 困った時

| 症状 | 確認すること |
|---|---|
| 撃てない | 弾薬が入っているか、対応弾薬を持っているか確認する |
| リロードできない | 銃と弾薬の種類が合っているか確認する |
| 右クリックでチェストを開けない | `O` のインタラクトキーを使う |
| ズームできない | ADS中に `V` を押しているか、スコープが付いているか確認する |
| `Alt + T` で設定が開けない | Cloth Config API が導入されているか確認する |
| 伏せられない | 銃が伏せ対応か、サーバー設定で伏せが有効か確認する |
| キーが競合する | Minecraft の操作設定から TaCZ のキーを変更する |

## 補足

TaCZ のデフォルト銃パックは、ゲーム起動時に上書きされる場合があります。銃パックを編集する場合は、`.minecraft/tacz` 内の `tacz-pre.toml` で `DefaultPackDebug` を有効にする必要があります。上書きされた場合、バックアップは `.minecraft/tacz_backup` に保存される可能性があります。
