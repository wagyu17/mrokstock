# TaCZ キー割り当てと操作方法

## 解析元

- `tacz-1.20.1-1.1.8-release.jar`

`com.tacz.guns.client.input` 以下のキー登録と、`KeyConfig` の初期設定を逆コンパイルして確認した内容です。

## キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| 左クリック | 射撃 | 銃を持った状態で射撃します。 |
| 右クリック | 狙う | デフォルトでは押している間だけ ADS / エイムします。 |
| `R` | リロード | 弾薬がある場合にリロードします。 |
| `G` | 射撃モード切替 | セミオート、フルオートなど、対応銃の射撃モードを切り替えます。 |
| `H` | 武器を眺める | 銃の確認・インスペクト動作を行います。 |
| `Z` | カスタマイズ | 銃のアタッチメント変更・改造画面を開きます。 |
| `O` | 銃を持ちながらインタラクト | 銃を構えたままブロックやエンティティに干渉します。 |
| `V` | 近接 | 通常時は近接攻撃を行います。 |
| `V` | ズーム | エイム中はズーム操作として使われます。 |
| `C` | 伏せる | デフォルトでは押している間だけ伏せます。 |
| `Alt` + `T` | コンフィグを開く | TaCZ のクライアント設定画面を開きます。 |

## 基本操作

1. 銃と対応する弾薬を用意します。
2. `R` でリロードします。
3. 左クリックで射撃します。
4. 右クリックを押している間、ADS / エイム状態になります。
5. 対応銃では `G` で射撃モードを切り替えます。
6. `Z` でカスタマイズ画面を開き、アタッチメントを変更します。
7. 右クリックがエイムに使われるため、銃を持ったまま通常の干渉をしたい場合は `O` を使います。

## 近接とズーム

- `V` は通常時には近接攻撃です。
- エイム中に `V` を押すとズーム操作になります。
- 同じキーに 2 つの機能が割り当てられていますが、状態によって処理が分岐します。

## 伏せ操作

- `C` はデフォルトで長押し式です。
- クライアント設定の `HoldToCrawl` が有効なため、押している間だけ伏せます。
- サーバー側で伏せが無効化されている場合や、銃が伏せ操作に対応していない場合は動作しません。

## 設定で変わる主な挙動

| 設定 | 初期値 | 内容 |
| --- | --- | --- |
| `HoldToAim` | `true` | 右クリックを押している間だけエイムします。 |
| `HoldToCrawl` | `true` | `C` を押している間だけ伏せます。 |
| `AutoReload` | `false` | 弾切れ時の自動リロードは初期状態では無効です。 |

## 注意点

- `R`、`G`、`H`、`Z`、`V`、`C` は他 mod と競合しやすいです。
- JujutsuCraft 系 mod と同時に使う場合、`R`、`G`、`Z`、`V`、`C` が特に重なります。
- Mahou Tsukai と同時に使う場合、`H` が Selective Displacement と競合します。
