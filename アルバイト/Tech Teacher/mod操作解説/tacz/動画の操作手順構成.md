# TaCZ 動画の操作手順構成

対象: 海外向けの TaCZ / Timeless and Classics Guns: Zero 解説動画  
目的: 視聴者に「自分でもできそう」と感じてもらい、長尺動画・PDF販売ページへ誘導する。

## 基本方針

- 画面上にキーボード・マウス overlay を表示する。
- 1つの場面で見せる操作は1つに絞る。
- 操作キーを押した瞬間が分かるように、overlay が光るタイミングを見せる。
- 英語字幕は短くする。
- まず結果を見せ、そのあと操作手順を見せる。
- 右クリックが通常インタラクトではなく ADS / Aim になるため、`O` キーの重要性を必ず見せる。

## Shorts 用構成 1本目

テーマ: AWM sniper in TaCZ Minecraft

想定尺: 25〜35秒

### 1. Hook

尺: 0〜3秒

見せる内容:

- AWM を構えた状態から、遠くのターゲットを一撃で撃つ。
- 画面上に overlay を出し、右クリックと左クリックが光る瞬間を見せる。

字幕例:

```text
AWM in Minecraft?
```

操作:

1. 右クリック長押しで Aim / ADS
2. 左クリックで Fire

### 2. Reload

尺: 3〜7秒

見せる内容:

- 撃った直後に `R` を押してリロードする。
- リロードアニメーションを短く見せる。

字幕例:

```text
Press R to reload
```

操作:

1. `R` を押す

### 3. Aim And Zoom

尺: 7〜13秒

見せる内容:

- 右クリックでスコープを覗く。
- スコープ装着時に `V` でズーム切り替えを見せる。

字幕例:

```text
Right click to aim
V changes scope zoom
```

操作:

1. 右クリック長押し
2. ADS 中に `V`

### 4. Customize

尺: 13〜20秒

見せる内容:

- `Z` を押してカスタマイズ画面を開く。
- スコープ・マズル・マガジンなどのスロットが見える場面を短く入れる。

字幕例:

```text
Press Z to customize attachments
```

操作:

1. 銃を持った状態で `Z`
2. スコープまたはマズルを選ぶ

### 5. Inspect

尺: 20〜25秒

見せる内容:

- `H` を押して武器確認アニメーションを見せる。
- 見栄えの良いシーンとして使う。

字幕例:

```text
H = inspect
```

操作:

1. `H`

### 6. Call To Action

尺: 25〜35秒

見せる内容:

- 最後にもう一度射撃シーン。
- 画面下に長尺動画またはPDFへの誘導文を出す。

字幕例:

```text
Full controls guide in the description
```

操作:

1. 右クリック長押し
2. 左クリック

## 長尺動画用構成

想定尺: 5〜8分

### 1. Opening Result

尺: 0〜20秒

見せる内容:

- AWM または分かりやすい銃で撃つ。
- リロード、ADS、射撃、Inspect をテンポ良く見せる。

目的:

- 最初に「この動画で何ができるようになるか」を見せる。

### 2. What Is TaCZ

尺: 20〜45秒

説明内容:

- TaCZ は Minecraft にリアルな銃、弾薬、アタッチメント、カスタマイズ要素を追加する mod。
- 銃ごとに使えるアタッチメントが違う。

字幕例:

```text
TaCZ adds guns, ammo, attachments, and weapon customization.
```

### 3. Required Setup

尺: 45〜75秒

見せる内容:

- mods フォルダに TaCZ を入れる場面は短く。
- バージョンが合っていることを確認する。
- `Alt + T` の設定画面には Cloth Config API が関係することを説明する。

注意:

- mod jar の再配布はしない。
- ダウンロードは公式ページから行うと説明する。

### 4. Basic Controls

尺: 75〜150秒

見せる順番:

1. 左クリック: Fire
2. 右クリック: Aim / ADS
3. `R`: Reload
4. `G`: Fire Mode
5. `Z`: Customize
6. `H`: Inspect
7. `O`: Interact
8. `V`: Melee / Scope Zoom
9. `C`: Crawl

ポイント:

- overlay を必ず表示する。
- キーを押す前に字幕でキー名を出す。
- キーを押した瞬間に overlay が光るようにする。

### 5. First Gun Workflow

尺: 150〜240秒

見せる順番:

1. Gunsmith Table を開く
2. 銃を作る
3. 対応する弾薬を作る
4. 銃と弾薬をインベントリに入れる
5. `R` でリロード
6. 右クリックで Aim
7. 左クリックで Fire

字幕例:

```text
Gun + matching ammo = ready to reload
```

### 6. Attachment Workflow

尺: 240〜330秒

見せる順番:

1. `Z` でカスタマイズ画面を開く
2. スコープを付ける
3. マズルを付ける
4. 拡張マガジンを付ける
5. 右クリックで Aim
6. `V` でズーム切り替え
7. 射撃して違いを見せる

ポイント:

- 銃によって付けられるアタッチメントが違うことを説明する。
- PDFには銃ごとの対応表がある、と自然に誘導する。

字幕例:

```text
Each gun supports different attachments.
```

### 7. Important Key: O Interact

尺: 330〜390秒

見せる内容:

- 銃を持ったまま右クリックすると Aim になる。
- チェストや Gunsmith Table を開くには `O` を使う。

見せる順番:

1. 銃を持つ
2. チェストに向かって右クリックし、Aim になることを見せる
3. `O` を押してチェストを開く

字幕例:

```text
Use O to interact while holding a gun
```

### 8. Common Problems

尺: 390〜480秒

見せる内容:

- 撃てない
- リロードできない
- スコープズームできない
- チェストを開けない
- 設定画面が開かない

説明順:

1. 撃てない: 弾薬とリロードを確認
2. リロードできない: 対応弾薬を確認
3. ズームできない: スコープ装着と ADS 状態を確認
4. チェストを開けない: `O` を使う
5. `Alt + T` が開かない: Cloth Config API を確認

### 9. PDF CTA

尺: 最後の20〜30秒

見せる内容:

- PDF の表紙または販売ページを短く表示する。
- PDFに含まれる内容を3つだけ言う。

字幕例:

```text
The PDF includes controls, beginner steps, and attachment compatibility tables.
```

言う内容:

1. Basic controls
2. Step-by-step beginner workflow
3. Gun and attachment compatibility table

## 撮影順チェックリスト

実際に録画するときは、動画の順番通りではなく、以下の素材を先に撮ってから編集で並べる。

2. リロードシーン
3. ADS / スコープズームシーン
4. `Z` カスタマイズ画面
5. `H` Inspect シーン
6. `O` でチェストや Gunsmith Table を開くシーン
7. Gunsmith Table で銃・弾薬を作るシーン
8. アタッチメントを付け替えるシーン
9. 撃てない・リロードできないなどの失敗例
10. PDF販売ページまたはPDF表紙の画面
