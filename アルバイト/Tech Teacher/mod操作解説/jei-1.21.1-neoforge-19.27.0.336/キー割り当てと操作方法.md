# JEI キー割り当て・独自コマンド・操作方法

## MOD概要

JEI は、アイテム一覧・レシピ検索・用途確認をインベントリ画面に追加する、レシピ確認用の定番MODです。

## 解析元

- 対象ファイル: `jei-1.21.1-neoforge-19.27.0.336.jar`
- MOD ID: `jei`
- バージョン: `19.27.0.336`
- 表示名: `Just Enough Items`
- 解析方法: jar 内の `META-INF/neoforge.mods.toml`、lang、`InternalKeyMappings`、GUI/command 関連 class を確認

## キー割り当て

JEI は多数のキー設定を追加します。キー設定画面では、JEI 関連のカテゴリから変更できます。

### オーバーレイ・検索

| 操作 | デフォルトキー | 内容 |
| --- | --- | --- |
| JEIの表示 / 非表示 | Ctrl + O | インベントリ右側のアイテム一覧オーバーレイを表示・非表示にします。Mac環境では Command + O 相当です。 |
| 検索バーの選択 | Ctrl + F | JEI検索バーにフォーカスします。Mac環境では Command + F 相当です。 |
| 前のページ | 未割り当て | アイテム一覧の前ページへ移動します。 |
| 次のページ | 未割り当て | アイテム一覧の次ページへ移動します。 |
| ブックマーク一覧の表示 / 非表示 | 未割り当て | 左側のブックマーク欄を表示・非表示にします。 |

### アイテム上での操作

| 操作 | デフォルトキー | 内容 |
| --- | --- | --- |
| レシピの表示 | R | カーソルを合わせたアイテムの作り方を表示します。 |
| レシピの表示 | 左クリック | JEIリスト上のアイテムを左クリックしてレシピを表示します。 |
| 用途の表示 | U | カーソルを合わせたアイテムが何に使えるかを表示します。 |
| 用途の表示 | 右クリック | JEIリスト上のアイテムを右クリックして用途を表示します。 |
| ブックマーク追加 / 削除 | A | カーソルを合わせたアイテムをブックマークに追加、または削除します。 |

### 検索バー

| 操作 | デフォルトキー | 内容 |
| --- | --- | --- |
| 検索バーを消去 | 右クリック | 検索バー上で右クリックすると検索文字を消します。 |
| 前の検索履歴 | ↑ | 過去の検索履歴に戻ります。 |
| 次の検索履歴 | ↓ | 次の検索履歴に進みます。 |

### ブックマークしたレシピ

| 操作 | デフォルトキー | 内容 |
| --- | --- | --- |
| ブックマークしたレシピでクラフト | Shift + 左クリック | ブックマークしたレシピから1回分のクラフト転送を行います。 |
| ブックマークしたレシピで最大クラフト | Ctrl + 左クリック | 作れるだけまとめてクラフト転送します。Mac環境では Command + 左クリック相当です。 |

### レシピ画面

| 操作 | デフォルトキー | 内容 |
| --- | --- | --- |
| 前のレシピに戻る | Backspace | レシピ画面の履歴を1つ戻ります。 |
| 前のレシピページ | Page Up | 同じカテゴリ内の前ページへ移動します。 |
| 次のレシピページ | Page Down | 同じカテゴリ内の次ページへ移動します。 |
| 前のカテゴリ | Shift + Page Up | 前のレシピカテゴリへ移動します。 |
| 次のカテゴリ | Shift + Page Down | 次のレシピカテゴリへ移動します。 |
| レシピ画面を閉じる | Esc | レシピ画面を閉じます。 |
| レシピIDをコピー | 未割り当て | レシピIDをクリップボードにコピーします。必要ならキー設定で割り当てます。 |

### チートモード

| 操作 | デフォルトキー | 内容 |
| --- | --- | --- |
| チートモード切り替え | 未割り当て | JEIのチートモードを切り替えます。キー設定で割り当て可能です。 |
| アイテムを1個入手 | 左クリック | チートモード中、JEIリストのアイテムを1個入手します。 |
| アイテムを1個入手 | 右クリック | チートモード中、JEIリストのアイテムを1個入手します。 |
| アイテムを1スタック入手 | Shift + 左クリック | チートモード中、1スタック入手します。 |
| アイテムを1スタック入手 | ホイールクリック | チートモード中、1スタック入手します。 |
| チートモードボタン切り替え | Ctrl + 左クリック | JEI設定ボタン上で使うチートモード切り替え操作です。Mac環境では Command + 左クリック相当です。 |

### 編集・非表示モード

| 操作 | デフォルトキー | 内容 |
| --- | --- | --- |
| 編集モード切り替え | 未割り当て | アイテムの非表示設定を編集するモードを切り替えます。 |
| アイテムを非表示 / 表示 | Ctrl + 左クリック | 編集モード中、対象アイテムを非表示または表示にします。 |
| ワイルドカードで非表示 / 表示 | Ctrl + 右クリック | 同種アイテムをまとめて非表示または表示にします。 |

## 独自コマンド

JEI 独自のスラッシュコマンドはありません。

jar 内を確認した限り、JEI は独自コマンドを登録していません。チートモードでアイテムを出す機能は、サーバー側の権限やバニラの `/give` 実行権限を確認して動作します。そのため、サバイバルサーバーでは権限がないとアイテム入手機能は使えません。

## 基本操作

### レシピを見る

1. インベントリを開きます。
2. 右側のJEIアイテム一覧、またはインベントリ内のアイテムにカーソルを合わせます。
3. `R` キー、または左クリックでレシピを表示します。

例:

- Iron Ingot に `R`: 鉄インゴットの作り方を表示します。
- Bread に `R`: パンのクラフトレシピを表示します。

### 用途を見る

1. 調べたいアイテムにカーソルを合わせます。
2. `U` キー、または右クリックで用途を表示します。

例:

- Wheat に `U`: パン、ケーキ、繁殖など、Wheat を使うレシピや用途を表示します。
- Plank に `U`: 作業台、棒、チェストなど、板材から作れるものを確認できます。

### 検索する

検索バーに文字を入力すると、右側のアイテム一覧が絞り込まれます。

便利な検索記号:

| 記号 | 検索対象 | 例 |
| --- | --- | --- |
| `@` | MOD名 | `@create`、`@jei` |
| `#` | タグ | `#forge:ingots` など |
| `$` | ツールチップ | `$attack` など |
| `^` | 色 | `^red` など |
| `&` | リソースID | `&minecraft:diamond` など |
| `%` | クリエイティブタブ | `%building` など |

検索操作:

- `Ctrl + F`: 検索バーに移動します。
- 検索バー上で右クリック: 検索文字を消します。
- `↑` / `↓`: 検索履歴を切り替えます。

### レシピ転送を使う

レシピ画面に表示される転送ボタンを使うと、必要な材料をクラフトグリッドへ自動配置できます。

基本の流れ:

1. 作りたいアイテムのレシピを開きます。
2. 必要な材料をインベントリに用意します。
3. レシピ転送ボタンを押します。
4. クラフト台や作業GUIに材料が自動配置されます。

注意点:

- 材料が足りないと転送できません。
- インベントリに空きがないと失敗する場合があります。
- サーバー側がJEIのレシピ転送に対応していない場合、転送できないことがあります。

### ブックマークを使う

よく使うアイテムやレシピはブックマークできます。

操作:

- アイテムにカーソルを合わせて `A`: ブックマーク追加 / 削除
- ブックマーク欄のアイテムをクリック: レシピや用途をすぐ確認
- Shift + 左クリック: ブックマークしたレシピを1回分転送
- Ctrl + 左クリック: 作れるだけまとめて転送

補足:

- 設定によっては、ブックマークをドラッグして並び替えられます。
- ブックマーク欄は左側に表示されます。

### チートモードを使う

チートモードは、JEIリストから直接アイテムを入手する機能です。

使い方:

1. JEI設定またはキー設定でチートモードを有効にします。
2. JEIリストのアイテムをクリックします。
3. 権限があれば、アイテムがインベントリに入ります。

クリック操作:

- 左クリック / 右クリック: 1個入手
- Shift + 左クリック: 1スタック入手
- ホイールクリック: 1スタック入手

注意点:

- シングルプレイでもチート許可やクリエイティブ権限が必要になる場合があります。
- マルチプレイでは、サーバーの権限設定に依存します。
- 権限がない場合、見た目上クリックできてもアイテムは入手できません。

### 編集・非表示モードを使う

編集モードは、JEIの一覧から表示したくないアイテムを隠すための機能です。

基本操作:

1. キー設定で編集モード切り替えキーを割り当てます。
2. 編集モードを有効にします。
3. 非表示にしたいアイテムを Ctrl + 左クリックします。
4. 同種アイテムをまとめて隠したい場合は Ctrl + 右クリックします。

用途:

- 使わないスポーンエッグを一覧から消す
- クリエイティブ専用アイテムを隠す
- 大量に追加されたデバッグ用アイテムを整理する

## JEI設定で確認するとよい項目

| 設定 | 内容 |
| --- | --- |
| アイテム一覧表示 | 右側のJEI一覧を表示するかどうかを切り替えます。 |
| ブックマーク表示 | 左側のブックマーク欄を表示するかどうかを切り替えます。 |
| 検索バー位置 | 検索バーを中央寄せにするかどうかを変更できます。 |
| チートモード | アイテム入手機能の有効化や挙動を設定します。 |
| 非表示アイテム表示 | 隠したアイテムを表示するかどうかを変更できます。 |
| タグ内容表示 | アイテムタグの中身を表示するかどうかを設定できます。 |
| ブックマーク並び替え | ブックマークをドラッグで並び替えられるかを設定できます。 |

## 使い方の例

### 作りたいアイテムの材料を調べる

1. インベントリを開きます。
2. 検索バーにアイテム名を入力します。
3. 右側の一覧からアイテムを左クリックします。
4. 表示されたレシピを確認します。

### 手持ちアイテムの使い道を調べる

1. インベントリ内のアイテムにカーソルを合わせます。
2. `U` キーを押します。
3. そのアイテムを使うレシピ一覧を確認します。

### MOD名でアイテムを絞り込む

1. 検索バーに `@` を入力します。
2. 続けてMOD名を入力します。
3. 例として `@create` と入力すると、Create MODのアイテムだけを探しやすくなります。

## 注意点

- `R` はレシピ、`U` は用途、`A` はブックマークです。この3つを覚えると基本操作はほぼ使えます。
- JEI自体に独自スラッシュコマンドはありません。
- チートモードは権限がない環境では機能しません。
- 未割り当てのキーは、Minecraft のキー設定から必要に応じて割り当ててください。
