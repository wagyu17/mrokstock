# Mouse Tweaks キー割り当てと操作方法

## MOD概要

Mouse Tweaks は、インベントリ内のアイテム移動をマウス操作で高速化する便利系MODです。右ドラッグ、左ドラッグ、マウスホイールで、アイテムの分配・移動・補充がしやすくなります。

## 解析元

- jar: `MouseTweaks-fabric-mc26.1-2.31.jar`
- modId: `mousetweaks`
- loader: Fabric
- 主な解析対象: `fabric.mod.json`, `yalter/mousetweaks/Config.class`, `assets/mousetweaks/lang/*.json`

## キー割り当て

Mouse Tweaks 専用のキー割り当ては検出されませんでした。

このMODはキーボードではなく、主にマウス操作を拡張します。

## 独自コマンド

Mouse Tweaks 固有のスラッシュコマンドは検出されませんでした。

## 操作方法

### 右クリックドラッグでアイテムを分配する

アイテムスタックを持った状態で、インベントリ内のスロット上を右クリックしながらドラッグします。通過したスロットにアイテムを1個ずつ置く操作を素早く行えます。

クラフトグリッドに素材を均等配置したいときに便利です。

### 左クリックドラッグで同種アイテムをまとめて移動する

左クリックを押しながら同じ種類のアイテム上をドラッグすると、同種アイテムをまとめて移動できます。

チェスト整理や、同じブロックを一気にインベントリへ移す場面で使います。

### アイテムを持っていない状態の左ドラッグ

アイテムを持っていない状態でも、左ドラッグ操作によって同種スタックの移動を補助します。チェストや作業台など、開いている画面によって挙動が変わるため、まずはチェスト内で試すと分かりやすいです。

### マウスホイールでアイテムを移動する

スロット上にカーソルを置き、マウスホイールを回すと、アイテムをインベントリ間で移動できます。

例えばチェストを開いているときにホイールを回すと、カーソル下のアイテムをプレイヤーインベントリとチェストの間で素早く移せます。

## 設定で確認できた初期値

`yalter.mousetweaks.Config` から、次の初期設定を確認しました。

| 設定 | 初期値 | 意味 |
| --- | --- | --- |
| `RMBTweak` | `1` | 右クリックドラッグ補助が有効 |
| `LMBTweakWithItem` | `1` | アイテムを持った左ドラッグ補助が有効 |
| `LMBTweakWithoutItem` | `1` | アイテムを持たない左ドラッグ補助が有効 |
| `WheelTweak` | `1` | マウスホイール移動が有効 |
| `WheelSearchOrder` | `1` | ホイール移動時の検索順は後ろから前 |
| `WheelScrollDirection` | `0` | ホイール方向は通常 |
| `ScrollItemScaling` | `0` | スクロール量は比例方式 |
| `Debug` | `0` | デバッグ表示は無効 |

## 注意点

- このjarは Fabric 版です。NeoForge環境では基本的に使えません。
- バニラのクリック操作を置き換えるのではなく、ドラッグやホイール操作を拡張するMODです。
- インベントリ画面を大きく変更するMODと併用すると、画面によって挙動が異なる場合があります。
