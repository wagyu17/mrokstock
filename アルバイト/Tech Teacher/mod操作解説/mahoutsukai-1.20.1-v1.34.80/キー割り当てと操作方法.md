# Mahou Tsukai キー割り当てと操作方法

## 解析元

- `mahoutsukai-1.20.1-v1.34.80.jar`

`stepsword.mahoutsukai.Keybinds` と、クライアント側のキー処理 `ManaClientEvents`、関連するガイド文を逆コンパイルして確認した内容です。

## キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| `M` | 魔法陣を描く | 出血状態のときに、足元または前方に魔法陣を描きます。 |
| `Y` | 魔術礼装切り替え | Mystic Code や Replica を持っているときに、切り替え用ホイールを開きます。 |
| `.` | Mahou Settings GUI | Mahou Tsukai の設定画面を開きます。 |
| `H` | Selective Displacement | Selective Displacement 発動中に、対象の入れ替え操作を行います。 |

## 魔法陣の基本操作

1. 短剣などで血を使い、出血状態を作ります。
2. 出血状態で `M` を押すと、魔法陣を描きます。
3. 魔法陣に粉末状の触媒などを右クリックで追加します。
4. 条件を満たした魔法陣を右クリックして、対応する魔法を発動します。

`M` は出血状態でないと魔法陣描画の packet が送られません。

## Mystic Code / Replica の操作

- Mystic Code は `Shift` + 右クリックでインベントリを開きます。
- Mystic Code を持った状態で `Y` を押すと、登録したスクロールの切り替えホイールを開きます。
- 選択中のスクロールは、左クリックまたは右クリックで使用します。
- Replica を持っている場合も `Y` で専用の切り替えホイールを開きます。

## Selective Displacement の操作

- Selective Displacement の効果が有効な間だけ `H` が意味を持ちます。
- 対象を見て `H` を短く押すと、自分と対象を入れ替えます。
- `H` を長押しして複数対象を選ぶと、選んだ対象同士を入れ替える操作になります。
- クールタイム中や対象が取れていない場合は発動しません。

## 注意点

- 言語ファイルには `fireNobu`、`gunDown`、`gunUp` などのキー名も含まれていますが、このバージョンの `Keybinds` では有効なキー登録として確認できませんでした。
- `M` は JujutsuCraft 系の反転術式や Kimetsu mod の痣発現と競合しやすいです。
- `H` は JujutsuCraftV の呪術スロットアクションや TaCZ の武器確認と競合しやすいです。
