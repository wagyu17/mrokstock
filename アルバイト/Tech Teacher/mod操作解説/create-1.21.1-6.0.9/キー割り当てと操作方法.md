## Create 本体

### キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| 左 `Alt` | ツールメニュー / 概略図オーバーレイのフォーカス | 概略図などのツール操作中に、画面上のツールメニューやホログラム操作をフォーカスします。 |
| 左 `Alt` | 近くの工具箱にアクセス | 近くにある工具箱へアクセスするキーとしても登録されています。状況によって上のツールメニュー操作と使い分けられます。 |
| 左 `Ctrl` | ツール操作の補助キー | キー設定画面には出ない固定キーです。概略図と羽根ペンの `Ctrl` + スクロールなど、Create の一部ツール操作に使われます。 |
| 未割り当て | ブロック回転メニュー | 必要ならキー設定から割り当てます。ブロックの回転メニューを開くためのキーです。 |

### 基本操作

- レンチは右クリックで機械を回転させます。スニークしながら使うと、対応する機械を撤去します。
- エンジニアのゴーグルは、装備中に見ている機械の応力、容量、内容物などを HUD に表示します。
- 概略図と羽根ペンは右クリックで2つの角を選択します。`Ctrl` + スクロールで選択範囲や空中位置を調整し、左クリックで保存します。
- 概略図は持っている間にホログラム配置用のオーバーレイを表示します。スニークしながら右クリックすると、座標入力画面を開きます。
- リンクコントローラーは右クリックでオン/オフ、スニーク + 右クリックで手動設定画面を開きます。レッドストーンリンク受信機に右クリックすると割り当てモードになります。

### Create の道具・機械の詳しい使い方

Create は「動力源で回転を作る」「シャフトや歯車で回転を運ぶ」「ベルトやファンでアイテムを動かす」「加工機械に接続する」という順番で組むと理解しやすいです。

#### 動力の基本

1. 水車、風車、蒸気エンジン、ハンドクランクなどで回転力を作ります。
2. 回転している面にシャフトを接続して、回転を直線方向へ伸ばします。
3. 歯車、ギアボックス、ベルトで回転方向や場所を変えます。
4. メカニカルプレス、ミキサー、ソー、ドリル、ファンなどの加工機械へ接続します。
5. 機械が止まる場合は、エンジニアのゴーグルで応力不足や過負荷を確認します。

#### シャフト

シャフトは Create の一番基本になる動力伝達ブロックです。

- 回転しているブロックの軸方向に接続すると、同じ方向へ回転を伝えます。
- 水車や風車の中心からシャフトを伸ばすと、近くの機械へ動力を渡せます。
- シャフト同士は一直線に並べるとそのままつながります。
- 別方向へ曲げたい場合は、歯車やギアボックスを挟みます。
- 見た目や当たり判定を整えたい場合は、ケース入りシャフトにできます。

よく使う形は `水車 -> シャフト -> メカニカルプレス`、または `水車 -> シャフト -> メカニカルベルト` です。

#### 歯車 / 大きな歯車 / ギアボックス

歯車は、シャフトの向きや回転速度を変えるために使います。

- 小さな歯車同士を横に噛み合わせると、回転方向が反転します。
- 大きな歯車と小さな歯車を噛み合わせると、回転速度を上げたり下げたりできます。
- ギアボックスは回転を90度曲げたいときに便利です。
- 垂直ギアボックスは、縦方向と横方向の動力をきれいにつなぐときに使います。
- クラッチはレッドストーン入力で動力を切断します。
- ギアシフトはレッドストーン入力で回転方向を反転させます。
- シーケンスギアシフトは、一定角度だけ回す、往復させるなどの決まった動作を作るときに使います。

「方向が逆で機械が思った向きに動かない」ときは、歯車を1つ挟むかギアシフトを使うと直せます。

#### シャフト + メカニカルベルト

メカニカルベルトは、2本のシャフトをセットで使うアイテム搬送装置です。

1. まずシャフトを2本、同じ向きで平行に置きます。水平ベルトでも縦ベルトでも、2本のシャフトの向きがそろっている必要があります。
2. メカニカルベルトを持って、片方のシャフトからもう片方のシャフトへベルトを張ります。
3. どちらかのシャフトに動力を入れると、ベルトが回ります。
4. ベルトの上に置かれたアイテムやエンティティは、ベルトの進行方向へ運ばれます。
5. 進行方向を変えたい場合は、入力しているシャフトの回転方向を反転させます。

ベルト単体では自動搬入・自動搬出まではできません。チェストから出す、チェストへ入れる、仕分ける、といった処理にはファンネル、トンネル、シュート、デポを組み合わせます。

よく使う形は `チェスト -> ファンネル -> ベルト -> ファンネル -> チェスト` です。

#### ファンネル / 真鍮ファンネル / トンネル

ファンネルは、チェストやベルトからアイテムを出し入れするための部品です。

- ベルトの横や上に付けると、ベルトへアイテムを出したり、ベルト上のアイテムを回収したりできます。
- チェストや樽に付けると、そのインベントリから搬出、またはインベントリへ搬入できます。
- 向きが違うと出し入れが逆になるので、レンチで向きを調整します。
- 真鍮ファンネルはフィルターや数量指定など、より細かい搬送制御に向いています。
- トンネルはベルト上のアイテムを分配したり、複数ラインをまとめたりするときに使います。

ベルトラインを作るときは、まずベルトで流れを作り、次にファンネルで「どこから出すか」「どこへ入れるか」を決めると組みやすいです。

#### シュート

シュートは、アイテムを縦方向に落としたり持ち上げたりするための搬送部品です。

- 下方向へは重力でアイテムを落とせます。
- インベントリの下に置くと、アイテムを下へ送る縦搬送として使えます。
- ベルトの終点や加工機械の出口から、下のチェストへ落とす用途に向いています。
- スマートシュートはフィルターや制御が必要な場面で使います。

#### デポ

デポは、アイテムを1個ずつ置いて加工するための台です。

- メカニカルプレスの下に置くと、板材や圧縮加工に使えます。
- メカニカルソー、デプロイヤー、ファン処理などの待機場所としても使えます。
- ベルトよりも「1個を確実に加工したい」場面に向いています。

#### メカニカルプレス

メカニカルプレスは、上から押しつぶす加工機です。

1. デポまたは鉢の真上にメカニカルプレスを置きます。
2. プレスにシャフトで回転を入力します。
3. デポにインゴットを置くと板材へ加工します。
4. 鉢を使うと圧縮や一部レシピの加工ができます。
5. ベルトでアイテムを流す場合は、プレスの下を通るようにベルトを配置します。

よく使う形は `水車 -> シャフト -> メカニカルプレス` と `ベルト -> プレス下 -> ベルト` です。

#### メカニカルミキサー / 鉢 / ブレイズバーナー

メカニカルミキサーは、鉢の中身を混ぜる加工機です。

1. 鉢を置き、その真上にメカニカルミキサーを置きます。
2. ミキサーへシャフトで動力を入れます。
3. 鉢に材料を入れると混合レシピが進みます。
4. 加熱が必要なレシピでは、鉢の下にブレイズバーナーを置きます。
5. ブレイズバーナーに燃料を与えると加熱状態になり、より高度なレシピが使えます。

基本セットは `ブレイズバーナー -> 鉢 -> メカニカルミキサー` の縦3段です。

#### 石臼 / 破砕ホイール

石臼と破砕ホイールは、鉱石や素材を粉砕する加工機です。

- 石臼は単体で使いやすい初期の粉砕機です。上からアイテムを入れ、回転を入力すると粉砕します。
- 破砕ホイールは2つを向かい合わせに置き、互いに逆向きに回すことで中央を通るアイテムを砕きます。
- 破砕ホイールはベルトやシュートと組み合わせると自動化しやすいです。

よく使う形は `チェスト -> ファンネル -> ベルト -> 破砕ホイール -> ベルト -> チェスト` です。

#### ケース入りファン（エンケーストファン）

ケース入りファンは、風でアイテムやエンティティを押したり、素材を加工したりします。

- シャフトで回転を入力すると風を出します。
- 回転方向によって、押し出しと吸い込みが変わります。
- 水を通した風は洗浄、溶岩や火を通した風は加熱系、特殊なブロックを通した風は変換系の加工に使えます。
- ベルト上やデポ上のアイテムに風を当てると、対応する加工が進みます。

洗浄ラインの例は `破砕済み鉱石 -> ベルト -> 水越しのファン -> 回収用ファンネル` です。

#### メカニカルソー / メカニカルドリル

メカニカルソーとメカニカルドリルは、設置したブロックや流れてくる素材を破壊・加工する機械です。

- メカニカルソーは原木の加工、木材処理、前方のブロック破壊に使います。
- メカニカルドリルは石や鉱石などの掘削に使います。
- どちらも回転方向と設置向きが重要なので、レンチで向きを調整します。
- からくりに取り付けると、自動採掘機や伐採機の先端として使えます。

#### デプロイヤー

デプロイヤーは、プレイヤーの手の操作を機械で再現する装置です。

- 手に持たせたアイテムで右クリック操作や左クリック操作を行えます。
- レンチや設定画面で動作モードを切り替えられます。
- フィルターを設定すると、特定アイテムだけを対象にできます。
- 苗木の植え付け、骨粉使用、アイテムの設置、加工補助などに使えます。
- からくりに載せると、移動しながら設置や使用を行う機械になります。

#### スパウト / アイテム排液口 / 液体パイプ

液体を使う加工は、スパウト、アイテム排液口、ポンプ、パイプを組み合わせます。

- スパウトは、上のタンクやパイプから液体を受け取り、下のアイテムへ注ぎます。
- アイテム排液口は、液体入りアイテムから液体を抜きます。
- メカニカルポンプはパイプ内の液体を押し流します。
- ホースプーリーは大きな液体源を吸い上げたり、液体を放出したりします。

よく使う形は `タンク -> パイプ -> ポンプ -> スパウト -> ベルト上のアイテム` です。

#### メカニカルアーム

メカニカルアームは、複数の搬入元と搬出先を行き来してアイテムを運ぶ機械です。

- デポ、ベルト、チェスト、ファンネルなどを対象にして、アイテムを拾う場所と置く場所を設定します。
- 設定した対象の間をアームが自動で往復します。
- レンチや設定操作で、順番に配る、空いている場所へ配るなどの動作を調整できます。
- フィルター付きの搬出先と組み合わせると、簡単な仕分け装置になります。

大量搬送はベルト、複数地点への振り分けはメカニカルアームが向いています。

#### 超粘着剤 / シャーシ / からくり

からくりは、複数ブロックをまとめて動かす仕組みです。

- 超粘着剤は、隣接ブロックをからくりとしてまとめるために使います。
- リニアシャーシは直線方向にブロックをまとめます。
- ラジアルシャーシは周囲のブロックをまとめます。
- メカニカルベアリングは、まとめた構造物を回転させます。
- メカニカルピストンは、ピストン延長ポールと一緒に直線移動させます。
- ローププーリーは、構造物を上下に動かします。
- ガントリーシャフトとガントリーキャリッジは、長距離の直線移動に向いています。

伐採機や採掘機を作る場合は、`シャーシ -> ドリル/ソー -> ポータブルストレージインターフェース -> 動力源` の流れで考えると組みやすいです。

#### ポータブルストレージインターフェース

ポータブルストレージインターフェースは、動くからくりと固定設備の間でアイテムを受け渡すために使います。

1. からくり側にポータブルストレージインターフェースを取り付けます。
2. 固定設備側にも向かい合うように設置します。
3. からくりが近づくと、両方のインターフェースが接続されます。
4. 接続中にチェストやベルトへアイテムを移動できます。
5. 液体用にはポータブルフルイドインターフェースを使います。

動く採掘機の回収口、列車や移動倉庫の荷下ろし口として重要です。

#### レッドストーンリンク / リンクコントローラー

レッドストーンリンクは、同じ周波数同士でレッドストーン信号を送受信します。

- 送信側と受信側に同じアイテムを入れて周波数を合わせます。
- リンクコントローラーは、6つのボタンに周波数を割り当てて遠隔操作できます。
- 右クリックでコントローラーをオン/オフします。
- スニーク + 右クリックで手動設定画面を開きます。
- レッドストーンリンク受信機を右クリックすると、コントローラーのボタンへ割り当てできます。

離れた場所のクラッチ、ギアシフト、ドア、採掘機の起動停止などに向いています。

#### 概略図 / 概略図と羽根ペン / 概略図砲

建築物を保存して再配置するための道具です。

- 概略図と羽根ペンで範囲を選択し、構造物を `.nbt` として保存します。
- 右クリックで角を選び、`Ctrl` + スクロールで位置や範囲を調整します。
- 保存した概略図を持つと、配置用のホログラムを表示できます。
- 概略図砲に必要素材を入れると、ホログラムに沿って自動建築できます。

大きな装置を別ワールドや別拠点へコピーしたいときに便利です。

#### よく使うセット例

| 作りたいもの | 組み合わせ | 使い方 |
| --- | --- | --- |
| 初期の板材加工 | `水車 -> シャフト -> メカニカルプレス -> デポ` | インゴットをデポに置くと板へ加工できます。 |
| アイテム搬送ライン | `チェスト -> ファンネル -> メカニカルベルト -> ファンネル -> チェスト` | チェストから出したアイテムを別チェストへ運びます。 |
| 自動粉砕ライン | `ベルト -> 破砕ホイール -> ベルト` | 鉱石や素材を中央に通して粉砕します。 |
| 洗浄ライン | `ベルト -> 水越しエンケーストファン -> 回収ファンネル` | 破砕済み鉱石などを洗浄処理します。 |
| 混合加工 | `ブレイズバーナー -> 鉢 -> メカニカルミキサー` | 加熱が必要な混合レシピに使います。 |
| 液体注入 | `タンク -> パイプ -> ポンプ -> スパウト -> ベルト` | ベルト上のアイテムへ液体を注ぎます。 |
| 動く採掘機 | `シャーシ -> メカニカルドリル -> ポータブルストレージインターフェース` | 移動しながら掘削し、回収口でアイテムを渡します。 |
| 遠隔操作 | `リンクコントローラー -> レッドストーンリンク -> クラッチ/ギアシフト` | 離れた場所から装置をオン/オフします。 |

#### 動かないときの確認ポイント

- シャフトが機械の回転入力面につながっているか確認します。
- 回転方向が逆なら、歯車を1つ挟むかギアシフトで反転します。
- 応力不足なら、水車や風車を増やすか、回転速度を下げます。
- ベルト搬送が止まる場合は、シャフトが2本とも正しくベルトに接続されているか確認します。
- ファンネルが搬入向きか搬出向きか、レンチで確認します。
- 加工レシピが進まない場合は、デポか鉢の位置、加熱の有無、必要な液体や触媒を確認します。

### 列車運転台 / からくり操作盤

Create の列車運転台やからくり操作盤は、独自キーではなくバニラの移動キーをそのまま読み取ります。

| キー | 送信される操作 |
| --- | --- |
| `W` | 前進操作 |
| `S` | 後退操作 |
| `A` | 左操作 |
| `D` | 右操作 |
| `Space` | ジャンプ操作 |
| `Shift` | スニーク操作 |
| `Esc` | 操作終了 |

運転台や操作盤に入ると、これらの押下状態がサーバーへ送信されます。実際の動きは、接続されている列車やからくり側の制御処理に依存します。

## Mekanism

### キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| `N` | アイテムモードの切り替え | メインハンド、または条件によってオフハンドのモード対応アイテムを切り替えます。 |
| `V` | 頭装備モードの切り替え | ヘルメット枠のモード対応装備を切り替えます。 |
| `G` | 胴装備モードの切り替え | チェストプレート枠のモード対応装備を切り替えます。 |
| `J` | 脚装備モードの切り替え | レギンス枠のモード対応装備を切り替えます。 |
| `B` | 足装備モードの切り替え | ブーツ枠のモード対応装備を切り替えます。 |
| 左 `Shift` | 詳細を表示 | GUI やツールチップ上で詳細情報を表示する補助キーです。 |
| `Shift` + `N` | 説明を表示 | GUI やツールチップ上で説明文を表示します。 |
| `\` | モジュール設定 | MekaSuit や Meka-Tool など、設定可能なモジュールを持つ装備の設定画面を開きます。 |
| 左 `Ctrl` | 加速 | ジェットパックやブースト系モジュールなど、加速入力を使う装備で押下状態を送信します。 |
| `H` | HUD の表示 | Mekanism HUD 表示を切り替えます。 |

### 操作方法

- モード切り替えキーは、対応する装備やアイテムを持っているときだけ動作します。
- スニーク中にモード切り替えを行うと、多くのモード対応アイテムでは逆方向に切り替わります。
- `\` は設定可能なモジュールを持つアイテムや装備がある場合に、モジュール設定画面を開きます。
- 左 `Ctrl` の加速キーは押下と離上の両方がサーバーに送られるため、長押し式のブースト操作として扱われます。

| `S` | 磁気グローブ盾の切り替え | 対応装備でダブルタップ操作として使われます。 |
| 未割り当て | ケミカル放射器のタンク切り替え | マルチタンク構成などで使う場合は、キー設定から割り当てます。 |
| 中クリック | レールガンズーム | ズーム可能なレールガンを持っているときに使います。 |

## 独自キーが確認できなかった mod

以下の jar では、今回の逆コンパイルでユーザー操作用の独自 `KeyMapping` は確認できませんでした。

- `create-new-age-1.1.7c+neoforge-mc1.21.1.jar`
- `createdeco-2.1.2.jar`
- `FarmersDelight-1.21.1-1.2.9.jar`
- `burgermod-neoforge-2.10.3-1.21.1.jar`
- `prefab-neoforge-1.0.4.jar`
- `cloth-config-15.0.140-neoforge.jar`

これらは主に右クリック、スニーク + 右クリック、通常 GUI、または各アイテム・ブロックの画面内操作で扱います。

## キー競合の注意

- 左 `Alt` は Create のツールメニューと Epic Fight の回避が重なります。同時導入する場合はどちらかを変更すると安定します。
- `G` は Mekanism の胴装備モード切り替え、他 mod のロックオンや領域系キーと衝突しやすいです。
- `H` は Mekanism HUD、TaCZ の武器確認、Mahou Tsukai の Selective Displacement と重なりやすいです。
- `V` は Mekanism の頭装備モード、TaCZ の近接/ズーム、JujutsuCraft のメイン技などと競合しやすいです。
- `S` は通常の後退キーですが、Immersive Engineering の磁気グローブ盾切り替えにも使われます。誤爆する場合は IE 側を変更してください。
