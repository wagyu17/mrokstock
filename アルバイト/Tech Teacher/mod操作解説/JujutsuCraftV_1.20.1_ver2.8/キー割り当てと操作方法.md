# JujutsuCraft 系 mod キー割り当てと操作方法

## 解析元

- `JujutsuCraft-ver48-forge-1.20.1.jar`
- `JujutsuCraftV_1.20.1_ver2.8.jar`
- `jujutsu_kasien_addon-remake-ver2.jar`
- 同梱されている操作系 mod: `curios`, `ParCool`, `ProjectE`, `JEI`

各 jar の `KeyMapping` 登録クラスと、キー押下時に呼ばれる procedure / packet 処理を逆コンパイルして確認した内容です。

## 前提MOD

`JujutsuCraft-ver48-forge-1.20.1.jar` 本体の `META-INF/mods.toml` を確認したところ、以下の前提MODが `mandatory=true` として明記されています。

| 前提MOD | modId | 必須/任意 | 記載元 |
| --- | --- | --- | --- |
| GeckoLib | `geckolib` | 必須 | `JujutsuCraft-ver48-forge-1.20.1.jar` の `META-INF/mods.toml` |
| Player Animator / playerAnimator | `playeranimator` | 必須 | `JujutsuCraft-ver48-forge-1.20.1.jar` の `META-INF/mods.toml` |

そのため、Jujutsu Craft (Sorcery Fight) 本体を Forge 1.20.1 環境で動かすには、GeckoLib と Player Animator を前提MODとして一緒に入れる必要があります。

補足:

- `JujutsuCraft-ver48-forge-1.20.1.jar` には、`geckolib` と `playeranimator` が `mandatory=true` の依存として書かれています。
- 同じ `mods.toml` 内に `mandatory=false` の重複記載もありますが、`mandatory=true` の依存定義が存在するため、前提MODとして扱うのが安全です。
- `JujutsuCraftV_1.20.1_ver2.8.jar` 側では、`geckolib` と `playeranimator` は `mandatory=false` の任意依存として記載されています。必須指定は JujutsuCraft 本体側にあります。
- このフォルダ内には `geckolib-forge-1.20.1-4.8.3.jar` はありますが、確認時点では Player Animator 系の jar は見当たりませんでした。実際に導入する場合は、Forge 1.20.1 対応の Player Animator も追加してください。

## JujutsuCraft 本体のキー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| `R` | 技切替 | 押すたびに使用する術式・技を切り替えます。 |
| `Z` | 技発動 | 選択中の技を発動します。押下・長押し・離すタイミングが技の処理に使われます。 |
| `G` | 簡易領域の使用 | 簡易領域を使用します。押している時間も送信されます。 |
| `X` | 技の選択リセット | 選択中の技をリセットします。 |
| `M` | 反転術式の使用 | 反転術式を使用します。押下・離上の両方が処理されます。 |
| `S` + `Space` | バックステップ | ジャンプが必要な回避操作です。`S` と `Space` は mod 側でも監視されています。 |
| `N` | 領域展延の使用 | 領域展延を切り替えます。 |
| `Space` | ダブルジャンプ | 通常のジャンプとは別に、mod のダブルジャンプ判定にも使われます。 |
| `W` | 前進 | 通常移動キーですが、術式・移動処理にも押下状態が送信されます。 |
| `C` | 1つ目/2つ目の術式に切り替え | 術式スロットを切り替えます。 |
| `V` | メイン技を使用する | メイン技を直接使用します。押下・離上の両方が処理されます。 |
| `A` | 左 | 通常移動キーですが、mod 側でも押下状態が送信されます。 |
| `D` | 右 | 通常移動キーですが、mod 側でも押下状態が送信されます。 |

### 基本操作

1. `R` で使いたい技を選びます。
2. `Z` で選択中の技を発動します。
3. 技選択をやり直したいときは `X` を押します。
4. 反転術式は `M`、簡易領域は `G`、領域展延は `N` で使用します。
5. `C` で術式スロットを切り替え、`V` でメイン技を直接使います。
6. 回避・移動系の処理では `W/A/S/D` と `Space` の押下状態も使われるため、基本的にはバニラの移動キーと同じ割り当てのまま使うのが安全です。

## JujutsuCraftV アドオン

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| `H` | 呪術スロットアクション | 武器庫呪霊の Curios スロット用アクションです。 |
| `Shift` + `H` | 呪術スロットアクションの別操作 | 武器庫呪霊内のアイテムを切り替えます。 |

### 操作方法

- 武器庫呪霊を Curios スロットに装備した状態で `H` を押すと、武器庫呪霊のインベントリを開きます。
- `Shift` を押しながら `H` を押すと、武器庫呪霊内のアイテムを切り替えます。

## Jujutsu Kasien Addon

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| `B` | 0.2s domain | 解放済みの場合、押している間だけ 0.2 秒領域を開きます。離すと閉じます。 |
| `M` | Sense of breath | 解放済みの場合、押している間だけ呼吸感知系の効果を付与します。離すと解除されます。 |

### 操作条件

- `B` の 0.2s domain は、アドオン側の `domain_02` 進捗を解放している必要があります。
- `M` の Sense of breath は、アドオン側の `glgw` 進捗を解放している必要があります。
- `M` は JujutsuCraft 本体の `反転術式の使用` と競合します。両方を使う場合は、どちらかを別キーへ変更してください。

## 同梱 mod の主なキー

### Curios

| キー | 機能 |
| --- | --- |
| `G` | Curios インベントリを開く/閉じる |

`G` は JujutsuCraft 本体の簡易領域と競合します。

### ParCool

| キー | 機能 |
| --- | --- |
| `Ctrl` + `P` | ParCool の有効化/無効化 |
| `C` | クロール / ブロック内に隠れる |
| 右クリック | 崖つかまり、ヴォルト、ジップライン、ぶら下がり、壁滑りなどの文脈操作 |
| `R` | 受け身、ドッジ、水平壁走り |
| 左 `Ctrl` | 高速走行 |
| `Space` | 壁ジャンプ |
| `Alt` + `P` | ParCool 設定画面 |

`R`、`C`、`Space` は JujutsuCraft 本体と重なりやすいです。

### ProjectE

| キー | 機能 |
| --- | --- |
| `Shift` + `X` | ヘルメット効果 |
| `X` | ブーツ効果 |
| `V` | チャージ |
| `C` | 追加機能 |
| `R` | 投射物発射 |
| `G` | モード変更 |

JujutsuCraft 本体の `R`、`C`、`V`、`G` と競合しやすいです。

### JEI

| キー | 機能 |
| --- | --- |
| `Ctrl` + `O` | JEI オーバーレイ表示/非表示 |
| `Ctrl` + `F` | 検索バー選択 |
| `R` | カーソル上アイテムのレシピ表示 |
| `U` | カーソル上アイテムの用途表示 |
| `A` | ブックマーク追加 |
| `Backspace` | 前のレシピへ戻る |
| `PageUp` / `PageDown` | レシピページ切り替え |
| `Esc` | レシピ画面を閉じる |

JEI の `R` と `A` は、JujutsuCraft の技切替・左移動フックと同じキーです。インベントリ画面内で主に使うキーなので、戦闘操作とは場面が分かれます。

## キー競合のおすすめ調整

- `G` は JujutsuCraft の簡易領域、Curios、ProjectE が重なります。簡易領域を使うなら Curios か ProjectE を別キーにするのがおすすめです。
- `M` は JujutsuCraft 本体の反転術式、Jujutsu Kasien Addon の Sense of breath と重なります。どちらかを空いているキーに移してください。
- `R` は JujutsuCraft の技切替、ParCool、ProjectE、JEI レシピ表示で多用されます。戦闘中に誤爆する場合は ParCool / ProjectE 側を変更すると扱いやすくなります。
- `C` は JujutsuCraft の術式切替、ParCool、ProjectE が重なります。術式切替をよく使う場合は他 mod 側を変更してください。
- `V` は JujutsuCraft のメイン技と ProjectE のチャージが重なります。
