# Sophisticated Backpacks キー割り当てと操作方法

## MOD概要

Sophisticated Backpacks は、大容量のバックパックと多機能アップグレードを追加する収納拡張MODです。アイテム回収、補充、圧縮、精錬、クラフト、ツール切り替えなどをバックパック内で扱えます。

## 解析元

- jar: `sophisticatedbackpacks-1.21.4-3.25.45.1744.jar`
- modId: `sophisticatedbackpacks`
- 主な解析対象: `assets/sophisticatedbackpacks/lang/ja_jp.json`, `net/p3pp3rf1y/sophisticatedbackpacks/client/KeybindHandler.class`, `net/p3pp3rf1y/sophisticatedbackpacks/commands/**`

## キー割り当て

`KeybindHandler` のキー登録処理から、次の初期キーを確認しました。

| キー | 機能 |
| --- | --- |
| `B` | バックパックを開く |
| `C` | インベントリ連携操作 |
| 未割り当て | ツール切り替え |
| `Alt + Z` | アップグレード1のオン・オフ |
| `Alt + X` | アップグレード2のオン・オフ |
| 未割り当て | アップグレード3のオン・オフ |
| 未割り当て | アップグレード4のオン・オフ |
| 未割り当て | アップグレード5のオン・オフ |

アップグレード切り替えキーは、バックパック内に入っている対応アップグレードを素早く有効・無効にするためのものです。よく使う機能を1番・2番スロットに置いておくと扱いやすくなります。

## 独自コマンド

管理者・MODパック制作者向けのコマンドが含まれています。

| コマンド候補 | 内容 |
| --- | --- |
| `/sb list <playerName>` | プレイヤーのバックパック情報一覧 |
| `/sb give <backpackUuid>` | UUID指定でバックパックを付与 |
| `/sb template list` | バックパックテンプレート一覧 |
| `/sb template create ...` | テンプレート作成 |
| `/sb template delete ...` | テンプレート削除 |
| `/sb template give ...` | テンプレートからバックパック付与 |
| `/sb template export ...` | テンプレート書き出し |
| `/sb dynamic begin <templateName> <backpackItem> [baseTemplateName]` | 動的テンプレート作成開始 |
| `/sb dynamic addItem <slot>` | 動的テンプレートにアイテムを追加 |
| `/sb dynamic addUpgrade ...` | 動的テンプレートにアップグレードを追加 |
| `/sb dynamic end` | 動的テンプレート作成終了 |
| `/sb removeNonPlayer onlyWithEmptyInventory` | プレイヤー以外に紐づくバックパックの整理候補 |
| `/sophisticatedbackpacks ...` | `/sb` と同系統の正式名コマンド |

通常プレイヤーが日常的に使うのはキー操作とGUI操作で、コマンドは主に管理・配布・検証用です。

## 操作方法

### バックパックを開く

バックパックを所持している状態で `B` を押すと、バックパックGUIを開きます。バックパックを手に持って右クリックして開く使い方も基本操作です。

バックパックには革、銅、鉄、金、ダイヤ、ネザライトなどの段階があり、上位のものほど容量やアップグレードスロットが増えます。

### アップグレードを入れる

バックパックGUI内のアップグレードスロットにアップグレードアイテムを入れると、追加機能が有効になります。

よく使う例は次の通りです。

| アップグレード | 主な使い方 |
| --- | --- |
| Pickup Upgrade | 拾ったアイテムをバックパックへ自動収納 |
| Magnet Upgrade | 周囲のドロップアイテムを引き寄せて収納 |
| Filter Upgrade | 対象アイテムを指定して収納・移動を制御 |
| Deposit Upgrade | チェストなどへバックパック内のアイテムを預ける |
| Restock Upgrade | チェストなどからバックパックへ補充・回収 |
| Refill Upgrade | 手持ちやインベントリのアイテムをバックパックから補充 |
| Feeding Upgrade | 食料を自動使用 |
| Compacting Upgrade | インゴットからブロック化など、圧縮クラフトを補助 |
| Void Upgrade | 指定アイテムを消去 |
| Smelting Upgrade | バックパック内で精錬 |
| Crafting Upgrade | バックパック内でクラフト |
| Tool Swapper Upgrade | 適した道具への切り替えを補助 |
| Stack Upgrade | 収納スタック数を増加 |
| Tank / Pump Upgrade | 液体の保持・吸い上げ |
| XP Pump Upgrade | 経験値の出し入れ |

### アイテムを自動回収する

Pickup Upgrade を入れると、拾ったアイテムをバックパックに入れられます。さらに Filter Upgrade と組み合わせると、「鉱石だけ入れる」「丸石は入れない」のような制御ができます。

採掘用バックパックでは、Pickup + Filter + Compacting を組み合わせると、鉱石や素材の整理が楽になります。

### チェストとやり取りする

Deposit Upgrade は、バックパック内のアイテムをチェストなどへ預ける用途です。Restock Upgrade は、スニーク右クリックしたチェストからバックパックへアイテムを補充・回収する用途として言語ファイル上で説明されています。

倉庫整理では、Depositで不要品を預け、Restockで必要な素材だけ補充する流れが便利です。

### アップグレードをキーで切り替える

`Alt + Z` でアップグレード1、`Alt + X` でアップグレード2を切り替えます。MagnetやPickupなど、状況によってオン・オフしたいアップグレードを前のスロットに置いておくと扱いやすいです。

## 注意点

- 自動回収・磁石・虚無化系のアップグレードは、フィルター設定を間違えると必要なアイテムまで移動・消去する可能性があります。
- サーバーでは一部アップグレードや容量が設定で制限されている場合があります。
- コマンドは管理者向けです。授業で説明する場合は、まず `B` で開く、アップグレードを入れる、Pickup/Magnet/Depositを試す、という順番が分かりやすいです。
