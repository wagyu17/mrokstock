# AppleSkin キー割り当てと操作方法

## MOD概要

AppleSkin は、食べ物の満腹度・隠し満腹度・体力回復量などをゲーム画面上で見やすく表示する情報表示系MODです。食料を食べる前に「どれくらい回復するか」を判断しやすくなります。

## 解析元

- jar: `appleskin-neoforge-mc26.1-3.0.9.jar`
- modId: `appleskin`
- loader: NeoForge
- 主な解析対象: `META-INF/neoforge.mods.toml`, `assets/appleskin/lang/*.json`, `squeek/appleskin/**`

## キー割り当て

このMOD専用のキー割り当ては検出されませんでした。

ただし、バニラの操作と組み合わせて次の表示が使われます。

| 操作 | 内容 |
| --- | --- |
| インベントリで食べ物にカーソルを合わせる | 食べ物の回復量情報をツールチップに表示 |
| `Shift` を押しながら食べ物を見る | 設定によって、詳細な満腹度・隠し満腹度を表示 |
| 食べ物を手に持つ | 満腹度バー上に、食べた後の回復見込みを半透明表示 |
| デバッグ画面表示時 | 設定が有効なら、満腹度・隠し満腹度・消耗度などを追加表示 |

## 独自コマンド

AppleSkin 固有のスラッシュコマンドは検出されませんでした。

## 操作方法

### 食べ物の回復量を確認する

1. インベントリを開きます。
2. 食べ物アイテムにマウスカーソルを合わせます。
3. ツールチップに、満腹度と隠し満腹度の回復量が表示されます。

設定によっては、`Shift` を押したときだけ詳細情報を表示する形式になります。常に表示したい場合は、AppleSkin のクライアント設定でツールチップの常時表示を有効にします。

### 食べる前に満腹度バーで確認する

食べ物を手に持つと、画面下の満腹度バーに「食べた後にどこまで回復するか」が半透明で表示されます。満腹度が減っているときに、どの食べ物を使うか判断しやすくなります。

### 体力回復の目安を見る

自然回復が有効な環境では、食べ物によって回復が見込める体力がハート上に表示されます。戦闘前や探索中に、食べるべきタイミングを判断する用途で使います。

## 設定で変えられる主な項目

- ツールチップに食料値を表示するか
- `Shift` なしで常に食料値を表示するか
- 手に持った食べ物の回復見込みをHUDに表示するか
- オフハンドの食べ物も表示対象にするか
- 体力回復量の予測をハートに表示するか
- 隠し満腹度や消耗度の表示を有効にするか
- デバッグ画面に食料関連の詳細を表示するか

## 注意点

- AppleSkin は情報表示MODで、食べ物の性能そのものは変更しません。
- サーバー側に入っていない場合でもクライアント表示として使えることがありますが、環境によって表示精度が変わる場合があります。
- 他のHUD改変MODと併用すると、満腹度バー周辺の表示が重なる可能性があります。
