# Prime Punch Patch キー割り当てと操作方法

## 解析元

- `primepunchpatch.jar`
- `player-animation-lib-forge-1.0.2-rc1+1.20.jar`
- `photon-forge-1.20.1-1.1.14.jar`
- `Pehkui-3.8.2+1.20.1-forge.jar`
- `geckolib-forge-1.20.1-4.8.2.jar`

`primepunchpatch.jar` の `PrimepunchModKeyMappings`、各キーの `Message`、押下時に呼ばれる `Procedure`、画面オーバーレイ、言語ファイルを逆コンパイルして確認した内容です。依存ライブラリ側はアニメーション、エフェクト、サイズ変更、モデル表示の補助が中心で、このフォルダ内で Prime Punch 用に追加されている主要キーは下の8個です。

## キー割り当て

| キー | 機能 | 操作方法 |
| --- | --- | --- |
| `Z` | 技スロット1 | 現在選択中の種族技、または格闘スタイル技の1番目を使います。内部処理は `ZOnKeyPressedProcedure` です。 |
| `X` | 技スロット2 | 現在選択中の2番目の技を使います。内部処理は `XOnKeyPressedProcedure` です。 |
| `C` | 技スロット3 | 現在選択中の3番目の技を使います。内部処理名は `RealCPressedProcedure` です。 |
| `V` | 技スロット4 | 現在選択中の4番目の技を使います。内部処理名は変則的に `COnKeyPressedProcedure` です。 |
| `G` | 技スロット5 / 大技 | 現在選択中の5番目の技を使います。強力な技や究極技に割り当てられていることが多いです。内部処理は `GPressedProcedure` です。 |
| `R` | 種族技 / 格闘スタイル技の切り替え | 押すたびに `styleselected` が `0` と `1` で切り替わります。`0` が種族技、`1` が格闘スタイル技です。画面にも `R to switch styles` と表示されます。 |
| `H` | 短時間の移動速度ブースト | クールダウンが残っていない時に、約2秒間の移動速度上昇を付与します。効果量はレベルに応じて上がります。再使用まで約7.5秒です。 |
| `M` | ステータスメニュー | Rank、Level、XP、Health、Strength、Stat Points を表示するメニューを開きます。ポイントがあれば Health / Strength に振れます。 |

すべてのキーは「押した瞬間」に処理されます。長押し、離した瞬間、押している時間による別処理は確認できませんでした。

## 基本操作の流れ

1. 種族または格闘スタイルのアイテムを右クリックして、使う系統をセットします。
2. `R` で種族技ページと格闘スタイル技ページを切り替えます。
3. 画面下部の表示を見ながら `Z`、`X`、`C`、`V`、`G` で技を発動します。
4. クールダウン中、レベル不足、スタン中は技が出ません。
5. レベルアップなどで得たステータスポイントは `M` のメニューから Strength / Health に振ります。

## 種族と格闘スタイルのセット方法

| アイテム | セットされる内部状態 | 使える技枠 |
| --- | --- | --- |
| `Human V1` | `racestyle = humanv1` | 2枠 |
| `Human V2` | `racestyle = humanv2` | 5枠 |
| `Cyborg V1` | `racestyle = cyborgv1` | 2枠 |
| `Cyborg V2` | `racestyle = cyborgv2` | 5枠 |
| `Esper v1` | `racestyle = esperv1` | 2枠 |
| `Esper v2` | `racestyle = esperv2` | 5枠 |
| `Speed O Sound Sonic Style` | `fightingstyle = sonic` | 格闘スタイル側の技 |
| `Limit Breaker` | `fightingstyle = limitbroken` | 5枠 |

V1系の種族は基本的に `Z` と `X` の2技構成です。V2系や Limit Breaker は `Z`、`X`、`C`、`V`、`G` の5枠を使います。Speed O Sound Sonic 系の一部技は `Speed O Sound Katana` が必要で、未装備だと `Must have sword equipped` というメッセージが出る処理があります。

## 技スロットの解除条件

| スロット | キー | 解除目安 |
| --- | --- | --- |
| 1 | `Z` | Level 1 以上 |
| 2 | `X` | Level 5 以上 |
| 3 | `C` | Level 10 以上、かつそのフォームが3技以上持つ |
| 4 | `V` | Level 20 以上、かつそのフォームが4技以上持つ |
| 5 | `G` | Level 30 以上、かつそのフォームが5技以上持つ |

画面の `Locked` 表示はこの条件を見ています。V1フォームのように技枠自体が2つしかない場合、3枠目以降はレベルを上げても基本的に使えません。上位フォームや別スタイルへ切り替える必要があります。

## 技の内容

技名はゲーム内表示ではなく、内部リソース名と処理名から読めるものです。

| 系統 | `Z` | `X` | `C` | `V` | `G` |
| --- | --- | --- | --- | --- | --- |
| Human | Heavy Punch 系 | Uppercut 系 | Serious Crack / Knuckle 系 | Rapid Punch / Table Flip 系 | Human Ultimate / Dropkick 系 |
| Cyborg | Cyborg Shot 系 | Cyborg Kick / Launch 系 | Cyborg Thunder Kick 系 | Cyborg Laser / Machinegun Blows 系 | Incinerate 系 |
| Esper | Esper Push 系 | Esper Lift / Target 系 | Esper Combo 系 | Esper Fly / Esper Boom 系 | Esper Meteor / Esper Ultimate 系 |
| Sonic | Sonic Slice / Flash 系 | Sonic 連撃系 | Sonic 斬撃系 | Fourfold Flash / Final Slice 系 | Sonic Ultimate 系 |
| Limit Breaker | Normal Punch 系 | Consecutive Normal Punches / Barrage 系 | Knuckle / Smash 系 | Omnidirectional Punch 系 | Serious Punch 系 |

実戦では「今のページが種族か格闘スタイルか」を `R` で確認してから技を押すのが重要です。同じ `Z` でも、種族ページなら Human / Cyborg / Esper 側の技、格闘スタイルページなら Sonic / Limit Breaker 側の技になります。

## クールダウンの見方

- 種族技ページでは `racecd1` から `racecd5` が使われ、`Cd1` から `Cd5` のオーバーレイが表示されます。
- 格闘スタイル技ページでは `fightstylecd1` から `fightstylecd5` が使われ、`Fscd1` から `Fscd5` のオーバーレイが表示されます。
- `Z` は1番、`X` は2番、`C` は3番、`V` は4番、`G` は5番のクールダウンに対応します。
- `H` の移動速度ブーストは、技スロットとは別に `COOLDOWN_1` 効果を使って再使用を制限しています。

## ステータスメニュー

`M` を押すと `StatmenuScreen` が開きます。表示される主な項目は `Rank`、`Level`、`XP`、`Health`、`Strength`、`Stat Points` です。

- Strength の `add` ボタンは、ステータスポイントを1消費して攻撃力計算用の内部値を増やします。
- Health の `add` ボタンは、ステータスポイントを1消費して `healthstat` を増やし、最大体力系の強化と回復系効果を付け直します。
- ステータスポイントが0以下の場合、ボタンを押しても加算されません。

## キー競合の注意

この mod は `Z`、`X`、`C`、`V`、`G`、`R`、`H`、`M` をまとめて使うため、他の戦闘系 mod とかなり競合しやすいです。

- `R` は Epic Fight、JujutsuCraft、JEI のレシピ表示などと競合しやすいです。
- `G` は Epic Fight のロックオン、Curios、ProjectE などと競合しやすいです。
- `M` は JujutsuCraft 系の変身 / 特殊効果キーと競合しやすいです。
- `C`、`V`、`X` は JujutsuCraft、ProjectE、ParCool などと競合しやすいです。
- `H` は JujutsuCraftV の Curios スロット操作と競合しやすいです。

複数 mod を同時に遊ぶ場合は、Minecraft の「操作設定」から Prime Punch 側か他 mod 側のどちらかを別キーへ移すのがおすすめです。Prime Punch のキー名は言語ファイル上では `Z`、`X`、`C`、`V`、`G`、`M`、`R`、`H` のようにそのまま表示されます。
