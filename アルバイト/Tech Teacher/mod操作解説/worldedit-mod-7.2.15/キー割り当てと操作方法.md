# WorldEdit キー割り当てと操作方法

## MOD概要

WorldEdit は、ゲーム内で広範囲のブロック編集、建築物のコピー・貼り付け、地形生成、ブラシ編集などを行えるワールド編集 MOD です。通常のキー操作よりも、専用アイテムとコマンドを組み合わせて使うタイプの MOD です。

## 解析元

- 対象ファイル: `worldedit-mod-7.2.15.jar`
- MOD ID: `worldedit`
- バージョン: `7.2.15+6463-5ca4dff`
- 対応 Minecraft: `1.20.x`
- 主要設定ファイル: `defaults/worldedit.properties`
- コマンド解析元: `com/sk89q/worldedit/command/*CommandsRegistration.class`

## キー割り当て

WorldEdit には、Minecraft のキー設定画面に追加される専用キーバインドはありません。操作は主に以下の 3 種類です。

| 操作 | 内容 |
|---|---|
| `/` キー | Minecraft 標準のチャット欄を開き、WorldEdit コマンドを入力する |
| 左クリック | 選択ワンド、ナビゲーションワンド、ブラシ、ツールの主要アクションを実行する |
| 右クリック | 選択ワンド、ナビゲーションワンド、一部ツールの副アクションを実行する |

## デフォルトの専用アイテム

| アイテム | 役割 | デフォルト設定 |
|---|---|---|
| 木の斧 | 範囲選択ワンド | `minecraft:wooden_axe` |
| コンパス | ナビゲーションワンド | `minecraft:compass` |

`defaults/worldedit.properties` では、木の斧が選択用、コンパスが移動用のワンドとして設定されています。

## 木の斧による範囲選択

WorldEdit の基本は「2 点を選択して、その範囲にコマンドを実行する」流れです。

| 操作 | 内容 |
|---|---|
| `//wand` | 木の斧を取得する |
| 木の斧でブロックを左クリック | 選択範囲の 1 点目を設定する |
| 木の斧でブロックを右クリック | 選択範囲の 2 点目を設定する |
| `//pos1` | 現在位置を 1 点目にする |
| `//pos2` | 現在位置を 2 点目にする |
| `//hpos1` | 見ているブロックを 1 点目にする |
| `//hpos2` | 見ているブロックを 2 点目にする |

基本例:

```text
//wand
木の斧で角のブロックを左クリック
反対側の角のブロックを右クリック
//set stone
```

この例では、選択した直方体範囲が石に置き換わります。

## コンパスによる移動操作

コンパスはナビゲーションワンドとして動作します。

| 操作 | 内容 |
|---|---|
| `/thru` | 視線方向の壁を通り抜ける |
| `/jumpto` | 見ているブロック付近へ移動する |
| コンパス左クリック | 壁抜け移動を実行する |
| コンパス右クリック | 見ている地点へジャンプする |

設定上の最大距離は `nav-wand-distance=50` です。

## 主要コマンド

WorldEdit のコマンドは、`//set` のように `//` で始まるものと、`/brush` や `/tool` のように通常の `/` で始まるものがあります。

### コマンドグループの別名

jar 内の `PlatformCommandManager` では、以下のコマンドグループと別名が登録されています。

| 正式コマンド | 別名 | 内容 |
|---|---|---|
| `/schematic` | `/schem` | スキーマティックの保存・読み込み |
| `/snapshot` | `/snap` | スナップショットからの復元 |
| `/superpickaxe` | `/pickaxe`, `/sp` | スーパーピッケル |
| `/brush` | `/br` | ブラシ設定 |
| `/worldedit` | `/we` | WorldEdit 本体設定・情報表示 |

### 範囲選択

| コマンド | 内容 |
|---|---|
| `//wand` | 選択ワンドを取得する |
| `//toggleeditwand` | 選択ワンド機能の有効・無効を切り替える |
| `//sel` | 選択形状を変更する |
| `//pos1` / `//pos2` | 現在位置を選択位置にする |
| `//hpos1` / `//hpos2` | 見ているブロックを選択位置にする |
| `//chunk` | 現在いるチャンクを選択する |
| `//expand <量> [方向]` | 選択範囲を広げる |
| `//contract <量> [方向]` | 選択範囲を縮める |
| `//shift <量> <方向>` | 選択範囲だけを移動する |
| `//outset <量>` | 選択範囲を外側に広げる |
| `//inset <量>` | 選択範囲を内側に縮める |
| `//size` | 選択範囲のサイズを確認する |
| `//count <ブロック>` | 範囲内の指定ブロック数を数える |
| `//distr` | 範囲内のブロック分布を表示する |

選択形状の例:

```text
//sel cuboid
//sel poly
//sel ellipsoid
//sel sphere
//sel cyl
//sel convex
```

### 範囲編集

| コマンド | 内容 |
|---|---|
| `//set <ブロック>` | 選択範囲を指定ブロックで埋める |
| `//replace <元ブロック> <新ブロック>` | 範囲内のブロックを置き換える |
| `//walls <ブロック>` | 選択範囲の壁だけを作る |
| `//faces <ブロック>` | 選択範囲の外側の面を作る |
| `//overlay <ブロック>` | 地表面にブロックを重ねる |
| `//center <ブロック>` | 選択範囲の中心にブロックを置く |
| `//smooth` | 地形をなめらかにする |
| `//move <距離> [方向]` | 選択範囲の中身を移動する |
| `//stack <回数> [方向]` | 選択範囲を連続複製する |
| `//regen` | 選択範囲をワールド生成状態に戻す |
| `//deform <式>` | 数式で地形や形状を変形する |
| `//hollow [厚さ] [ブロック]` | 選択範囲を中空にする |
| `//naturalize` | 地表を自然な土・石構成にする |
| `//line <ブロック> [太さ]` | 選択点を結ぶ線を作る |
| `//curve <ブロック> [太さ]` | 選択点を通る曲線を作る |
| `//forest` | 選択範囲に木を生成する |
| `//flora` | 選択範囲に草花を生成する |

よく使う例:

```text
//set oak_planks
//replace dirt grass_block
//walls stone_bricks
//stack 5 north
//move 3 up
```

### 生成系

| コマンド | 内容 |
|---|---|
| `//sphere <ブロック> <半径>` | 球を生成する |
| `//hsphere <ブロック> <半径>` | 中空の球を生成する |
| `//cyl <ブロック> <半径> [高さ]` | 円柱を生成する |
| `//hcyl <ブロック> <半径> [高さ]` | 中空の円柱を生成する |
| `//pyramid <ブロック> <サイズ>` | ピラミッドを生成する |
| `//hpyramid <ブロック> <サイズ>` | 中空のピラミッドを生成する |
| `//generate <ブロック> <式>` | 数式に基づいて形状を生成する |
| `//generatebiome <バイオーム> <式>` | 数式に基づいてバイオームを生成する |
| `/forestgen <サイズ> [種類] [密度]` | 周囲に森を生成する |
| `/pumpkins <サイズ>` | 周囲にかぼちゃを生成する |

### コピー・切り取り・貼り付け

| コマンド | 内容 |
|---|---|
| `//copy` | 選択範囲をクリップボードにコピーする |
| `//cut` | 選択範囲を切り取ってクリップボードに入れる |
| `//paste` | クリップボードの内容を貼り付ける |
| `//rotate <角度>` | クリップボードを回転する |
| `//flip [方向]` | クリップボードを反転する |
| `/clearclipboard` | クリップボードを空にする |

コピーと貼り付けは、プレイヤーの立ち位置が基準点になります。

```text
建物を選択
貼り付け時の基準にしたい場所に立つ
//copy
移動する
//paste
```

向きを変えて貼り付けたい場合:

```text
//copy
//rotate 90
//paste
```

### 履歴操作

| コマンド | 内容 |
|---|---|
| `//undo` | 直前の WorldEdit 操作を取り消す |
| `//redo` | 取り消した操作をやり直す |
| `/clearhistory` | 編集履歴を消去する |

設定上の履歴数は `history-size=15` です。

### 移動系

| コマンド | 内容 |
|---|---|
| `/unstuck` | ブロックにはまった状態から抜ける |
| `/ascend` | 上の空間へ移動する |
| `/descend` | 下の空間へ移動する |
| `/ceil` | 天井付近へ移動する |
| `/thru` | 見ている方向の壁を通り抜ける |
| `/jumpto` | 見ているブロック付近へ移動する |
| `/up <高さ>` | 足元にブロックを置いて上へ移動する |

### ブラシ

ブラシは、手に持っているアイテムに編集効果を割り当てて、クリックした場所に形状や効果を適用する機能です。

| コマンド | 内容 |
|---|---|
| `/brush sphere <ブロック> <半径>` | 球形ブラシを設定する |
| `/brush cylinder <ブロック> <半径> [高さ]` | 円柱ブラシを設定する |
| `/brush clipboard` | クリップボード内容を貼るブラシにする |
| `/brush smooth` | 地形をなめらかにするブラシにする |
| `/brush extinguish` | 火を消すブラシにする |
| `/brush gravity` | 浮いたブロックを落とすブラシにする |
| `/brush butcher` | 範囲内の Mob を削除するブラシにする |
| `/brush heightmap` | 高さマップ系のブラシを設定する |
| `/brush deform <式>` | 数式で変形するブラシにする |
| `/brush set <ブロック>` | 表面を指定ブロックにするブラシにする |
| `/brush forest` | 木を生やすブラシにする |
| `/brush raise` | 地形を盛り上げるブラシにする |
| `/brush lower` | 地形を下げるブラシにする |
| `/brush snow` | 雪を積もらせるブラシにする |
| `/brush biome <バイオーム>` | バイオームを変更するブラシにする |
| `/brush none` | 現在のブラシを解除する |

ブラシの最大半径は設定上 `max-brush-radius=10` です。

基本例:

```text
石を持つ
/brush sphere stone 3
地面をクリック
```

これでクリックした地点を中心に、半径 3 の石の球形ブラシが実行されます。

マスクを組み合わせる例:

```text
/brush sphere grass_block 3
/mask dirt
```

この状態では、土ブロックにだけブラシ効果が適用されます。

### ツール

ツールは、手に持っているアイテムに特殊な編集機能を割り当てます。

| コマンド | 内容 |
|---|---|
| `/tool selwand` | 現在のアイテムを選択ワンドにする |
| `/tool navwand` | 現在のアイテムをナビゲーションワンドにする |
| `/tool info` | クリックしたブロック情報を見るツールにする |
| `/tool tree` | 木を生成するツールにする |
| `/tool stacker` | クリック方向へブロックを積むツールにする |
| `/tool repl <ブロック>` | クリックしたブロックを置き換えるツールにする |
| `/tool cycler` | ブロック状態を切り替えるツールにする |
| `/tool floodfill <ブロック> <範囲>` | つながった同種ブロックを塗りつぶすツールにする |
| `/tool deltree` | 木を削除するツールにする |
| `/tool farwand` | 遠くのブロックを選択できるワンドにする |
| `/tool lrbuild <左クリック用> <右クリック用>` | 左右クリックで別ブロックを置くツールにする |
| `/tool unbind` | 現在のツールを解除する |

`/tool none` やグローバルコマンドの `/none` も登録されていますが、このバージョンではツール系のグローバル名は将来的に削除予定という警告文が含まれているため、基本は `/tool unbind` 形式で覚えるのが安全です。

ツール補助コマンド:

| コマンド | 内容 |
|---|---|
| `/mask <条件>` | ツールやブラシの対象ブロックを制限する |
| `/material <ブロック>` | ツールやブラシの素材を変更する |
| `/range <距離>` | ツールやブラシの射程を変更する |
| `/size <サイズ>` | ツールやブラシのサイズを変更する |
| `/tracemask <条件>` | 視線判定の対象ブロックを制限する |

置き換えツールの例:

```text
棒を持つ
/tool repl stone_bricks
置き換えたいブロックを左クリック
```

`/tool repl` は、右クリックでクリックしたブロックを素材として読み取り、置き換え先を切り替える動作も持っています。

### スーパーピッケル

スーパーピッケルは、ブロック破壊を強化する機能です。

| コマンド | 内容 |
|---|---|
| `//` | スーパーピッケルの有効・無効を切り替える |
| `/sp single` | 1 ブロック破壊モード |
| `/sp area <サイズ>` | 周囲の範囲をまとめて破壊する |
| `/sp recursive <サイズ>` | つながった同種ブロックをまとめて破壊する |

設定上の最大サイズは `max-super-pickaxe-size=5` です。

### スキーマティック

建築物をファイルとして保存・読み込みする機能です。

| コマンド | 内容 |
|---|---|
| `/schem save <形式> <名前>` | クリップボード内容を保存する |
| `/schem load <名前>` | スキーマティックを読み込む |
| `/schem delete <名前>` | スキーマティックを削除する |
| `/schem list` | 保存済みスキーマティックを一覧表示する |

基本例:

```text

/schem load house01
//paste
```

設定上の保存先ディレクトリ名は `schematics` です。

## よく使う操作手順

### 選択範囲をブロックで埋める

```text
//wand
木の斧で 1 点目を左クリック
木の斧で 2 点目を右クリック
//set stone
```

### 床だけを張り替える

```text
//wand
範囲を選択
//replace grass_block polished_andesite
```

### 建物の壁だけを作る

```text
//wand
建物サイズの範囲を選択
//walls stone_bricks
```

### 建築物をコピーして貼り付ける

```text
//wand
建築物を選択
基準にしたい場所に立つ
//copy
貼り付けたい場所へ移動
//paste
```

### 同じ建物を横に並べる

```text
建築物を選択
//stack 5 east
```

### ブラシで地形を盛る

```text
棒などのアイテムを持つ
/brush sphere dirt 4
地形をクリック
```

解除するとき:

```text
/brush none
```

### 水を抜く

```text
水辺に立つ
//drain 20
```

### 操作を取り消す

```text
//undo
```

やり直す場合:

```text
//redo
```

## 注意点

- 大きすぎる範囲に `//set` や `//replace` を使うと、ワールドが重くなったりサーバーが停止したりする可能性があります。
- 最初は `//limit 10000` のように編集上限を設定しておくと安全です。
- 重要なワールドでは、作業前にバックアップを取ってください。
- サーバー環境では、各コマンドに権限が必要な場合があります。
- 設定では `use-inventory=false` のため、デフォルトではインベントリ内のブロック所持数を消費せずに編集できます。
- 設定上、岩盤、TNT、松明、苗木など一部ブロックは制限対象に含まれています。
