# Farmer's Delight キー割り当て・独自コマンド・操作方法

## MOD概要

Farmer's Delight は、農業・食材加工・料理設備を追加して、Minecraft の食事作りを大きく広げる料理系MODです。

## 解析元

- 対象ファイル: `FarmersDelight-1.21.1-1.3.2.jar`
- MOD ID: `farmersdelight`
- バージョン: `1.3.2`
- 表示名: `Farmer's Delight`
- 解析方法: jar 内の `META-INF/neoforge.mods.toml`、lang、recipe JSON、主要 class を確認

## キー割り当て

Farmer's Delight は、専用の新規キー設定を追加していません。基本操作は Minecraft 標準の操作を使います。

| 操作 | デフォルト操作 | 内容 |
| --- | --- | --- |
| ブロックや道具を使う | 右クリック / 使用キー | まな板、鍋、コンロ、かご、ロープ、フライパンなどを操作します。 |
| スニークしながら使う | Shift + 右クリック | フライパンを設置する、鍋の吊り下げ表示を切り替える、ロープを巻き取るなど、一部の特殊操作に使います。 |
| フライパン調理中に返す | 左クリック / 攻撃キー | 手持ちフライパンで調理中、使用を続けたまま攻撃キーを押すと食材をひっくり返します。 |
| インベントリを開く | E | 料理や食材、容器の確認に使います。 |

## 独自コマンド

独自のスラッシュコマンドはありません。

jar 内を確認した限り、Farmer's Delight 用の Brigadier コマンド登録や `RegisterCommandsEvent` は見つかりませんでした。

## 主要アイテム・ブロックの操作方法

### ナイフ

ナイフは、食材加工と軽い近接攻撃に使う道具です。

- 草を壊すと Straw などの素材を得られる場合があります。
- 動物を倒したとき、通常より多くの副産物を得られる場合があります。
- 攻撃速度は速めですが、単発の攻撃力は低めです。
- カボチャに右クリックすると、くり抜いたカボチャにできます。
- ケーキや対応するケーキ系ブロックに使うと、スライスとして切り分けられます。
- まな板の加工レシピでは、ナイフ系ツールとしてよく使います。

### まな板

まな板は、食材やブロックを道具で加工するための作業台です。

基本の流れ:

1. まな板を設置します。
2. 加工したい食材やブロックを持って、まな板を右クリックして置きます。
3. ナイフ、斧など、レシピに対応した道具を持ってまな板を右クリックします。
4. 加工結果がドロップします。
5. 何も持たずに右クリックすると、まな板の上のアイテムを回収できます。

代表例:

- 原木 + 斧: 皮を剥いだ原木と Tree Bark を作れます。
- Cabbage + ナイフ: Cabbage Leaf に切り分けられます。
- Raw Chicken + ナイフ: Chicken Cuts などに加工できます。

補足:

- まな板には1種類のアイテムを置けます。
- 加工に使った道具は耐久値を消費します。
- レシピに合わない道具を使っても加工されません。

### 鍋

鍋は、複数の食材を入れて料理を作る調理ブロックです。

基本の流れ:

1. 鍋を設置します。
2. 鍋の下に熱源を置きます。コンロや焚き火など、熱源として扱われるブロックが必要です。
3. 鍋を右クリックしてGUIを開きます。
4. 食材スロットに材料を入れます。
5. 必要に応じて容器スロットに Bowl などの容器を入れます。
6. 調理が進むと完成品ができます。
7. 完成した料理はGUIから取り出すか、対応する容器を持って鍋を右クリックして取り出します。

操作のポイント:

- 空の手でスニーク + 右クリックすると、鍋の吊り下げハンドル表示を切り替えられます。
- 完成料理に対応した容器を手に持って右クリックすると、1食分を直接すくえます。
- 鍋を壊すと中身がドロップします。
- 調理に使ったレシピ経験値は、鍋を壊したときなどに排出されます。

代表的な料理:

- Beef Stew: 牛肉、にんじん、じゃがいもなどから作成します。
- Fried Rice: Rice、卵、にんじん、玉ねぎなどから作成します。

### コンロ

コンロは、鍋の熱源として使えるほか、単体で食材を焼くこともできる調理ブロックです。

基本操作:

- Flint and Steel や Fire Charge を使うと点火できます。
- シャベルや水入りバケツを使うと消火できます。
- 火がついているコンロの上面に、焼ける食材を右クリックで置けます。
- 焼き上がった食材はコンロ上から回収できます。

注意点:

- 点火中のコンロに触れるとダメージを受ける場合があります。
- 上面がふさがれていると、食材を置けない場合があります。
- コンロを壊すと、置かれていた食材がドロップします。

### フライパン

フライパンは、手に持って調理することも、ブロックとして設置して使うこともできます。

手持ち調理:

1. フライパンを片手に持ちます。
2. もう片方の手に焼きたい食材を持ちます。
3. 火の近く、または熱源の近くに立ちます。
4. フライパンを右クリック長押しで使います。
5. 調理中に左クリック / 攻撃キーを押すと、食材をひっくり返します。
6. 調理が完了すると、焼けた料理を入手できます。

設置して使う:

1. スニークしながら右クリックすると、フライパンをブロックとして設置できます。
2. 設置したフライパンに食材を右クリックで入れます。
3. 熱源の上で加熱されると調理が進みます。
4. 空の手で右クリックすると、中の食材や完成品を取り出せます。

補足:

- 手持ち調理では、もう片方の手に材料を持っていないと調理できません。
- 水中では手持ちフライパン調理ができません。
- 設置するにはスニーク操作が必要です。

### ロープ

ロープは、縦方向に伸ばしたり、巻き取ったりできる設置ブロックです。

基本操作:

- ロープを設置します。
- 既存のロープにロープを右クリックすると、基本的には下方向に延長されます。
- スニークしながら既存ロープに使うと、クリックした面の方向へ延長できます。
- 空の手でスニーク + 右クリックすると、ロープを下側から巻き取るように回収できます。

補足:

- 設定で Rope Reeling が有効な場合、スニーク + 空手右クリックで巻き取りできます。
- ロープの上にベルがつながっている場合、ロープを右クリックしてベルを鳴らせることがあります。

### かご

かごは、アイテムを入れられる収納ブロックです。

基本操作:

- かごを設置します。
- 空の手で右クリックすると、かごのGUIを開けます。
- かごを壊すと、中のアイテムがドロップします。
- レッドストーン比較器で中身の量を検出できます。

## レシピの種類

Farmer's Delight には、主に次の独自レシピがあります。

| レシピ種別 | 内容 |
| --- | --- |
| Cooking | 鍋で食材を煮込む料理レシピです。 |
| Cutting | まな板と道具で食材やブロックを加工するレシピです。 |
| Salvaging | 一部アイテムを素材に戻す分解系レシピです。 |
| Integration | 他MODやバニラ要素との連携レシピです。 |

## 料理効果

Farmer's Delight の料理には、通常の満腹度回復以外に独自効果を持つものがあります。

| 効果 | 内容 |
| --- | --- |
| Nourishment | 体力を回復する場合を除き、満腹度ゲージの減少を防ぎます。 |
| Comfort | 空腹、移動速度低下、弱体化に対する耐性を与えます。 |

## 使い方の例

### まな板で食材を切る

1. まな板を置きます。
2. Cabbage などの食材をまな板に置きます。
3. ナイフを持ってまな板を右クリックします。
4. Cabbage Leaf などの加工品を入手します。

### 鍋で料理を作る

1. コンロを置いて点火します。
2. コンロの上に鍋を置きます。
3. 鍋に食材を入れます。
4. 容器スロットに Bowl などを入れます。
5. 完成した料理を取り出します。

### フライパンで外出先調理をする

1. フライパンをメインハンドに持ちます。
2. オフハンドに焼きたい食材を持ちます。
3. 火や熱源の近くに立ちます。
4. 右クリック長押しで調理します。
5. 左クリックで食材を返しながら焼きます。

## 注意点

- 専用キー設定は追加されないため、操作はほぼ右クリック、スニーク、攻撃キーで完結します。
- 鍋やフライパンは、熱源がないと調理が進みません。
- まな板は、置いたアイテムと使う道具の組み合わせがレシピに合っている必要があります。
- フライパンをブロックとして設置するには、スニークしながら使う必要があります。
