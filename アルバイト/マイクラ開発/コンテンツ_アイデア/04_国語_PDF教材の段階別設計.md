# 国語②：「PDF教材の段階別設計」

## 基本情報

| 項目 | 内容 |
|------|------|
| 対象教科 | 国語（読解力・ひらがな・漢字・文章理解） |
| 対象学年目安 | 小学1〜4年相当 |
| 難易度 | ★☆☆☆☆（設計ガイド） |
| 想定対象者 | 放課後等デイサービス利用児童・支援学級児童 |
| 主目的 | すべての児童が「読める」「理解できる」段階別教材の統一設計 |

---

## このドキュメントの目的

Minecraftを使った国語学習は、**「絶対に否定されない安全な環境」で読む喜びを体験すること**が最大の目的である。

本ドキュメントでは、1つのコンテンツ（例：村人からの建築依頼）に対して、**3段階のレベル分けされたPDF教材を統一的に設計・製作する方法**を示す。

これにより、同じ時間に同じワールドで学習していても、子ども1人1人の読解レベルに合わせた教材配布が可能になる。

## 段階別教材レベル定義

### レベル1：「ひらがな中心・図説型」

**対象児童：**
- ひらがなの学習途中（小学1年相当）
- 漢字がまだ読めない時期
- 文字量が多いと不安になる児童
- 視覚的な情報（図・イラスト）で理解するタイプ

**文字設計：**
- **ひらがな**：100%（すべての内容がひらがな）
- **漢字**：0%（使用なし）
- **カタカナ**：固有名詞のみ。すべてに小さいふりがなを付ける
- **フォントサイズ**：18pt以上
- **フォント種**：ゴシック体（楷書体NG - 読みづらい）
- **行間**：1.5倍以上（余裕を持たせる）

**文構成：**
- 1つの看板・指示＝1～2文（短文）
- 1ページ＝最大150文字（A4用紙でも数行程度）
- 「です・ます体」ではなく「～だよ」「～ってね」という親しみやすい表現

**イラスト比率：**
- テキスト：イラスト＝3：7（イラストを大きく、メインに）
- テキストは吹き出しで村人キャラクターに喋らせる形式
- 指示内容を図解する（「ここに置く」という矢印など）

**記入欄：**
- なし、または極小（回答はチェックボックスで○をつけるのみ）

**色使い：**
- 背景：白または薄いクリーム色（目に優しい）
- 文字：濃い黒（コントラストを強く）
- 吹き出し枠：温かみのある色（オレンジ・薄緑）
- 強調：アンダーラインでなく、背景を薄い色で塗る

**例文：**

```
タロウさん：「こんにちは！ぼくは タロウです。」

「あたらしい おうちが ほしいです。
 ブラウンいろで つくってください。
 まどは 2つ つけてね。」

→ ひとこと：「ほんが すきなので、ほんだなも ください」

【 いま あなたが することは 】
□ ブラウンブロックを じゅんびする
□ タロウさんの すむばしょに いく
□ おうちを つくる
```

### レベル2：「漢字混在・ふりがな付き型」

**対象児童：**
- 小学1～2年相当（学習漢字80～160字の習得段階）
- 文を読むことに慣れてきた児童
- 少し長めの説明文でも理解できる児童

**文字設計：**
- **ひらがな**：70～80%
- **漢字**：小学1～2年配当字（80～160字）。すべてに**ふりがな付き**
- **カタカナ**：固有名詞のみ。ふりがな付き
- **フォントサイズ**：16pt
- **フォント種**：ゴシック体
- **行間**：1.3倍以上

**文構成：**
- 1つの指示＝2～3文（中文）
- 1ページ＝最大300文字（A4でも余白を保つ）
- 「です・ます体」を基本としながら、親しみやすさも意識
- 条件を「・」で箇条書きにすると理解しやすい

**イラスト比率：**
- テキスト：イラスト＝5：5（テキストとイラストが同等の役割）
- 図解や矢印で指示を視覚化（「ここに窓をつける」など）
- キャラクター吹き出しと説明文の両方を併用

**記入欄：**
- 簡単な記述欄あり（1～2行程度）
- チェックボックスまたは○×式の回答

**色使い：**
- 背景：白
- 文字：濃い黒
- 強調部分：薄い背景色（黄色・オレンジ）でマーク
- 枠線：濃いグレー（優しく見える程度）

**例文：**

```
【 タロウさんからのおねがい 】

タロウは、あたらしいおうちがほしいです。

【 条件（じょうけん） 】
・ブラウンブロックで つくること
・窓（まど）を 2つ つけること
・本棚（ほんだな）を 作ること

【 つくるばしょ 】
　村（むら）の東（ひがし）がわです。
　看板（かんばん）をさがしてください。

【 期限（きげん） 】
　1週間（しゅうかん）以内（いない）

【 あなたのしごと 】
1. 依頼書（いらいしょ）を よみました。（□よんだ）
2. 材料（ざいりょう）を あつめます。
3. おうちを つくります。
4. 完成（かんせい）させます。
```

### レベル3：「読み応え型・段落構成」

**対象児童：**
- 小学3～4年相当（学習漢字200～320字の習得段階）
- 段落のある文章が読める児童
- 「なぜ」という理由を理解できる児童

**文字設計：**
- **ひらがな**：50～60%
- **漢字**：小学3～4年配当字（200～320字）。難読字のみふりがな付き
- **カタカナ**：固有名詞・技術用語に使用。ふりがなは不要
- **フォントサイズ**：14pt
- **フォント種**：ゴシック体（本の設定に合わせて）
- **行間**：1.2～1.3倍

**文構成：**
- 1つの指示＝3～4文以上（長文・複合文）
- 1ページ＝最大500～800文字
- 段落を明確に分ける（改行・空白行で視認性を確保）
- 「～です」「～ます」の文体統一
- 「～ために」「～ので」など、因果関係を示す表現

**イラスト比率：**
- テキスト：イラスト＝7：3（テキストがメイン）
- イラストは補足的な役割（文を読めば理解できるが、理解を助ける）
- グラフや表を活用可能

**記入欄：**
- 複数行の記述欄あり（段落構成の作文練習）
- 記述式問題（「なぜ」という理由を問う）

**色使い：**
- 背景：白
- 文字：濃い黒
- 見出し：濃いめの色（紺色・濃いオレンジ）
- 強調：線または背景色（控えめに）
- 枠線：細い線（デザインがうるさくならないよう）

**例文：**

```
【 建築依頼の背景 】

タロウは、村に新しく移り住んできた住民です。現在、仮住まいで生活していますが、
自分の家を持ち、本をたくさん読める空間を作りたいと考えています。

【 依頼内容の詳細 】

タロウの希望する家には、以下の条件があります。

（1）外観について
   ブラウンブロックを主材料とした家を希望しており、木の温かみが感じられる
   デザインを望んでいます。また、採光を大切にしているため、左右対称に
   2つの窓を配置することが必須となっています。

（2）内装について
   タロウの最大の特徴は、本を愛読することです。そのため、室内には本棚を
   設置し、彼が落ち着いて読書できるスペースを確保することが重要です。
   本棚の配置や大きさについては、建築者の創意工夫に任せられています。

【 建築者としての課題 】

この依頼を通じて、以下のスキルを磨くことができます：
・依頼主の要望を正確に読み取り、優先順位をつける能力
・限られた条件の中で、自分なりの工夫を加える創造性
・完成後、自分の選択理由を言語化する表現力

【 期限と評価基準 】

完成予定日：1週間以内
評価：「完璧な再現」ではなく、「依頼主の要望にどれだけ応えられたか」
      と「工夫をどう加えたか」が評価の中心となります。
```

はじめまして。本文では、村に転入してきたタロウと申します。

現在、仮住まいでの生活を送っていますが、自分専用の家を持つことで、

また、採光の重要性から、左右対称に2つの窓を配置することを
強く希望しています。北側と南側、または東側と西側など、
配置については建築者の判断にお任せします。

【 第2章：内部空間の設計について 】

私の最大の特徴は、読書を何より愛していることです。

そのため、室内に本棚を設置し、心穏やかに読書できるスペースの確保が

建築者の創意工夫に委ねられています。

【 第3章：完成期限と評価基準 】

完成予定日：提案日より1週間以内

評価について：
本依頼では、「正確さ」よりも「要望への応答性」と「個性的な工夫」を

その上でどのような独自の工夫を加えられるかが重要なのです。

完成後には、建築の過程で工夫した点について、

よろしくお願いいたします。
タロウ

- タロウのキャラクターイラスト：25%
- テキスト（複数段落、見出し含む）：75%
- テキスト全体：約450～550字
- 読了時間：7～10分

## フォント・レイアウトガイドライン

### レベル別フォント設定表

| 項目 | レベル1 | レベル2 | レベル3 |
|------|--------|--------|--------|
| 本文フォントサイズ | 18pt以上 | 16pt | 14pt |
| 見出しサイズ | 24pt以上 | 20pt | 16pt |
| フォント種 | ゴシック体 | ゴシック体 | ゴシック体 |
| 行間 | 1.5～1.8倍 | 1.3～1.5倍 | 1.2～1.3倍 |
| 字間（文字間隔） | 広め | 通常 | 通常 |
| 行の長さ | 最大25字 | 最大35字 | 最大40字 |
| 上下の余白（ページ） | 余裕あり | 余裕あり | 余裕あり |

### カラー設計ガイドライン

**背景色：**
- すべてのレベル：白またはオフホワイト（#FFFFF または #FFFEF0）
- 目に優しく、読みやすさを最優先

**文字色：**
- 本文：濃い黒（#1A1A1A）または濃いグレー（#2C2C2C）
- ふりがな：本文より1段階薄い色

**見出し：**
- レベル1：オレンジ（#FF8C42）
- レベル2：濃いオレンジ（#E07A3F）
- レベル3：紺色（#003D7A）

**強調・マーク：**
- ハイライト：薄い黄色（#FFFACD）など、目に優しい色
- 枠線：濃いグレー（#CCCCCC）- 線を引くのではなく、背景を塗る

**吹き出し（キャラクター台詞）：**
- 枠線色：オレンジ系（#FFB84D）
- 背景色：薄いオレンジ（#FFF5E6）
- 矢印：キャラクターに向かう形で配置

### レイアウト・デザイン原則

1. **余白の確保**
   - ページの上下左右に余白を設ける（最小1cm）
   - 行と行の間にも十分な空白を持たせる
   - 「ぎゅっと詰まった感じ」を絶対に避ける

2. **視覚的階層の明確化**
   - 見出し > 本文 > 注釈 の3段階を、サイズと色で区別
   - レベル1は見出しと本文の差を特に大きく

3. **キャラクター吹き出しの活用**
   - 村人キャラクターに話しかけさせる形式を採用
   - 「指示」ではなく「お願い」という親密さを表現
   - キャラクターイラストは親しみやすく、淡い色使いで

4. **図解・矢印の活用**
   - 「ここに置く」という指示は必ず矢印で示す
   - グリッド図やマップを含める
   - 視覚的情報で理解を補助

5. **ページ番号と統一感**
   - 各ページの下部にページ番号を配置
   - 同じドキュメント内のすべてのページで統一的なデザイン
   - ヘッダー/フッターで教材の名称を記載

## 読解力評価表

### 読解力レベルの診断と教材選択ガイド

#### 【診断チェックリスト】

子どもの読解力レベルを判定するため、以下のチェックを行う。

**レベル1の指標：**
- □ ひらがなは読めるが、漢字がほぼ読めない
- □ 2語以上の複合文になると理解が難しい
- □ 文章より図や矢印で指示されると安心する
- □ 1度に複数の指示があると混乱する

**レベル2の指標：**
- □ 小学1～2年配当の漢字なら、ふりがながあれば読める
- □ 3～4文程度の段落なら理解できる
- □ 「なぜなら」という理由説明は少し難しいが、単純なら理解できる
- □ テキストと図解の両方があるとより理解しやすい

**レベル3の指標：**
- □ 小学3～4年配当の漢字が読める
- □ 複数段落の文章を読んで理解できる
- □ 「～ために」「～ので」という複合文の意味が分かる
- □ テキストだけでも理解できる（図解がなくても問題ない）

#### 【選択判定フロー】

```
はじめに、子どもに「簡単な文を読んでみようか」と促す

     ↓

レベル1の短文を読ませる
「ぼくは ほんが すきです」

     ↓

□ すぐに読める    →「ひらがなは得意だね」
□ 少し時間がかかる →「ゆっくり読もうか」
□ 読めない        →「ふりがながあるといいかな」

     ↓

次に、簡単な漢字を含む文を読ませる
「本（ほん）が 好（す）きです」

     ↓

□ ふりがなを見てすぐに読める   →レベル1.5への移行を検討
□ ふりがながないと難しい       →レベル1確定
□ ふりがながなくても読める     →レベル2へ進む

     ↓

最終的に、その子どもの現在地レベルを判定
同じ教材内でも、異なるレベルのPDFを配布することで
全員が「読める・わかる」を実感できる環境を実現
```

### 読解力別の声かけポイント

**レベル1で学習する子どもへ：**
- 「ひらがなが読めてすごいね」と肯定
- 「タロウさんが何がほしいのか、わかったね」と内容理解を褒める
- 「難しい字は読み飛ばしてもいいんだよ」と安心させる

**レベル2で学習する子どもへ：**
- 「漢字も頑張って読んでるね」と努力を褒める
- 「複数の条件が書いてあるけど、全部気づけたね」と細部への注意を褒める
- ふりがなを活用する姿勢を肯定的に扱う

**レベル3で学習する子どもへ：**
- 「長い文も理解できるんだね」と読解力の高さを認める
- 「『～ために』という理由まで読み取れた」と深い理解を褒める
- 「タロウさんの気持ちをちゃんと感じて作ってくれたね」と共感力を評価

## 複数児童への配布・運用方法

### シナリオ1：同じワールドで異なるレベルの教材を使用

**状況：** 放課後等デイサービスで、学年が異なる3名がMinecraftをしている

**配布方法：**

1. **事前準備**
   - 「建築依頼書」をレベル1・2・3で3パターン印刷
   - 各子どもの名前を教材に記入
   - 子ども1人1人に「自分用」という意識を持たせる

2. **配布時**
   - 「あなたはこの書いてある条件でお願いね」と個別に説明
   - 「いっしょにタロウさんのおねがいを読もうか」と、その子のレベルのを一緒に読む
   - 他の子どもが別の文を読んでいることは「説明しない」（比較させない）

3. **制作中**
   - 3名が「同じワールド内の異なる場所」で建築
   - 先生が回りながら、各レベルの質問に対応
   - レベル1の子に「内容わかった？」と確認しながら、レベル3の子は「自分で判断できるね」と任せる

4. **完成後**
   - 全員を褒める。「みんなで村を作ったね」という協同感を演出
   - レポート作文も、その子のレベルに合わせた形式で記述させる

---

### シナリオ2：1人の子どもが複数レベルを段階的に体験

**状況：** 同じ子どもが何度も同じシリーズの授業を受ける場合

**段階的な進め方：**

1. **1回目：レベル1で実施**
   - 「とにかく内容が分かる」「タロウさんが何をしてほしいか理解できた」を目指す
   - 読む喜び、完成の達成感を優先

2. **2回目：同じストーリー、レベル1で再実施**
   - 「前回はタロウさんだったけど、今度はハナコさんだよ」
   - 「少し新しい条件があるけど、前と似てるから大丈夫」
   - 達成感の反復で自信を増す

3. **3回目：レベル1→レベル2への橋渡し**
   - 「今度は、ちょっと大きい字の手紙だよ」と説明
   - ふりがながあることを強調
   - 「1～2個漢字が増えてるけど、ふりがなを見れば読めるよ」

4. **4回目以降：レベル2で定着**
   - 「漢字も読めるようになってきたね」と成長を実感させる
   - レベル3への橋渡しも、同様に段階的に

**ポイント：**
- レベルの変更を「レベルアップ」と肯定的に言い換える
- しかし、「急ぐ必要はない」という文化を作る
- 「このレベルが心地よい」なら、ずっとこのレベルでもOKという安全感を持たせる

## 段階別「建築レポート」のテンプレート

### レベル1版：「そのあとで」シート

```
【 つくったもの 】

ぼくが つくったのは　　　　　　　　　です。

【 たいせつだったこと 】

タロウさんが いったことで、
いちばん たいせつだったのは：

□ ブラウンいろ
□ まど 2つ
□ ほんだな

つくってみて、たのしかったことは：



むずかしかったことは：


```

**ポイント：**
- チェックボックスで選択式（記述量を最小に）
- 空白欄も小さめ（1～2行）
- 「楽しかった」「難しかった」という感情を重視

---

### レベル2版：「ふりかえりシート」

```
【 建築（けんちく）のおおまかな説明（せつめい） 】

   理由（りゆう）：

2. 　　　　　　　　　　を こう しました

   理由（りゆう）：

【 自分（じぶん）の作品（さくひん）について 】

上手（じょうず）くいったことは：



もっと工夫（くふう）したかったことは：


```

**ポイント：**
- 「何を工夫したか」を具体的に記述させる
- 「なぜ」という理由を重視
- 成功と課題の両方を記述（自己評価の向上）

## 【重要】放課後等デイサービス利用児童への実装ガイド

### 基本原則：「絶対に否定されない」環境作り

このドキュメント全体の根底には、以下の哲学がある。

**「発達支援における読み書き学習は、『正確さ』よりも『できた喜び』を優先する」**

- 読み間違いがあっても、「読もうとした努力」を褒める
- 記述が短くても、「自分の言葉で表現できた」ことを喜ぶ
- ブロック配置にずれがあっても、「依頼主の条件を理解しようとした」ことを評価する

### 声かけの工夫

**言ってはいけない言葉：**
- 「ここ漢字、違う」
- 「もっていう前に、ちゃんと読んで」
- 「この作文、短いね」
- 「看板の指示、見間違えてる」

**言うべき言葉：**
- 「その文字、難しいね。一緒に読もうか」
- 「タロウさんが何をしてほしいのか、わかったね」
- 「自分の考えを書けたね。素晴らしい」
- 「看板を見ながら、こんなに工夫できたんだ」

### 修正・やり直しの扱い

「直す」ことは「失敗」ではなく「工夫」である。

- 「もう1回やってみようか」→「完璧を目指す」ではなく「別の選択肢を試す」
- 「ここもう少し変えてみたら？」→「あなたの工夫を応援している」というメッセージ
- 修正後は「さらに良くなったね」と成長を実感させる

## まとめ：段階別設計の効果

このような3段階のレベル分けを導入することで：

1. **「読める喜び」の全員体験**
   - 自分のレベルで読み、理解できる体験が、読書への自信につながる

2. **差別化されない学習環境**
   - 同じワールド、同じストーリーで学習するため、「自分だけ違う教材」という劣等感がない
   - むしろ「自分用に作ってくれた」という個別対応の温かさを感じる

3. **段階的な成長支援**
   - 無理なく次のレベルへ進める環境を整える
   - 「今のレベルが心地よい」なら、ずっとそこでいい、という安全感

4. **表現力の多様性の認可**
   - 「短い作文」「長い作文」どちらでも価値がある、という文化
   - 「自分の言葉で表現する」ことそのものを最高価値とする

これらすべてが、**「絶対に否定されない安全な環境」の実現**につながる。
