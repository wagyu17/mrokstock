## ワールド設計

### マップ全体構成

ワールドは「平和な村」として設計する。温かみのある村バイオームに、あらかじめ以下の要素を配置しておく。

**事前配置する要素：**

- **村のゲート**：木製フェンスで作った装飾的な門。ここがスタート地点
- **村人たちの家**：ブロック制限を受けない形式の家を3～4軒配置。各家の前に「建築依頼ボード」を設置
- **タロウさんの家**（メインキャラクター）：緑の屋根が目印。ここから「建築依頼書」を受け取る設定
- **建設エリア**：村の端に、色付きウール（緑）で区画の外枠だけ示しておく。子どもはこの中に建築する
- **資材置き場**：チェストに必要なブロック（木材、石、土など）を用意
- **お手本エリア**：ワールドの見やすい場所に「完成した家」の見本を1つ配置

**マップサイズ：** 約100×100ブロックの範囲に収める。村全体が見える広さ。

**ワールドボーダー：** 村の外には見えない壁（バリアブロック）を設定し、迷子にならないようにする。

### ギミック詳細

#### ギミック1：「建築依頼書」を読もう（読解力・指示理解）

**仕掛け：**
- 「建築依頼書」をPDF形式で配布。村人キャラクター「タロウさん」からのお手紙として設計
- PDF内に、依頼主のキャラクター情報・要望・条件がストーリー形式で記載
- 画像では：村のゲート付近に「タロウさんの家」を配置。プレイヤーが近づくと看板に「建築依頼書を読んでね」とメッセージが表示

**建築依頼書の内容例（レベル1 - ひらがな中心）：**

```
タロウさんからの おねがい

こんにちは！ぼくは タロウです。
このむらに あたらしい おうちが ほしいです。

【 おねがい 】
・ブラウンブロックで つくってください
・まどは 2つ つけてください
・めの マークが あります
・ここが つくるばしょです

【 そのた 】
・ぼくは ほんが すきです
・だから、ほんだなも つけてください
・1週間以内（いない）に できたら うれしいです

すてきなおうちを まってます！
タロウより
```

**学習ポイント：**
- 「何を・どこに・いくつ」という条件を読み取る
- 「依頼主が何を求めているのか」を理解する
- 指示を段階的に整理して理解する力

**褒めポイント（先生用）：**
- 「依頼書、ちゃんと読めたね！」
- 「タロウさんが何をしてほしいか、わかったね」
- 読み間違いがあっても「もう一回一緒に読んでみようか」と励ます

#### ギミック2：看板の指示に従ってブロックを配置（ひらがな・漢字読解）

**仕掛け：**
- 建設エリア内の地面に、複数の看板を配置
- 各看板には短い指示が書かれている

**看板の指示例（段階別）：**

**レベル1（ひらがな）：**
```
看板1：「ここに うちを つくってね」
看板2：「1階は 5×4 ブロック」
看板3：「まどは ここ」（矢印）
看板4：「あかいブロックを かさねよう」
```

**レベル2（漢字＋ひらがな）：**
```
看板1：「建築エリア：4×6ブロック」
看板2：「1階の窓：2つ配置」
看板3：「木製の扉を玄関に」
看板4：「屋根の高さ：3ブロック」
```

**レベル3（読み応えのある文章）：**
```
看板1：「この建設エリアは4×6ブロックの広さです。タロウさんの希望に合わせて家を建てましょう」
看板2：「玄関には木製の扉を設置し、左右に窓を2つ配置することが条件です」
看板3：「屋根の高さはブロック3段とし、完全な直角形状を保つようにしてください」
看板4：「内部にはタロウさんが本を読むための書斎スペースを作る工夫をしましょう」
```

**学習ポイント：**
- 平仮名・漢字・カタカナが混在した文を読む
- 数字・矢印などの視覚情報と組み合わせて理解する
- 「どのレベルの子どもか」で看板の難度を変える

**褒めポイント（先生用）：**
- 「看板を読んで、ちゃんと置けたね！」
- 「指示をちゃんと守って作れたね」
- 読み間違いがあったら「看板にもう一回目を向けてみようか」

#### ギミック3：「建築レポート」を書こう（作文力・表現力）

**仕掛け：**
- 完成後、PDF形式の「建築レポート」シートを配布
- 「何を・どこに・なぜそうしたのか」を子ども自身の言葉で書く

**レポートの例（記入欄）：**

**レベル1（ひらがなのみ、語呂合わせ）：**
```
【 せつめい 】

ぼくが つくったのは ___________________

【 くふう 】

ここに ________________ を つけました。
なぜなら ________________________________

【 かんそう 】

つくってみて、たのしかったことは ____________________
```

**レベル2（漢字混在）：**
```
【 建築の説明 】

わたしが作った家の特徴は ___________________

【 工夫したところ 】

・ ________________ の配置を工夫しました
・ ________________ について、こんな工夫をしました

理由は ________________________________

【 感想 】

建築をしてみて：
楽しかったことは ____________________________
難しかったことは ____________________________
```

**レベル3（段落構成を意識した文章）：**
```
【 建築概要 】
タロウさんの依頼に応じて、以下の特徴を持つ家を建築しました：
- 外観：
- 内装：
- こだわった点：

【 工夫のプロセス 】
最初は ________________________________ と思っていました。
しかし ________________________________ ので、
そこで ________________________________ という工夫をしました。
結果として ________________________________

【 自己評価 】
成功したこと：
課題だったこと：
次に挑戦したいこと：
```

**学習ポイント：**
- 「事実」（何を作ったか）と「意見」（なぜそうしたか）を分ける
- 自分の行動を振り返り、言語化する力
- 段落を意識した文章構成

**褒めポイント（先生用）：**
- 「自分の考えを文に書けたね！」
- 「どうしてそう思ったのか、ちゃんと説明できているね」
- 文法の誤りではなく「思いを言葉にできたこと」を褒める

## 授業設計

### 学習目標

| レベル | 目標 |
|--------|------|
| 知識 | 依頼内容を読み取り、必要な要素を理解する |
| 技能 | 文字（ひらがな・漢字）を読んで、その指示に従い実行できる |
| 思考 | 「何をするべきか」を文章から判断し、工夫を加える |
| 態度 | 他者の要望を理解し、それに応えようとする姿勢 |

### 授業の流れ（60分の場合）

#### 導入（10分）

1. **興味喚起（3分）**：「今日は村人さんたちからお手紙が届いたよ。誰が何をしてほしいのか、読んでみようか」
2. **「建築依頼書」の配布と読み合わせ（5分）**：タロウさんのキャラクター設定を説明。「タロウさんはどんな人？」と問いかけ、子どもの想像力を引き出す
3. **ワールド確認（2分）**：村の見本エリアを見せる。「こんな感じに作れたらいいね」

#### 展開1：依頼内容を理解する（15分）

1. **建築依頼書の精読**：子どもが自分のペースで読む（黙読または音読）
2. **内容確認**：「タロウさんは何がほしいのかな？」「どんな家かな？」と質問
3. **条件の整理**：
   - 「ブラウンブロックで作ってね」→ ブロックの種類を理解
   - 「窓は2つ」→ 数を理解
   - 「本棚もほしい」→ 追加要素を理解
4. **図解・メモ**：子どもが「やることリスト」を書く

**声かけ例：**
- 「タロウさんのお手紙、ちゃんと読めたね」
- 「つまりね、タロウさんは『ブラウンで、窓2つの家がほしい』ってことだね」
- 「何か質問ある？」と開かれた雰囲気を作る

#### 展開2：看板の指示を読んで建築する（25分）

1. **看板の確認**：「建設エリアに看板があるよ。何て書いてあるかな？」
2. **段階的実施**：
   - 看板1を読む → 「1階の大きさは5×4なんだね」と確認 → ブロック配置
   - 看板2を読む → 「窓は左右に2つか。場所はこことここだな」 → ガラス板設置
   - 看板3を読む → 「玄関に扉か。ここに置こう」 → 扉設置
   - 看板4を読む → 「本棚も作ろう」 → 本棚ブロック配置
3. **途中の褒め**：完成するたびに「ここまでできたね！」と声かけ
4. **見本との比較**：「見本と比べてみようか。どう？」

**声かけ例：**
- 「看板を読んで、ちゃんと置けたね！」
- 「窓の場所、タロウさんのお手紙に書いてあったね」
- 「自分で考えて工夫してるね」

#### 展開3：建築レポートを書く（15分）

1. **レポートシート配布**：「今から、自分がどんなふうに作ったか、書いてみようか」
2. **記入支援**：
   - 「どんな家を作った？」→ 1文で答える
   - 「どこに工夫した？」→ 2～3か所を挙げる
   - 「なぜそうしたの？」→ 理由を言葉にする
3. **読み上げ（任意）**：「自分の作品について、みんなに説明してみる？」

**声かけ例：**
- 「自分の考えを文に書けたね」
- 「どうしてそこに窓をつけたのか、良い理由だね」
- 「タロウさんが喜びそうだね」

#### まとめ・振り返り（5分）

1. **完成を褒める**：「すごい家ができたね。タロウさんがきっと喜ぶよ」
2. **振り返り**：「読むこと、作ること、書くこと。全部できたね」
3. **展開（任意）**：「今度は別の村人さんの依頼も受けてみる？」

### 先生用ガイド

#### つまずきやすいポイントと対応

| つまずき | 原因 | 対応方法 |
|---------|------|---------|
| 依頼書が読めない | 文字が多い、分かりにくい | 一緒に音読する。「ここが大事」と指摘する。カタカナに注目させる |
| 指示を忘れてしまう | 情報量が多い | 「やることリスト」を一緒に作る。1つずつ確認しながら進める |
| 看板の指示が理解できない | 「～して」という命令文が分かりにくい | 「つまりね」と言い換える。「ここに置いてね」→「ここにガラスを置く」 |
| ブロックの操作が難しい | マイクラ操作に慣れていない | 「まっすぐ向いて、ここに置こう」と一緒に手を添えて操作する |
| レポート作文が進まない | 「何を書いたらいいか」分からない | 「タロウさんにお手紙を書くみたいに」と例える。「こんなふうに工夫したんだね」と先生が発見を促す |
| 「間違えてないか」と不安 | 完璧主義傾向 | 「タロウさんの依頼が入ってればOK！」と安心させる。修正は「工夫」として肯定的に扱う |

#### 評価基準

| 評価観点 | A（十分達成） | B（概ね達成） | C（支援が必要） |
|---------|-------------|-------------|---------------|
| 読解力 | 依頼内容を正確に読み取り、条件を漏らさず理解した | ほぼ依頼内容を理解したが、1つ2つ見落としがあった | 依頼内容の把握に支援が必要だった |
| 指示の実行 | 看板の指示に完全に従い、正確に実行した | ほぼ指示に従い、実行できた | 指示の実行に大幅な支援が必要だった |
| 工夫・創意 | 自分なりの工夫を加え、独自の工夫が見られた | 指示通りに作ったが、その中に小さな工夫があった | 工夫より完成を優先した |
| 表現力 | レポートで「なぜ」を明確に述べられた | レポートで「何を」は書けたが「なぜ」が不十分 | 記述が少なく、表現の支援が必要だった |

**重要：** C評価であっても「人の要望を読もうとしたこと」「ブロックを置けたこと」「文を書いたこと」を必ず褒める。この教材の最大目的は「他者への共感力」と「自己表現の喜び」である。

## PDF教材構成案

### ページ構成（全8ページ）

| ページ | 内容 | 備考 |
|--------|------|------|
| 1 | 表紙「むらびとさんからの たのみごと」 | イラスト付き。村人キャラクターのイラスト。わくわく感 |
| 2 | 「きょうのミッション」説明 | 「村人さんたちのおねがいを聞いて、うちを作ろう」 |
| 3-4 | 建築依頼書（タロウさん編） | キャラクター紹介＋依頼内容。レベル別で3パターン用意 |
| 5 | 建設エリアの図 | どこに何を配置するかを視覚化。看板の位置も示す |
| 6 | 「たいせつなポイント」 | 「窓は2つ」「本棚を作る」など、依頼条件をまとめた表 |
| 7 | 建築レポート用紙 | 「何を作ったか」「どこに工夫したか」を書く欄 |
| 8 | ふりかえりシート | 「人のおねがいを理解できたか」「自分の考えを書けたか」 |

### デザイン指針

- **フォント**：ゴシック体、レベル1は18pt以上、レベル2は16pt、レベル3は14pt
- **ふりがな**：レベル1・2はすべての漢字に、レベル3は難読字のみ
- **色使い**：教科カラー（オレンジ系）をアクセントに。村人キャラクターは親しみやすい色
- **イラスト**：マイクラ風ピクセルアートで村人・家・ブロックを描く。キャラクターは可愛らしく設計
- **吹き出し**：村人キャラクターが吹き出しで話しかけるスタイル
- **記入欄**：大きめに取り、子どもが安心して書き込める大きさ
- **余白**：たっぷり取り、圧迫感を出さない
- **枠線**：優しい色（濃い枠ではなく、薄いグレー）で区切る

### 別紙：複数村人キャラクター版

この授業の拡張として、複数の村人からの依頼を用意することで、読解力の反復練習が可能。

**例：**
- **タロウさん**：「ブラウンの家に本棚がほしい」
- **ハナコさん**：「赤い屋根の農家がほしい。窓3つください」
- **イサムさん**：「石の壁の立派な家がほしい。玄関に階段をつけて」

各キャラクターに異なる条件を持たせることで、子どもは「人それぞれ違う要望がある」ことを学べる。
