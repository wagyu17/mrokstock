## ワールド設計

### マップ全体構成

ワールドは「あらかじめ自然の中に村がある」設定で設計する。児童が「防災を考える」スタートポイントになる。

**事前配置する要素：**

- **村の中心エリア**：畑・民家3〜4軒・商店・神社など、生活感のあるNPC村の要素をそのまま使用
- **山エリア**：村の北側に急な山を配置。地滑り危険地帯をシミュレート
- **川エリア**：村の西側に川を流す。雨の日の増水による洪水危険を想定
- **平坦な安全地帯**：村の東側に広い平坦地。「ここなら安全」という判断の根拠を示す
- **現在の危険箇所マーカー**：
  - 河岸近くの地面に赤のカーペット（浸水危険）
  - 山の斜面に赤のラテライト（地滑り危険）
  - これらは「探す教材」となる
- **完成見本エリア**：別のワールドやスクリーンショットで「こんな防災マップが完成する」を示す
- **チェックポイント看板**：各役割ごとの作業開始地点に看板を配置
  - 「地形調査員スタート」
  - 「計測係スタート」
  - 「記録係スタート」
  - 「設計係スタート」

**マップサイズ：** 約150×150ブロック。複数の異なった地形を含めるため、ワールド動物園より広い

**ワールドボーダー：** 村全体を見守るように、見えない壁で囲む。児童が迷わないように。

### ギミック詳細

#### 理科パート：「地形を読む」→ 危険箇所を特定する

**仕掛け：**
- 山・川・平地の3つの異なった地形を配置
- 山の斜面には土ブロックが多く、不安定な地形を表現
- 川の両岸には砂利や砂ブロックを配置。雨で増水すると危険な様子を実感
- PDF教材に「山・川・平地の特ちょう」を図解
  - 山：重力がある。雨で土が流れやすい
  - 川：水がある。雨で増水する可能性
  - 平地：高さがない。比較的安全

**学習ポイント：**
- 「自分たちの周りの地形には、危険がある」という気づき
- 「地形を読む」スキルの習得（野外活動の前段階）
- 科学的観察：「どうしてこの場所は危ないのか」を、地形から判断

**児童の役割（地形調査員）：**
1. 村全体を歩き回り、「危険そうな場所」を探す
2. 見つけた場所に、色付きウール（赤）を置いて標識する
3. PDF教材の「危険箇所マップ」に、見つけた場所を記入
4. 理由を書く：「ここは山だから、地滑りが起きやすい」

**褒めポイント（先生用）：**
- 「こんなに細かく危ない場所を見つけたね！」
- 「なぜここが危ないか、理由が言えるね。観察力がすごい！」

#### 算数パート：「避難経路の距離を計算」→ 最短ルートを考える

**仕掛け：**
- 村の各民家から、安全な避難場所（例：広場）までの距離を測定させる
- ブロック数 = メートル数という単純な換算ルール
- 複数の経路が存在：「山の麓を迂回するルート」「川を橋で渡るルート」「直線距離」など
- PDF教材に「最短ルートを計算しよう」というワークシート

**学習ポイント：**
- 距離の測定：「数える」から「計算する」へ
- 複数の経路を比較し、「最短はどちら？」と判断
- 実生活での算数の活用：「避難の時間を短くするには」

**児童の役割（計測係）：**
1. 各民家の位置と避難場所の位置をマイクラで確認
2. 複数の経路（AルートとBルート）のブロック数を数える
3. 「A＝25ブロック（25m）」「B＝18ブロック（18m）」と記録
4. 「どちらが近い？」「25−18＝7なので、Bが7m近い」と計算

**褒めポイント（先生用）：**
- 「距離が正確に測れたね！」
- 「どちらのルートが短いか、計算で比べられたね」
- 「もし自分が避難するなら、どのルート選ぶ？」と応用的な問い

#### 国語パート：「防災標識の意味を読み取る」→ 情報の正確な理解

**仕掛け：**
- 村中に様々な防災標識（看板）を配置
  - 「洪水注意エリア」（川付近）
  - 「地滑り危険」（山付近）
  - 「避難経路→」（複数個所）
  - 「ここに集まって」（広場）
  - 「帰宅困難者一時滞在施設」（役場など）
- PDF教材に「標識の意味を読み取って、絵の中に書き込もう」というワーク

**学習ポイント：**
- 防災情報は「正確に読み取る」ことが生命にかかわる
- 「読み取り」だけでなく「行動に移す」イメージを持つ
- 文字だけでなく、記号（矢印など）の意味も理解

**児童の役割（記録係）：**
1. 村中の標識を見つけて、写真やスクリーンショットに記録
2. 各標識が「何を示しているのか」を読み取る
3. 「避難経路→の看板が5個あった」「危険な場所が7箇所あった」などと集計
4. 防災マップに、標識の位置を記入

**褒めポイント（先生用）：**
- 「標識をちゃんと読めたね！」
- 「この標識があったから、避難できるんだね」

#### 社会パート：「公共施設の位置関係を考える」→ 防災インフラの配置

**仕掛け：**
- ワールドに事前配置されたNPC村の施設（役場、学校など）を確認させる
- 児童に「ここに消防署を作ろう」「ここに避難所を作ろう」と、施設を追加配置させる
- 配置のルール：
  - 避難所は「全ての民家から30m以内」
  - 消防署は「川・山の近く」

**学習ポイント：**
- 公共施設は「偶然ここにある」のではなく、「計画的に配置」されている
- 社会的な課題（防災）と、空間的な計画の関係性
- 「働く人たち」への感謝の気持ちを育む

**児童の役割（設計係）：**
1. 現在の村の施設を把握する
2. 「ここに何があると、みんなが安全かな？」と話し合う
3. 消防署・避難所・備蓄センターなど、必要な施設を提案
4. 実際にマイクラで建築。色分け（例：避難所は青、消防署は赤）

**褒めポイント（先生用）：**
- 「いい場所に避難所を作ったね！」
- 「この施設があると、みんなが安心だね。こういう仕事をしている人がいるんだ」

### 使用ブロック一覧

| ブロック名 | 用途 | 事前配置場所 |
|-----------|------|------------|
| 土ブロック | 山の地形、自然な土地 | ワールド各所に事前配置 |
| 砂利・砂 | 川岸、地滑り危険地帯 | 川エリア |

| カーペット | 地面の標識、危険表示 | 事前配置 |
| バリアブロック | ワールド境界 | 外周に事前配置 |

## 授業設計

### 学習目標（教科統合版）

| 教科 | 目標 |
|------|------|
| **理科** | 自然現象（地滑り・洪水）の危険性を理解し、地形から危険を予測できる |
| **算数** | 距離を測定・計算し、複数の情報から最適な選択ができる |
| **国語** | 情報を正確に読み取り、その意味を理解・伝達できる |
| **社会** | 公共施設の役割を理解し、防災計画の重要性を認識する |
| **総合** | 複数の視点から「安全な村」を設計できる |

### 授業の流れ（4〜5回に分割：各45〜60分）

#### 事前準備（先生向け）：班編成と役割分担

**班構成：** 4〜5名×複数班
**役割分担：**
- **地形調査員**：理科的視点、観察力が必要
- **計測係**：算数的思考、正確さが必要
- **記録係**：国語的思考、伝達力が必要
- **設計係**：統合的思考、リーダーシップが必要
- **サポート役**（5名いる場合）：全体をサポート、複数役をこなす

**重要：** 役割を「能力別」ではなく「興味別」「挑戦したいこと別」で分配。「この役割なら頑張れそう」という自己肯定感から始める。

#### 第1回：導入・地形調査（45分）

**目標：** 村全体の地形を観察し、危険箇所を特定する

1. **導入（5分）**
   - 「この村は自然の中にあります。自然は素敵だけど、危ないこともあります」
   - 「みんなで、この村を安全にするための『防災マップ』を作ります」
   - 「デジタルな村で、本当の防災を学ぼう」

2. **ワールド説明（10分）**
   - 村全体を一周。山・川・平地を見せる
   - 「山は、雨が降るとどうなる？」「川は？」と問いかけ

3. **役割説明（5分）**
   - 「地形調査員のみんなは、危ない場所を見つけてね」
   - 「見つけたら、赤いウールを置いてね」

4. **実施（20分）
   - 児童が村を歩き回り、危険箇所を探す
   - 地形調査員班が「ここ危ないね」と相談しながら赤ウールを設置
   - 先生は横で見守り、「なぜここが危いと思う？」と問いかけ

5. **まとめ（5分）
   - 設置した赤ウール（危険箇所マーカー）を眺める
   - 「すごく危ない場所が見つかったね」

**先生の声かけ例：**
- 「山の近くはどう？何か危いことがありそう？」
- 「川はどうしてこんなに水があるんだろう？」
- 「ここに赤いウール置いた理由は？」

#### 第2回：地形の学習・距離計測準備（45分）

**目標：** 地形の特徴を理科的に理解し、距離計測の準備をする

1. **導入（5分）**
   - 前回の成果物（赤ウール）を見る
   - 「今日は、これらの場所がなぜ危いのかを、もっと詳しく調べます」

2. **理科パート（20分）**
   - 山について：「土が重力で流れる」「雨で流れやすい」をシミュレート
     - 実際に山の上に砂ブロックを置いて、落下を見せる
     - 「こうやって地滑りが起きます」
   - 川について：「水がある場所は、雨で増えます」
     - 川の水を少し追加し、「水かさが増えるでしょ」と見せる
   - 平地について：「高さがないから、水や土が流れていく」

3. **算数パート（15分）：計測係の準備
   - 「今から、民家から避難場所までの距離を測ります」
   - 「1ブロック＝1メートルって決めます」
   - 複数ルートを見せ、「どちらが近い？」と問いかけ

4. **まとめ（5分）
   - 「地形がわかると、危険も見える」

**先生の声かけ例：**
- 「砂が落ちた。これが地滑りだ」
- 「なぜ山は危いのか、見えた？」

#### 第3回：距離計測・避難ルート策定（60分）

**目標：** 複数ルートを計測比較し、最短避難経路を決定する

1. **導入（5分）**
   - 「今日は『どうやって逃げるか』を考えます」
   - 「民家から広場まで、どのルートが一番近い？」

2. **計測係の実施（30分）**
   - 各民家から避難場所への複数ルートを設定
     - ルートA：直線距離（短いが、危険地帯を通る）
     - ルートB：迂回ルート（長いが、安全）
   - 児童がブロック数を数え、距離を計算
   - PDF教材のワークシートに記入

3. **記録係の活動（15分）**
   - 計測結果を集約
   - 「ルートAは25m、ルートBは18m。Bが7m短い」と記録
   - スクリーンショットで記録

4. **設計係の検討（10分）
   - 全ルートの結果を見て、「どのルートが安全か」を話し合い
   - 「短いからといって、危険な場所を通るのは×」という学び

**先生の声かけ例：**
- 「25と18、どちらが大きい？」
- 「短いけど危ない道と、長いけど安全な道。どちらを選ぶ？」

#### 第4回：国語パート＋施設配置（60分）

**目標：** 防災標識を読み取り、必要な施設を配置する

1. **導入（5分）**
   - 「防災には、標識（看板）が大事です」
   - 「どんな標識があるか、探してみよう」

2. **記録係の活動：標識探索（20分）**
   - 村中の看板を見つけ、内容を読み取る
   - 「『洪水注意』『避難経路→』『ここに集まって』など」
   - スクリーンショットで記録
   - PDF教材に「見つけた標識」を記入

3. **設計係の活動：施設配置（25分）
   - 現在の施設を確認
   - 「消防署はどこに置く？」「避難所は？」と相談
   - レンガで施設を建築
   - 色ウール（赤＝消防署、青＝避難所など）で区別

4. **全体共有（10分）
   - 各班の施設配置を見学
   - 「いい場所に置いたね」と共有

**先生の声かけ例：**
- 「『洪水注意』はどこに置いてある？」
- 「消防署、ここに置いたのはなぜ？」

#### 第5回：まとめ・全体共有（60分）

**目標：** 防災マップを完成させ、学習全体を統合する

1. **全班の成果物の見学（15分）**
   - 各班の防災マップを全体で見学
   - スクリーンショット撮影

2. **各班からの発表（25分）**
   - 各班が「これが危険箇所」「これが避難ルート」「ここに避難所を置いた」と説明
   - **重要：** 全員が何か言える環境を作る。発言しない子には「君の班はここを担当したんだね。ここについて教えて」と促す

3. **振り返り（15分）**
   - PDF教材の振り返りシートに記入
     - 「わかったこと」：地形の危険性、防災計画の必要性、公共施設の役割など
     - 「むずかしかったこと」：距離計測、複数視点を持つことなど
     - 「つぎやりたいこと」：防災訓練、本物の村を見てみたい、消防署を訪問したいなど

4. **展開・次のステップ（5分）
   - 「実は、本当の町でも、こういう計画があります」
   - 「消防署を訪問したい人いる？」と、現実世界へのつながりを示唆

**先生の声かけ例：**
- 「4つの視点から、村を安全にできたね！」
- 「理科・算数・国語・社会、全部使ったね」
- 「これが『教科横断』だ」

### 先生用ガイド

#### つまずきやすいポイントと対応

| つまずき | 原因 | 対応方法 |
|---------|------|---------|
| 役割分担で「自分の仕事」がわからない | 役割が抽象的 | 「君たちは『危ない場所を見つける人たち』ね。歩き回ってね」と具体的に |
| 班活動がまとまらない | リーダーシップがない | サポート役を指名するか、先生が適切にサポート |
| 距離計測がずっと数えている | ブロック数が多い | 「10個ずつ数えるとやりやすいよ」と工夫を示唆 |
| 施設配置で「どこに置く？」が決まらない | 複数視点の統合が難しい | 「危険な場所は避けた方がいい。危険地図を見て」とヒント |
| 国語パートで標識の意味がわからない | ひらがなが多くても難しい | 先生が読み上げて、一緒に考える |
| 発表で話せない | 人前での発言が苦手 | 「ここに何を置いた？」と選択肢を提示し、簡潔に促す |
| 時間が足りない | 予定より長くなった | 無理に全部やらない。「今回はここまで。全部ができなくても大丈夫」と伝える |

#### 評価基準

| 評価観点 | A（十分達成） | B（概ね達成） | C（支援が必要） |
|---------|-------------|-------------|---------------|
| 地形観察 | 自分で危険箇所を見つけ、理由が説明できた | ヒントで危険箇所が見つかった | 先生と一緒に見つけた |
| 距離計測 | 複数ルートを正確に計測・比較した | 計測はしたが、比較が不完全 | 数え間違いがあり、サポート必要 |
| 情報読み取り | 標識を読み、意味を正確に理解した | 読み取ったが、意味がまだ曖昧 | 標識の意味を理解するのに支援が必要 |
| 施設配置 | 複数視点から、適切な場所に配置できた | 基本的には配置できたが、理由が曖昧 | 先生の指示で配置した |
| 協働 | チーム内で役割を果たし、他の役割を尊重した | 基本的に協働できたが、主体性に欠ける | 協働が難しく、個別対応が必要 |

**重要：** C評価であっても「班活動に参加できたこと」「1つのタスクでも完了したこと」を必ず褒める。この教材の最大目的は「複数視点で問題を考える力」と「協働の経験」「自己肯定感」である。

### デザイン指針

- フォント：ゴシック体、18pt以上（教科横断なので、読みやすさ最優先）
- ふりがな：すべての漢字にふりがな付き
- 色使い：教科ごとに色分け（理科＝緑、算数＝青、国語＝赤、社会＝黄）
- イラスト：マイクラ風で、山・川・平地を区別しやすく
- 記入欄：班用と個人用の両方用意。大きめで圧迫感なし
- 発表用テンプレート：「我々の班は〜が危ないと思いました」的なフォーマット（任意）

## 発展学習・実践化への展開

### アフターアクティビティ

1. **自分たちの学校・地域の防災マップ作り**
   - マイクラで学んだことを、実際の学校・地域に適用
   - 親子でハザードマップを見てみる

2. **消防署・役場訪問**
   - 実際の防災計画を見学
   - 「マイクラで作ったのと似てる！」という気づき

3. **防災訓練の実施**
   - マイクラで学んだ避難ルートを、実際に歩いてみる
   - 「マイクラは本当に役立つ！」という実感

### 発展的な学習内容（次のステップ）

1. **複数村の防災マップ**
   - 異なった地形（山の多い村、川が多い村）での防災計画比較

2. **時系列での防災**
   - 「雨が降った場合」「地震が起きた場合」など、シナリオ別の対応

3. **デジタル×実践**
   - マイクラで作ったマップを元に、実際のハザードマップを読み解く

4. **国際理解**
   - 「他の国の防災はどう？」と、多角的視点を広げる

## 注：特別な配慮について

### 放課後等デイサービスにおける配慮

この教材は「絶対に否定されない安全な環境」を前提としています。以下の点に留意してください：

**協働活動での配慮：**
- 役割分担は「個性」を認め、「この子はこの役目が得意」と肯定的に枠付ける
- 班活動で意見が異なる場合も「どちらも良い考えだね」と統合する
- 発言を促すときは「答えがある質問」ではなく「あなたの考えは？」とオープン

**学習の進め方：**
- 完璧を求めない。「ここまでできたね」と段階的達成感を与える
- 計測間違い・理解不足があっても「それもいい学習だ」と言語化
- 「できない」ではなく「今は、こういう段階」と捉え直す

**発表での配慮：**
- 全員発言を強要しない。「見守る」役もある
- 発言が短くても「ちょっと言ってくれた。ありがとう」と認める
- 大人向けの完全な説明は求めない。同年代の友達に「ここが危ないんだ」と説明できればOK

**最終的な評価：**
- 防災知識の定着度ではなく「協働できたか」「自己表現できたか」「自己肯定感が高まったか」を重視
