# Minecraft "Movement Tutorial" ワールド リバースエンジニアリング結果

対象のワールド（`tutorial` ディレクトリ）の構造および仕組みを解析した結果を報告します。
このワールドは Minecraft: Education Edition 向けの「移動チュートリアル (Movement Tutorial)」であり、初学者がキーボードとマウスを使った基本操作（視点移動、歩行、ジャンプ等）を学ぶために設計されています。

## 全体構造

ワールドデータは以下の標準的な構成要素から成り立っています。
- **`level.dat`, `levelname.txt`, `db/`**: ワールドの基本セーブデータ。
- **`behavior_packs/bp0/`**: チュートリアルの「ロジック（仕組み）」を制御するビヘイビアパック。
- **`resource_packs/rp0/`**: チュートリアルの「見た目、テキスト、UI」を制御するリソースパック。
- **`texts/`**: ワールド全体で使われる汎用テキスト（リソースパック側にも各国の言語ファイルがあります）。

---

## チュートリアル駆動の仕組み (Behavior Pack)

このチュートリアルは、MinecraftのスクリプトAPI (JavaScript等) ではなく、**コマンド (McFunction) とスコアボード、ダミーエンティティ** の組み合わせによって完全に制御されています。

1. **初期化とメインループ (`i.mcfunction`, `loop.mcfunction`)**
   - ワールドに入ると初期化処理 (`i.mcfunction`) が走り、プレイヤーのインベントリやゲームルールがリセットされます。
   - 同時に `rwm:utility` という透明なダミーエンティティが召喚され、これに `game` というタグが付与されます。
   - `loop.mcfunction` が毎ティック実行され、このダミーエンティティ（`tag=game`）のスコアボード値 `act` と `phase` を監視します。

2. **進行管理 (`act` と `phase` スコア)**
   - **`act=0` (ロビー部):** `act/0/lobby.mcfunction` で管理されます。ここでは「金色のブロックを見る」「振り向く」など、マウスによる視点移動の練習が行われます。プレイヤーの視線 (`lookat` スコア) を検知して `phase` を進め、最終的にゲートを開けます (`open_gate.mcfunction`)。
   - **`act=1` (本編フェーズ):** 公園に入った後の処理で、`act/1/phase.mcfunction` で管理されます。プレイヤーが特定の座標範囲 (x, y, z の `dx, dy, dz` ボリューム) に到達すると `chat.mcfunction` が呼ばれ、適切なアクションバーのメッセージ（ヒント）が表示されます。

## ユーザーインターフェース (Resource Pack)

1. **カスタムHUD (`ui/hud_screen.json`)**
   - 画面の下部に操作のヒントを常時表示するため、アクションバー（`actionbar_message`）のUI定義がカスタマイズされています。
   - 「ヒント (instruction.prefix)」という固定テキストと、状況に応じたメッセージの2段組で表示されるよう工夫されています。

2. **多言語対応とテキスト表現 (`texts/ja_JP.lang` 等)**
   - チュートリアルの指示内容は言語ファイルに定義されています。
   - 日本語の例：
     - `act.1.instruction.actionbar.0` = 「W」キーを押すと前進します。
     - `act.1.instruction.actionbar.1` = 「A」と「D」キーで左右に移動。
     - `act.1.instruction.actionbar.5` = 「W + スペースキー」 で前方にジャンプ。
   - コマンド (`titleraw`) からはこれらの翻訳キー（例: `{"translate":"act.1.instruction.actionbar.0"}`）を呼び出すことで、プレイヤーの言語設定に合わせた指示が出ます。

## まとめ

このワールドは、**「透明な進行管理エンティティ (`rwm:utility`)」**を中心とし、そのスコアボード値（`act`, `phase`）とプレイヤーの位置情報や視線情報を毎ティック (`loop.mcfunction`) 監視することで進行する、堅牢なステートマシンとして設計されています。
また、リソースパックのUIカスタマイズ機能と翻訳ファイルを活用することで、教育用途に特化したわかりやすい操作ヒントを画面上に表示する仕組みを実現しています。
