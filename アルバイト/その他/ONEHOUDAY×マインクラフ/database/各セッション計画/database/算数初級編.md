**【算数・初級編】制作計画書案
テーマ：『ブロックで学ぶ「広さ（面積）」と「かけ算」』
1. 学習のねらい（教育的意図）**
抽象的な「数字」や「九九」を、マインクラフトの「ブロックの数」として可視化し、直感的に理解させる。
• **レベル:** 小学校2年生（かけ算）〜4年生（面積）相当
• **ゴール:** 「縦×横＝全体の数（広さ）」という概念を、体感として理解する。**
2. コンテンツ概要（ストーリー）**
• **設定:** 「村の牧場が狭くて動物たちが困っている！ 新しい柵を作って広げてあげよう」
• **クエスト形式:**
    ◦ **Quest 1（導入）:** 3×2の柵を作る。「中にブロックは何個置けるかな？数えてみよう」
    ◦ **Quest 2（演習）:** 4×5の畑を作る。「いちいち数えるのは大変だね。計算で出せるかな？」
    ◦ **Quest 3（応用）:** 「面積が24になるような家を作ろう」（自由に形を考える＝6×4でも、3×8でもOK）**
3. 開発仕様（World Spec）**
開発工数を抑えつつ、見栄えを良くする構成です。
• **ワールドタイプ:** スーパーフラット（建築しやすい平坦な世界）
• **モード:** クリエイティブ（資材無限・死なない）／ピースフル（敵が出ない）
• **設定:**
    ◦ 「常に昼間」「天候変化なし」に設定（コマンドで固定）。
    ◦ **アドベンチャーモード**を基本とし、特定のエリア（牧場予定地）だけ「ブロック設置可能」にする。（※`deny`ブロックや`allow`ブロックを使用、または教育版の機能を使用）
• **ギミック:**
    ◦ **NPC（村人）:** クエストの発注者。「ありがとう！次は〇〇してほしいな」と喋らせる。
    ◦ **黒板（看板）:** ヒントや問題を掲示しておく。**
4. 納品物構成（PDF教材）**
ここが「3万円」の価値を決める重要な部分です。**
A. 生徒用ワークシート（冒険の書）**
• ゲーム画面を見ながら手元で書き込むプリント。
• 「クエスト1：つくった柵のなかのブロックを数えて書こう [　　] こ」
• 「クエスト2：しきを書いてみよう [　　] × [　　] ＝ [　　]」**
B. 指導者用マニュアル（虎の巻）**
• **事前準備:** ワールドの読み込み方、生徒へのPCの渡し方。
• **授業進行台本:**
    ◦ 「ここで生徒に『縦のブロックは何個ある？』と声をかけてください」
    ◦ 「もし生徒が操作に迷ったら、キーボードのここを押すよう指示してください」
• **トラブルシューティング:** 「ブロックを壊しすぎた時の直し方」**
