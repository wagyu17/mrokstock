- 小学２年生から中学２年生まで6年間サッカーをやっていたが，周りのレベルの高さからスタメンで出ることができなくなりやめた．中学生になったときに，サッカーのための体力づくりの一環として，陸上部に入って長距離を始めた．走るのは好きだったし，中学1年の体力測定の持久走やシャトルランで学年1位を取れて，自信が付いた．サッカーをやめたきっかけに陸上のほうが活躍できそうと感じたし，楽しくなってきたというものが合った．陸上部と駅伝部の顧問がめちゃ怖くて，ビビりながら練習していたが，自分のタイムが伸びることや，結果を出せるようになっていくのが楽しかった．しかし，中学時代が厳しくて高校では陸上部には入らないと卒業時点で思っていた．高校受験は第一志望だった仙台第三高等学校に落ちた．高校は帰宅部になって，浪人もしたので，合計4年間陸上とは無縁だった．高校受験のために通っていた塾のレベルが低くて，その悔しさから高いところを目指すには周りのレベルが高いところにいなくてはならにと潜在的に感じていたのだと振り返って思う．

  - ご家庭様のうちの一人が発達障害をもったこどもの療育を行っている会社経営者で、マイクラを使った療育コンテンツの作成を頼まれて現在作成中。時給3500円でプロジェクト全体で月収40万円近く稼げそう。

### 海外挑戦への憧れ
- マラソンの大迫傑選手がナイキの本社のあるオレゴンにトレーニング拠点を移し世界挑戦を続ける姿, ONE OK ROCK が北米ツアーで現地ファンを熱狂させる姿に刺激を受け, “日本発で世界に通用する仕事をしたい”という想いが芽生えた
- 特にONE OK ROCKは自分の人生のバイブルにもなっている。彼らのチャレンジャー精神、現状に満足しない精神、自分たちの軸をもって生き、結果的に多くの人を惹きつけるカリスマ性がある。

- 根は真面目。だが、必要性を感じないことだったり、やってる意味を感じられないことは不真面目にうまくサボろうとする特性がある。そのため、周囲の環境のレベルが高ければ自然と自分の視座も上がり、最終的にハイレベルになっていることが多いと感じる。ただ、自分の理想としている環境より、その環境のレベルが低い場合は、自ら脱退としてきた。

- 大学に入って，新歓のときに「箱根駅伝の予選会」を目指せるよと，先輩に言われて，駅伝はずっと見るのも好きだったので入部することにした．しかし，大学受験に失敗した経験から，なにか周りとは違うことをやって成長したいという反骨精神から「42Tokyo(2023年8月)というプログラミングスクールの入学試験」や「実際に業務ではどのようにプログラミングが，業務で使われているのかを知るためにインターンにも2週間参加した」．「河野玄斗の塾にもマーケティング分野で応募したが落ちた」．また，（2023年9月）都心での塾講師や家庭教師も，東京の金持ち家庭の家に入れたり，色々と提供してくれそうと思って，見ていた．そんな中，指導科目にMinecraftという案件を見つけてすかさず応募したところ採用された．それから合計7人ほどの小学1~4年生を半年以上見ている．また，インターンに参加して今やるべきことは将来身につくであろう長期インターンの経験より，大学生のうちしか取り組めない部活を頑張ろうと思い，本格的に陸上を頑張り始めた（2025/10）．森田さんという理科大陸上部OBの東京大学大学院で乳酸を研究している修士1年の先輩とコネができて，陸上をサイエンスとして見ている人もいるのかーと面白いと感じた．そこから陸上への熱が加速し始めて，本格的に練習を始めてから3ヶ月でハーフマラソンで76min10secで走れた．その一ヶ月後に，練習後に400mを55secで軽く走れて，自分にはスピードがあることに気がついた．そこから大学対抗戦に対抗枠で選手として出場したいという思いから800mを始めて，一回目のレースで1min59sec30で走れた．いきなり1分台を出せて適正があることを再確認できた．その2ヶ月後に1min56sec88を出して大学歴代記録が見えてきたがそれ以降ベストを更新できず．2つの対抗戦で優勝することができた．そして箱根駅伝予選会のメンバー争いに絡めそうだったため，10000mの練習を本格的に始めて34min02secと標準記録まで2秒のところまで迫ったが，走ることはできなかった．そして800mに切り替えたが，2minも切れなくなりシーズンアウト．どう頑張れば関東インカレの標準記録1min55sec00を切れるかのイメージを掴めず，シーズンアウトの記録会の会場にいた森田さんの共同研究者で見たことはあったが，ほとんど話したことない稲葉さんという方に声をかけて，東大の練習に参加させてもらうことができるようになった．1min50sec前後の記録を持っている東大中距離の選手に食らいついて，冬季シーズンを過ごし，シーズンインから4戦で1min55sec59と惜しいところまで結果を出せたが，2025関東インカレの標準記録には届かなかった．そこから，関カレ標準にチャレンジする機会もあったが，切れず．大学の学業が生活のウエイトの中で重くなってきて，陸上の練習も落としていき，箱根駅伝予選会を目指したが，アキレス腱の怪我の影響もあり十分に走り込みができずDNFでチーム内選考で落選した．そこから切り替えて，長距離練習で培った有酸素能力を活かして，800mで自己ベストの大学歴代5位となる1min55sec39を出した．

- 仲間思いなところもある。2025年の箱根駅伝予選会には選手として出場はできなかったが、チームとしての大学記録更新の目標を達成するためのツールをExcelで作成した。具体的には10km、15km、20km地点で応援する際に目標記録に対してどれくらいの1人あたりの貯金、借金があるかを算出する関数を組み込んだもの。
- また、選手の選考をするときにみんなライバル視して意見するのがはばかられてしまうところを、自分がマラソングランドチャンピオンシップの選考方法を理科大用にアレンジして、それをベースに考えればいいんじゃないかと提案したことでスムーズに話し合いが進んだ。意見できるタイプ。
- あとは、東大の練習で学んだ練習立案にあたっての考え方などを理科大の中距離チームに持って帰り、私が先頭を引っ張ってペースメイクし、今までより高いレベルでの練習をして、結果的にチーム全員の自己ベスト更新さらに全員2分切りという結果につながった。そして大学対抗戦でも、選手3名が8位以内に入り入賞することができた。また、自分自身は関東理工系学生陸上競技大会で優勝、8大学対抗戦で800mで2連覇をした。
