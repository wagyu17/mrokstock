## 聞くべきこと（質問リスト）

### 1. 1次面接で得た回答を起点にした深掘り

- 「1次面接の逆質問で『シリコンバレーのTechカンファレンスに行ける機会がある』と伺ったのですが、**若手社員でも実際に派遣された方の事例**はありますか？どんな経路でその機会を掴んでいるのか知りたいです」
- 「若手の意見がプロジェクト化される仕組みについて、**直近で実際に発足したプロジェクトの事例**を伺えますか？」

→ 「形式上ある」レベルか「実際に動いている」レベルかを見極める。

### 2. 配属の決まり方と動き方

- 「**最初の配属はどう決まりますか？**」「その後、プロジェクト間でどう動けますか？」
- 「**デジタルコンサルタント職の中でも、最新技術／PoC寄りのプロジェクトに入る人**と、**運用・PMO寄りの人**で分かれると思うのですが、最初の配属でその差はどう決まるんですか？」
- 「**技術寄りに行きたい場合、入社時点で何を持っていると配属の希望が通りやすい**ですか？」

### 3. 技術寄り組織への入り方（具体組織名を出す）

- **Accenture Labs**（米国・ダブリン・バンガロール等）
- **Innovation Hub**（東京含む世界各地）
- **AI Refinery**
- **Industry X**
- **Technology Vision**
- **GenAI Studio**

聞き方例：
- 「内定者期間中に**Antigravityを使った自動化ツールを自作**していて、エージェンティックAIを組み合わせる方向に強い関心があります。社内で**GenAI Studio / AI Refinery / Accenture Labs**のような取り組みに**若手から関われるルート**はありますか？」
- 「**Labs／Innovation Hubに入っている若手は、どんな経路でそこに辿り着いたか**を知りたいです」

### 4. 海外関連の実態

- 駐在制度、短期出張、グローバルプロジェクトのアサイン実態
- 「Google I/O / CES / Web Summit など**外部イベントに会社派遣で行ける機会**はあるか」

### 5. 内定者期間（〜2027年4月、残り約11ヶ月）の準備

### 6. 若手の実生活

- 担当案件、勉強時間の確保、社内コミュニティ（AI系勉強会等）

## 言っておくべきこと（自己発信）

- **「新規性・革新性・ワクワクが軸」**を素直に伝える（社員に覚えてもらうフック）
- **「組み合わせる側＝インテグレーター志向」**を整理して伝える
- **Antigravityツールの具体実装経験**を必ず話題に出す
  - 単なる「興味あります」ではなく、**実装した実績ベース**で語ることで、「この人は技術寄りだ」と社員の頭に印象を残す
  - 後日リクルーターチームに「丸岡さんは技術寄りらしい」と引き継がれる可能性

---

## 言わない方がいいこと

- **「PMOには行きたくない」と直接否定しない**
  - 今日の若手社員がまさにPMO担当の可能性が高い
  - 座談会の印象はリクルーターチームに残る場合がある
  - 代わりに「**最新技術・プロダクトに深く関われる案件に最初から関わっていきたい**」とポジティブ表現で
- 内定他社の話（NRI／IBM落ちの件は不要）
- 露骨な配属希望の押し付け（配属希望は別の公式ルートで伝える）
