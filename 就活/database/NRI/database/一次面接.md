- **NRIで築きたいキャリア**
  **AE（アプリケーションエンジニア）として築きたいキャリア**
  **【大目標】** 製造・建設現場の業務をデジタルで変革し、日本の産業基盤を次のステージへ導くソリューション（業界横断プラットフォーム）を創出すること。
  **【ステップ1（入社後1〜3年目）：現場業務の徹底理解とAE基盤の確立】**
  - **やること**: まずは特定の業界（製造業や建設業など）のプロジェクトに参画し、現場の泥臭い業務プロセスを徹底的に学ぶ。
  - **強みの発揮**: 機械工学で培った「物理的制約（現場の制約）の理解」と、相反するニーズをまとめる「利害調整力」を活かし、顧客の現場要件をシステム要件へと正確に翻訳する力を身につける。
  **【ステップ2（将来・長期）：業界横断の業務変革のリード】**
  - **やること**: 個別企業の課題解決で得た現場の知見を抽象化し、NRIの強みである「業界標準プラットフォーム」の構築へと展開する。
  - **目指す姿**: 単なるシステム導入にとどまらず、現場に定着するDX（DX 3.0 / Physical AIの社会実装）を牽引し、業界横断のビジネスモデル変革をリードする存在になる
- **開発工程って知ってる?**
  提供された資料によると、**V字開発工程（V字モデル）**とは、システム開発において「設計（開発）工程」と「テスト工程」がそれぞれ対応していることを示す品質保証モデルのことです。
  具体的には、以下のように左側の「Verification Phase（検証フェーズ・開発側）」と右側の「Validation Phase（妥当性確認フェーズ・テスト側）」がVの字のように対になっています。
  - **要件定義**（Requirement Gathering） ⇔ **受入テスト**（Acceptance Testing）
  - **基本設計**（System Analysis / Software Design） ⇔ **結合テスト**（Integration Testing / System Testing）
  - **詳細設計**（Module Design） ⇔ **単体テスト**（Unit Testing）
  このように、どの段階の設計をどのテストで確認・担保するかが明確になっているのが特徴です。NRIが手掛けるような大規模案件で基本となる「ウォーターフォール（上流から下流へ順次進める手法）」開発などにおいて、品質を担保するための重要な考え方とされています。
- **NRIの主要顧客は？**
  野村総合研究所（NRI）の主要顧客は、大きく分けて**「金融業（証券・保険・銀行など）」**と**「産業分野（製造・流通・サービスなど）」**の2つの柱から構成されています。
  提供された資料（インターンシップでの事業紹介データ等）による売上構成比の目安と、具体的な顧客企業の事例は以下の通りです。
  **1. 業種別の売上構成比と主要ターゲット**
  全体の売上高のうち、各業界が占める割合の目安は以下のようになっています。
  - **メーカー（製造業・サービス業等）：約27%**
    - 現在、NRIが最も注力し成長を牽引している最大の顧客基盤です。製品のサービス化（D2C）やデジタルツイン、サプライチェーンの最適化などのDX需要が旺盛です。
  - **証券業：約19%**
    - NRIの歴史的な強みであり、強固な顧客基盤を持っています。証券業界向けの「THE STAR」などのプラットフォームを業界標準として複数の顧客で共同利用するモデルにより、高い収益性を誇ります。
  - **保険業：約11%**
    - 既存システムのモダナイゼーション（刷新）や、複数企業間の大規模なシステム統合などを数多く手掛けています。
  - **銀行業：約10%**
    - 投資信託のシステム（Bestway）などの提供を行っています。
  - **流通業：約8%**
    - サプライチェーンマネジメント（SCM）の改革や、配送の最適化などの課題解決を支援しています。
  ※このほか、公共分野向けのコンサルティングやシステム開発案件なども手掛けています。
  **2. 具体的な顧客・プロジェクト事例**
  資料に記載されている具体的な顧客企業名とプロジェクトの事例は以下の通りです。
  - **ヤマト運輸（物流・流通）**：AIを活用した配送業務量の予測や、トラックの積載・経路など物理的な制約を考慮した配送ルート最適化プロジェクト。
  - **コマツ（製造・建設）**：NTTドコモ等と共に合弁会社「EARTHBRAIN」を設立し、建設機械の自律化や遠隔制御、地形データのデジタル化など、建設現場のDX（スマートコンストラクション）を共創。
  - **JAL（サービス）**：マイルの新しい価値を生み出す「どこかにマイル」という新サービスの事業創出を支援。
  - **KDDI（通信）**：「au WALLET」などに関わるシステム開発。
  - **あいおいニッセイ同和損害保険（保険）**：データ解析を活用した「テレマティクス保険」の開発支援。
  - **ヨークベニマル（流通）**：「スキャンカート」の導入支援。
  **まとめ**
  NRIは、証券や保険といった**「絶対に止めてはいけない金融インフラ」**を支える顧客基盤で高い利益と安定性を確保しつつ、現在はヤマト運輸やコマツ、JALといった**「製造・流通・サービス業（産業IT）」**の顧客に対して、現実世界（フィジカル）とデジタルを融合させた事業創出（DX 3.0）を強力に推し進めています
