# Claude / Codex 活用・設定メモ (draft)

## 1. スマホからの Git 同期設定 (2026-05-10 追加)

スマホの Claude アプリから `/git` と入力するだけで、PC側のリポジトリを GitHub へプッシュする仕組み。

### 仕組み
- **スキル定義**: `.agents/skills/git-sync/SKILL.md`
- **実行スクリプト**: `Scripts/git-sync.ps1`
- **動作内容**: `git add -A` -> `git commit` (自動メッセージ) -> `git push`

### 使い方
Claude とのチャット欄に以下のように入力する：
- `/git`
- 「git同期」
- 「pushして」

### 注意点
- 100MBを超えるファイルがある場合、GitHubに拒否されるためエラーになる。その場合はPC側でファイルを削除し、履歴をリセットする必要がある。

## 2. Remote Control 機能について (2026-05-10 追加)

ターミナルに表示される「Remote Control Active」は、他のデバイス（スマホやブラウザ）から現在の Claude Code セッションを操作できる状態を指す。

### メリット
- 外出先や別の部屋からスマホ版 Claude アプリで PC のコードを修正させたり、進捗を確認したりできる。
- `Space` キーで QR コードを表示し、スマホでスキャンするだけで接続可能。

### デメリット・注意点
- **セキュリティ**: 常に外部との接続を維持するため、長期間使わない場合はオフにするのが基本。
- **リソース**: わずかだが通信とプロセスを維持するため、バッテリー駆動時はオフが望ましい。

### 操作コマンド
- `/rc` または `/remote-control`: 状態のトグル（ON/OFF）。

## 3. MacBook を「Claude 据え置きサーバー」として使う (2026-05-10 追加)

メイン PC (Windows 等) を持ちつつ、MacBook を Claude 実行専用機とする運用。

### 推奨設定
- **スリープ防止**: `システム設定 > ディスプレイ > 詳細設定`（または省エネルギー）で、電源接続時にスリープしないよう設定。
- **電源接続**: 常に電源アダプタを接続し、ネットワークが途切れないようにする。

### 連携ワークフロー
1. Mac のターミナルで `claude --remote-control` を実行。
2. 表示された URL を **メイン PC (Windows) のブラウザ** で開く。
3. Windows の広い画面と使い慣れたキーボードで、Mac 内のプロジェクトに対して指示を出す。
