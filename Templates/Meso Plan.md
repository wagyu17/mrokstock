## 週次チェック項目

<!-- 毎週日曜日に確認するチェックリスト -->

### 共通チェック（全週）

- [ ] 計画距離の ±10% 以内で実施できたか
- [ ] キーセッションのペースは目標通りだったか
- [ ] 安静時心拍数に異常はないか（± 5 bpm 以内）
- [ ] 睡眠の質・時間は確保できているか（7 時間以上）
- [ ] 怪我・違和感はないか（部位を記録）
- [ ] 体重の急激な変化はないか（± 1 kg 以内）
- [ ] VO2Max 推定値（Garmin）の変化を記録

### 強度チェック

- [ ] インターバルの HR が目標ゾーン（Z4-Z5）に達しているか
- [ ] ジョグ / LSD が Z1-Z2 に収まっているか
- [ ] RPE と Garmin データの乖離がないか

### 疲労管理

- [ ] HRV（心拍変動）を確認（Garmin Body Battery / Whoop 等）
- [ ] 筋肉痛・疲労感の部位と程度を日記に記録
- [ ] 次週の強度調整が必要か判断
