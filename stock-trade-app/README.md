# 株取引メモ

ポーラ・オルビスHD `4927.T` を主に見るためのローカル株取引メモです。

## 起動

```powershell
node server.mjs
```

ブラウザで `http://127.0.0.1:5175/` を開きます。

## 主な機能

- 買った時の値段の左に買った日付、売った時の値段の右に売った日付をコンパクト表示
- 日付欄は通常 `2024年8月25日` のように表示し、選択すると `20240825` の数字入力に切り替わります
- `20240825` の入力で `2024年8月25日`、`20240800` の入力で `2024年8月` と表示
- `240825` のような6桁入力も `2024年8月25日` として扱います
- 取引表は10行ずつページ分割し、各ページの最下行に差分合計を表示
- 右側に実値チャートとニュースを固定表示し、スクロールしても追従
- 銘柄コードと会社名を入力して、ポーラ以外の会社も表示可能
- 12万円から14万円のレンジ監視
- Yahoo Finance の値動きデータを使った実値チャート
- 売買日と入力価格を実値チャートに重ねて比較
- 5分足ベースのRSIを自動更新し、20%付近で買い目安、80%付近で売り目安を表示
- Google News RSS からポーラ関連ニュースを自動更新
- 株主優待、配当利回り、決算確認チェック

外部データは無料取得できる範囲を使うため、リアルタイム性や正確性は提供元に依存します。
