以下の問１，問２，問３，問４に解答して

1986年に起きたスペースシャトル「チャレンジャー号」の爆発事故は、固体燃料補助ロケットのOリングの破損が原因であったと考えられる.S. R. Dalal, E. B. Fowlkes and B. Hoadley (1989) では、チャレンジャー号発射実験より以前の23回のスペースシャトルの打ち上げにおける、外気温 x とOリングの破損件数 y_i のデータを分析している.いま、打ち上げを通じて、y_i は0, 1, 2, 3の値をとり、i = 1, ..., 23 の23回の実験で y_i = 0 で破損がないとし、y_i = 1 で1つの破損件数を持つとする.ここで i = 1, ..., 23 とも、次の文より確率モデルが与えられている.

問１
破損数 y_i はベルヌーイ分布 Bin(1, π_i) に従う.

問２
ロジスティック回帰モデルでは、次の構造を仮定している.
  exp(π_i) / (1 + exp(π_i)) = β_0 + β_1 x

問３
チャレンジャー号の発射当日は、異常気象の影響で外気温は31°Fである.ロジスティック回帰モデルから y_i の値の期待値に基づき、外気温が53°F以下の時に破損率は0.996である.しかし外部専門家によると、民間企業とNASAの議論の中で、
  「最少値53°Fで0.939の確率で破損が起きることが分かった」

最少値53°Fで0.939の確率が正しいかどうか判定してください

問４
観測された外気温の最小値53°Fは摂氏11.7°Fである.また、事故当日の
「31°Fは摂氏−0.6°Cである」
この下解析が正しいかどうか判定せよ.
