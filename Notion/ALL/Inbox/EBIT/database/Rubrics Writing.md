**Common Rubric Errors**

これらの誤りを理解し、回避することは非常に重要です。ルーブリックのミスを頻繁に犯す投稿者は、プロジェクトから外される可能性があります。
❌ 明確な答えがある質問の答えがない。
特定の正解がある質問には、基準の一部として正解を含めなければなりません。回答が真実で正確であることを確認するために、徹底的な調査を行う責任があります。
悪い基準 ❌： 回答は米国の初代大統領の名前を挙げること。
悪い基準 ❌： 回答は提供された方程式 x + 3 = 5 の x を解きます。
悪い基準 ❌： 回答は名前をアルファベット順に並べ替えること
良い基準 ✅： 回答は米国の初代大統領はジョージ・ワシントンであ ると述べるべきである
良い基準 ✅： 応答は方程式x + 3 = 5に基づき、解はx = 2であると述べるべきである。
良い基準 ✅： アルファベット順に名前を並べ替える： アダム、チャーリー、スーザン、トリー

自由形式の質問には例がない。
基準は明確で客観的な真偽の記述とし、回答が基準を満たしているかどうかを容易に判断できるようにする。主観的な判断を求めたり、基準が正しいかどうかの評価が難しくなるような表現は避けましょう。
悪い基準 ❌： 回答はペーパークリップの可能な用途を列挙すべきである。
悪い基準 ❌： 回答はストレスを軽減する方法を述べるべきである。
悪い基準 ❌： 誕生日パーティーのテーマのアイデアを提案する。
良い基準 ✅： 良い基準 ✅: ペーパークリップの使い道として、書類を留めたり、 小さな鍵を開けたり、簡単なアクセサリーを作ったりするこ とができる。
良い基準 ✅： 回答は、マインドフルネスを実践する、体を動かす、心を落ち着かせる音楽を聴くなど、ストレスを軽減する方法を述べるべきである。
良い基準 ✅： 南国のルアウ、スーパーヒーローの冒険、80年代のレトロなパーティーなど、誕生日パーティーのテーマのアイデアを提案する。

主観的または曖昧な基準
基準は客観的な真偽の記述であるべきで、回答に基づいてそれが真か偽かを言うのは簡単である。主観的な判断を必要としたり、真実かどうか言いにくいような基準の書き方は避ける。
悪い基準 ❌： 良いリーダーの資質について述べている。
悪い基準 ❌： 回答は、気候変動がなぜ重要なのかを説明すべきである。
悪い基準 ❌： 回答は記事の主旨を要約すべきである。
良い基準 ✅： 良い基準 ✅: 良いリーダーとは、コミュニケーション、 共感、意思決定スキルなどの資質によって 定義されることを述べる。
良い基準 ✅： 良い基準 ✅: 気候変動は世界の生態系、気象パターン、人間の生 活に影響を与えるので重要であることを述べる。
良い基準 ✅： 良い基準 ❌：回答は、記事の主旨を要約し、都市化が生物多様性に及ぼす影響について述べている。

回答に対するルーブリックの過剰適合
基準はプロンプトに焦点を当てるべきである。与えられた回答からインスピレーションを得ることは構いませんが、4つの回答に従ってルーブリックを作成すべきではありません。ルーブリックは、プロンプトに対するすべての良い回答、悪い回答に対して一般化できるものでなければなりません。
プロンプト どうすれば歯を白くできますか？
回答1: ホワイトニングストリップを使えば歯を白くできます。
回答2: ホワイトニングストリップを使えば歯を白くできます。クレストにはすばらしい製品があります！
回答3: 果物や野菜を食べることで、歯を白くすることができます。リンゴ、ニンジン、セロリは噛んでいる間に自然に歯を磨いてくれるよ。
悪いルーブリック ❌：
回答は、重曹を使った歯磨きで歯を白くすることができると述べていなければならない。
回答は、リンゴ、ニンジン、セロリなどの果物や野菜を食べることで歯を白くすることができると述べなければならない。
良いルーブリック ✅：
回答は、ホワイトニングストリップ、自然に歯を白くする果物や野菜を食べること、ホワイトニング歯磨き粉など、少なくとも1つの歯を白くする方法を提示しなければならない。


❌ 組み合わせ基準
クライテリアは、モデルが何が正しくて何が間違っていたかを明確に理解できるよう、優れた回答の特定の側面に焦点を当てるべきである。優れた回答の複数の要素を一つの基準にまとめることは避ける。
悪い基準 ❌： 第一次エル・アラメインの戦いでは、連合軍は15万の兵力、179台の戦車、1,000以上の大砲、1,500以上の飛行機を保有していた。枢軸国軍は兵力9万6000（イタリア軍5万6000、ドイツ軍4万）、戦車70両、飛行機約500機。アンツィオの戦いでは、連合軍は兵力36,000、車両2,300両、飛行機2,700機。枢軸国軍はドイツ軍20,000人、イタリア軍4,600人、飛行機337機。
良い基準 ✅： 回答は、第一次エル・アラメインの戦いが始まった時点での連合軍兵士数（15万人）、戦車数（179両）、大砲数（1,000以上）、飛行機数（1,500以上）を明記しなければならない。
良い基準 ✅： 回答は、第一次エル・アラメインの戦い開始時の枢軸国軍の兵力数（96,000、うちイタリア軍56,000、ドイツ軍40,000）、戦車数（70）、飛行機数（約500）を明記すること。
良い基準 ✅： 回答は、アンツィオ戦開始時の連合軍兵力（36,000）、車両（2,300）、飛行機（2,700）の数を述べなければならない。
良い基準 ✅： アンツィオ戦開始時の枢軸国軍の兵力数（ドイツ軍20,000、イタリア軍4,600）と航空機数（337）を明記すること。


❌ 個人的な好みと過度に杓子定規な基準
基準は、プロンプトの具体的な要件、強力な回答に必要な必須情報、そして問題ない回答を優れた回答にするための特別な詳細に基づいて設定する必要があります。個人的な嗜好のみに基づく基準は避けましょう。
例題❓： 初デートにおすすめのレストランは？
悪い基準❌： 回答は、フォーリンシネマ、ビックス、ロリンダなど、合計3つのレストランのリストを勧めること。
悪い基準 ❌： お勧めのレストランはイタリア料理であること
悪い基準❌： 予算がないので、一人50ドル以下のレストランを推薦してほしい。
良い基準 ✅： 回答は、フォーリンシネマ、ビックス、ロリンダなど、1つ以上のレストランを推薦すること
良い基準✅： ロマンチックな雰囲気や、第一印象が良くなるような環境など、初デートにふさわしいお店を紹介していること。
良い基準 ✅： サンフランシスコにあるレストランであること


❌ 必須でない文体/書式の好み
プロンプトで特に要求されていない限り、特定の文体や書式の好みを指定する基準は含めないでください。
プロンプトの例❓： 初デートにおすすめのレストランを教えてください。
悪い基準 ❌： お勧めのレストランは箇条書きにすること。
悪い基準 ❌： 回答は極めて簡潔であるべき
悪い基準 ❌： 回答は読みやすく、情報を効果的に伝える良い書式で 書かれるべきである
不正確な基準
例題❓： 米国初代大統領の最も重要な業績を挙げてください。
悪い基準 ❌： ベンジャミン・フランクリンの生い立ち、形成期の経験、政治家としてのキャリアを網羅した伝記を共有してください。
この基準は不正確です。ジョージ・ワシントンはアメリカ合衆国の初代大統領である。

**❌ Unnecessary criteria (e.g. related but not essential)**
Bad Criteria ❌: The response should recommend where to go after dinner.
Good Criteria ✅: The response should recommend more than 1 restaurant, such as Foreign Cinema, Bix, and Lolinda
The prompt does not request date ideas beyond dinner. It is not necessary for the model to include other date activities, so they should not be included.
