#### 【chatGPTの意見】上腹部痛による5/13修正（2026-05-12 追加）

5/12の40分ジョグ中にも腹筋上部痛が出て、その後も押すと一日中軽い痛みが残った。これは単なる走行中の差し込みより、腹直筋上部・肋間・横隔膜付着部まわりの局所炎症/軽い筋損傷として扱う。

5/13朝のセルフチェック:
- 押して痛みが 0-1/10、深呼吸・咳・軽い腹圧で痛みなし → AMは20-30分 easy または予定AMを半分まで。PMは400m×5を流し寄りに抑え、PRは6000m上限。
- 押して痛みが 2/10以上、咳・深呼吸・体幹屈曲で響く → AMポイント中止。PMはジョグ30-40分またはエリプティカルのみ。400m×5と8000m PRは中止。
- 安静時痛、痛みの増悪、発熱/吐き気/腹部膨満/血便、ヘルニア様の膨らみがある → 練習中止、医療相談。

回復優先の処置は、48-72時間の体幹負荷カット、アイシング10-15分を1日2-3回、痛みが落ちた後の温熱、軽い腹式呼吸、十分な糖質・タンパク質・睡眠。痛み止めで隠してポイント練習を実施しない。

追記（2026-05-12 23:00）: 深呼吸では全く痛みなし。痛みは肋骨下縁の筋付着部を押した時のみ軽く出る。内臓系・横隔膜そのもの・肋骨ストレスというより、腹直筋上部/外腹斜筋/肋間周辺の軽い付着部痛として扱う。翌朝も深呼吸痛ゼロ・安静時痛ゼロなら、AMは20-30分 easy または 1000m×3-4 @4:00-4:10・HR145-150上限まで許容。ただし 400m×5 @74s と 8000m @3:30 のセットは腹圧・体幹固定が強いため、押して痛みが残る限り回避または大幅保守化する。

追記（2026-05-13 AM実績）: TM 1% / 16km/h（実質3:45/km）で 1000m×6、R=1分その場静止を実施。FIT上はTM距離補正のズレで総距離4.94km・各疾走800m前後だが、時間構造は225秒疾走×6本で一致するため、トレッドミル実距離を採用。疾走区間HRは 144/155 → 149/158 → 153/161 → 154/161 → 156/163 → 156/162。3本目以降はHRがほぼ頭打ちで、閾値内に収まった良好なAM刺激。ただし22分30秒の3:45/km走を入れているため、PMの400m×5 + 8000m@3:30は原則回避。腹部押痛ゼロならPMは30-40分ジョグ + 流し程度、押痛が残るならエリプティカル/完全休養へ変更する。

DT適性判定: このAMはダブルスレッショルドの1部目としては適切。3:45/km・1%で合計22分30秒、最大HR163（87%HRmax）に収まり、3本目以降のHRドリフトも小さいため、VO2maxではなくLT1.5〜LT2下限の管理された閾値刺激。ただし、これにPM 400m×5 + 8000m@3:30を重ねると、腹部付着部痛の再刺激と閾値超過リスクが高い。今日の2部目はLTではなく回復系に寄せる。

距離評価メモ: AM/PMの10分アップジョグ、5分ダウンジョグで得る距離は、月間走行距離としては意味がある。ただし30分以上連続したZ2ジョグ/ロング走とは別カテゴリ。短いアップ/ダウンは衝撃耐性・フォーム反復・血流促進・総走行距離の下支えには寄与するが、脂質代謝や後半の脚もちを作る主刺激にはならない。月間距離には含めるが、週の有酸素ベース評価では「補助走行距離」として扱い、ロング走や40分E走の代替にしない。

DT効果メモ: 5/13のように、AM/PMともHR160前後で出し切らない閾値系刺激に収める場合、狙いは乳酸を大量に溜めることではなく、乳酸の産生と再利用を高回転で反復すること。MCT1/ミトコンドリア/毛細血管/遅筋動員を刺激し、10000m中盤以降の「乳酸を処理しながらフォームを保つ能力」を育てる。HR168超え、腹部痛再発、ケイデンス低下、接地が重い感覚が出た場合は、ペースを落とすか打ち切る。

水曜AM標準プロトコル: PMチーム練習を主役にする限り、AM 1000mインターバルは **3:45/km・1%を基準** にする。まず3:45で6本→7本→8本へ増やし、3本目以降の最大HRが162-163以内、平均HRドリフトが小さい、RPE6以下、PMの質が落ちないことを確認する。3:40/kmへ上げるのは、3:45×8本を余裕を残して処理でき、かつPMでもHR160前後に収まる状態になってから。上げる場合も全本3:40ではなく、まずラスト1-2本だけ3:40、または3:42/km程度へ微修正する。AMでHR165超えが出る設定は、DTではなくPMを削る準ポイント練習になりやすい。

疲労下PM閾値の意味: AMで軽く糖質・神経・局所筋持久力を削った状態でPM閾値を行うと、同じペースでも内部負荷が上がる。狙いは、MCT1を中心とした乳酸取り込み/再利用、ミトコンドリア酸化能力、Type IIa線維の有酸素化、疲労下フォーム維持、10000m中盤以降に近い内部環境の再現。ただし成立条件は「疲労下でも閾値内」。PM序盤からHR168以上、腹部痛再発、ケイデンス低下、接地の重さ、RPE8以上が出る場合は、ペースを落としてHR160-166程度に戻す。

DT効果の発現スケジュール: 0-48時間は疲労・糖消費が前面に出る。3-7日でRPE低下、HR立ち上がり安定、フォームの戻り。7-14日で同ペースHR低下、HRドリフト低下、レストHR回復改善。14-28日が主効果で、MCT1/ミトコンドリア/毛細血管系の適応により3:30-3:45/km帯をHR160前後で扱いやすくなる。4-8週間で閾値ペースそのものが上がり、3:24/kmの相対負荷が下がる。8-12週間では総距離/ロング走と合わさり後半の脚もちとして出る。6/7で狙う直接効果は、3:24/kmのHRが5/10の168-170から数拍下がること、または同じHRでもフォーム崩壊開始が後ろへずれること。

AM閾値 + PM高強度の扱い: PMでHR170-175まで追い込む場合、それは純粋なDTではなく「AM LT + PM VO2max/レース特異刺激」のコンボ日。10000mの5-9km区間に近い内部環境、Type IIaの有酸素動員、疲労下フォーム維持には有効だが、筋損傷・自律神経負荷・翌日以降の質低下が大きい。通常の水曜DTではPMもHR160-166程度に収め、HR170以上のPMは10-14日に1回以下、かつ最後の1-2km/数本だけに限定する。序盤からHR170に乗る場合は設定過多。

HR140ジョグの扱い: 最大HR187基準ではHR140はZ3寄りで、完全なeasyではない。毎朝30分HR140や深夜14km HR140は有酸素刺激としては有効だが、DT翌日の回復走としては強い。本人はeasy dayをHR125-135程度に抑え、HR140前後は週1-2回の有酸素強化ジョグとして扱う。

才能プロファイル仮説: 本人の強みは、(1) 高強度のスピード練習への反応性、(2) 閾値系でも心拍が低く出る有酸素効率の2点。800m/400mのスピードリザーブ、4/25 VO2max系セッション、5/13 AM 1000m×6 @3:45・1%で最大HR163に収まった事実と整合する。一方で、心拍が低く出るぶん筋腱・腹部・中枢疲労を過小評価しやすい。6/7まではHRだけでなく、ケイデンス、GCT、腹部付着部痛、左膝の局所サインを同時に見る。

測定系メモ: 5/10東海大と5/13AMはCOROS Heart Rate Monitorで心拍測定。HRM 600は胸部センサーでGCT、GCTバランス、上下動、上下動比、Step Speed Lossまで取得できるため、フォーム崩壊分析・左右差確認・レース後半の接地悪化評価ではForerunner 265単体より優先価値が高い。HR閾値の継続比較ではセンサー差に注意し、導入初期はCOROS基準との平均HR/最大HR差を確認する。

## 🏃 フォーム発達トラック（2026-05-11 追加・身長180.5cm／60.5kg 構造課題への対応）

> 5/10 レース解析で判明: race-pace ストライド 1.609m（身長比 **0.89**）= ディスタンスエリート標準 0.95-1.05 に対して **5-10% 短い**。
> 神経筋ポテンシャル（800m 1:55 / 5/6 タッチアップ 1.77m）は存在するが、**長時間有酸素環境で大ストライドを維持するインフラ（腱剛性・偏心筋容量・遅筋ミトコンドリア配分）が未構築**。
> 6/7 までの 27 日間で **race-pace ストライド 1.66-1.68m（+4-5%）への引き上げ**を狙う「副トラック」を追加する。
> ※ 本トラックはメインのダブルLT・VO2max を毀損しないことが絶対条件。**置き換える**のではなく、流し・神経系刺激の **内容を変える**ことで実装する。

| 2. ラスト 1-2km kick のみ "5/6 モード" | kick 区間だけ意図的にストライド 1.75m を狙う（持続性ではなく出力フォーム） |
| 3. トレーニング期にインフラ構築 | ヒル・プライオ・ストライドラン・ドリルを既存の "流し" 枠に組み込む |
| 4. メインセッションには触れない | ダブルLT・VO2max・ロングは既存通り。フォーム介入はあくまで補助 |
