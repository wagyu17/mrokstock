## 前提・現状評価

| 項目 | 内容 |
|------|------|
| 目標 | 10000m **34:00突破（3:24/km）** |
| レース日 | 2026年6月7日（日）東海大学記録会 |

| 5/10レース評価 | DNF 6800m / 24:19。**原因は補食ミス（GI）= 走力起因ではない**。3:24/km @HR 167-169 = max 187の89-91%で走れており、sub-34走力は閾値到達を確認。 |
| 9月2025との比較 | 同ペース3:24/kmで **HR 8-10 bpm低下**（175-179 → 167-169）。有酸素効率は大幅改善済み。 |

### 気温リスク評価（最重要）

前年6/7 夜間 Garmin温度28.5℃（実気温は不明だが確実に5/10より高い）。

| 実気温（推定） | 3:24/kmでの想定HR | 判定 |
|------------|----------------|------|
| ≤ 22℃ | 167-169 bpm | 🟢 5/10と同条件・sub-34直接狙い |
| 23-25℃ | 170-172 bpm | 🟡 ギリギリ管理可能・ラスト kick を3kmへ延ばす |
| 26-28℃ | 173-176 bpm | 🔴 34:05-34:20狙いへ保守化。ペース3:26-3:27/kmに落とす |

**6/7の訓練目標**: 3:24/km @HR 163-165 bpm を実現（5/10比 -4 bpm）= 気温26-27℃でも HR 170前後の余裕を作る。

## 週間構造

```
水曜: AM 自主（TM・扇風機）+ PM チーム練習  ← ダブルLT Day
土曜: AM チーム練習 + PM 自主（TM・扇風機）
日曜: ロングラン
月・木: 完全休養 or エリプティカル（無衝撃）
火・金: イージー + 流し
```

### TM 運用ルール（扇風機使用時）

1. 扇風機は顔・胸に正面から当てる（皮膚血流低下 → HRドリフト抑制）
2. TM 実測 HR は屋外比 +2-3 bpm 高く出る → HR 上限を2-3 bpm低めに設定
3. 1% 傾斜を使用（空気抵抗補正）
4. 15分毎に水2-3口（少量頻回）
