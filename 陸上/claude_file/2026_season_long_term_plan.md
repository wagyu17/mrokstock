## 3. 重要な意思決定ポイント

### 3.1 8/2 vs 8/9 WTTK選択 → **8/2推奨**
- 8/13菅平合宿入りを考慮
- 8/2 → 11日間の回復・適応窓 → クリーンに合宿入り
- 8/9 → 4日間しか空かず、合宿初日から高地ストレス×レッグダメージで合宿前半を潰すリスク

### 3.2 9/19 一発決めの戦略的重要性
- 9/19で32:30クリア → 10/3回避 → HM準備に4週間フル使用（理想）
- 9/19未達 → 10/3挑戦 → HMまで2週間（**テーパー期計画原則に抵触**：直前1-2週の刺激は本番に反映されない）
- **9/19を一発で決めることが箱根予選会HMにも効く最重要条件**

### 3.3 全カレ1500m（9/6）出場の前提条件
- 8/9 WTTK で 3:47.50 達成時のみ（条件付き出場）
- **全力ピーク合わせはせず**、3-5日のミニテーパーのみ
- 位置付け: 9/19への高品質刺激＋経験値・「中継地点」として明文化
- 全カレで結果が出なくても9/19で取り返す想定 → 当日の判断（無理に上げない）がぶれない設計

### 3.4 6月の密度問題（6/7・6/14・6/20・6/28 = 21日で4レース）
- 1月膝受傷からの復帰5ヶ月目の負荷管理が論点
- 5/10で34分切れた場合は **6/7外し → 6/14 1500m集中** の選択肢が浮上
- 5/10結果を見て6月の組み立てを再相談する

## 4. 高地トレーニング科学的注意点（菅平合宿関連）

### 標高1300m 6日間の生理学的効果
- **完全な血液適応（EPO/Hbマス増）には不足**（3週間以上必要）
- **得られる効果**:
  - HIF経路活性化・ミトコンドリア生合成刺激
  - 換気応答の改善
  - 有酸素負荷耐性の引き上げ
  - 心理的・神経的「高地経験」の蓄積

### 帰京後の "post-altitude window"
- Day 1-3: 軽度パフォーマンス低下（再順応期）
- **Day 14-21: 残効ピーク窓**
- Day 30前後: 残効減衰開始

### 9/19レースとの関係
- 8/18帰京 → 9/19レース = **32日間**
- post-altitude window の終端に位置 → 残効を活かせる設計
- ただし 8/19-25 のリアダプテーション1週間は**強度を上げない**（高地疲労を引きずったまま負荷をかけると合宿効果を毀損）
