## 運用原則

1. 水曜・土曜のチーム練習日は、移動だけで疲労を作らないことを優先する。
2. イージー走・流し・ヒルスプリントは、原則として代々木公園周辺で完結させる。
3. ジム利用日は「ジム1往復で完結」を基本にする。ジム + トラックの2拠点移動は、5/20・5/27など核心ゲート日のみ例外。
4. 水曜は13:00-14:00の大学出席を固定予定として扱う。
5. ダブル日でも、午後が主役の日は午前を「刺激」ではなく「準備」に抑える。
6. 就寝は通常22:30、レース週は22:00、前日は21:30を目標にする。

## 移動負担が強い日の代替ルール

| 状況 | 変更 |
|---|---|
| 水曜AMジム + PMチームで移動疲労が強い | AMを近場のHR管理走に置換、PMを主役にする |
| 土曜AMチーム後に眠気・脚の重さが強い | PM自主は30分ジョグまたは完全休養へ変更 |
| ジム往復80分が回復を削る | エリプティカル40分より完全休養を優先 |
| ダブル2部目でHRが序盤から高い | ペース固定を捨て、3:35-3:40/kmへ落とす |
| GI違和感がある | ポイント練習よりGI回復を優先。補食をジェル・バナナ中心へ変更 |
