# Athletics Title Generator (Chrome拡張)

陸上の練習タイトルを英語表記でボタン入力から素早く生成・コピーするChrome拡張機能。
練習の詳細は紙ノートに記録し、本拡張は「タイトル文字列を素早く作る」ことだけを担う。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `manifest.json` | Manifest V3 定義 |
| `popup.html` | ポップアップUI |
| `popup.css` | Notion風スタイル |
| `popup.js` | ロジック（生成・コピー・テンプレート・履歴） |

## Chromeへの読み込み手順

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（`タイトル生成拡張`）を選択
5. ツールバーの拡張機能アイコンからポップアップを起動
   （アイコンが見えないときはツールバーのパズルピース型アイコン → ピン留め）

※ コードを編集したら `chrome://extensions` の本拡張カードで更新ボタン（↻）を押す。

## 使い方（4ステップのウィザード形式）

1. **Step 1 種目**：種目ボタンを押すと自動で次へ
2. **Step 2 場所**：場所を選ぶ／「場所なしで進む」も可（任意）
3. **Step 3 内容**：数値を入力 → 「タイトルを作成 →」
   - 既定では**コア項目のみ**表示。「任意項目もすべて表示」トグルで全項目を展開（設定は保存される）
   - レストは `min` / `sec` / `min:sec`（例 `5:30`）/ `m jog` / `m walk` などから単位を選択
4. **Step 4 タイトル**：自動生成。手動編集可。**Copy** でコピー（履歴に自動保存）

- 上部のステップ表示（1〜4）をクリックすると到達済みステップへ戻れる
- 「最初から」で入力をリセット
- **Templates**（Step 1）：現在の入力を名前付きで保存／クリックで復元
- **History**（Step 1）：Copyしたタイトル最新20件／クリックで復元

## Garmin 自動取り込み

Garmin Connect のアクティビティ画面など、ページ上部に
**距離・タイム・平均ペース・総上昇量・カロリー** が表示されているページで
拡張を開くと、その数値を自動で読み取りバナー表示する。

- Jog系 → Distance / Duration / Pace を自動入力
- Tempo系 → Distance / Pace を自動入力
- 自動入力された値は内容欄でそのまま**手動編集できる**
- ページ読み取りには `activeTab` 権限を使用（拡張アイコンを押した時のみ有効）

## タイトル構造

`[Workout Type] [Main Content] [Target] [Rest] [Pace] [Memo] [@Location]`
（空欄要素は出力しない / `r`=レップ間レスト, `R`=セット間レスト / 場所は末尾に `@Track` 形式）

生成例:
- `Easy Jog 60min`
- `Tempo Run 8km @3:40/km @Road`
- `Interval 400m x 5 x 2set r=200m jog R=8min @Track`
- `Repetition 300m x 5 r=8min walk`
- `Bike Tabata 30s x 8 r=4min`

## 保存について

テンプレート・履歴は `chrome.storage.local`（このPCのChrome内）にのみ保存される。
クラウド同期・外部連携はMVPでは行わない。
