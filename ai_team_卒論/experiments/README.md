# experiments

実験データ、手順、結果、分析を保存する。

## フォルダ

- `past_data/raw`: 過去の実験データをそのまま保存
- `past_data/processed`: 加工済みデータを保存
- `protocols`: 実験手順
- `results`: 実験結果
- `analysis`: 分析メモ・分析スクリプト出力
- `screenshots`: 実験中のスクリーンショット
- `logs`: 実験ログ

## ルール

- raw dataは上書きしない
- 加工したらprocessedへ保存する
- いつ・何を・どう取得したかを `05_DATA_INVENTORY.md` に記録する
