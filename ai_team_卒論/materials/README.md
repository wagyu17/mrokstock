## データ量の目安

| セクション | 容量 |
|-----------|------|
| 実験データ全体 | ~1.9 GB |
| 数値解析結果 | ~1.7 GB |
| レーザー顕微鏡 .cag | 各 ~570 MB（取扱注意） |
| Fortranソース | ~343 KB |
| 合計 | ~3.8 GB |

## ルール

- 出典を必ず残す
- 個人情報を含む資料は不用意にAIへ貼らない
- 実験生データ（実験/）は読み取り専用 — 加工したものは processed/ へ
- 卒論本文で使う資料は `06_MATERIALS_INVENTORY.md` に記録する
